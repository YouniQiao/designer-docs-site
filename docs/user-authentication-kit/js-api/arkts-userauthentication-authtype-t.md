# AuthType

```TypeScript
type AuthType = 'ALL' | 'FACE_ONLY'
```

Enumerates the authentication types.

**Since:** 6

**Deprecated since:** 8

**Substitutes:** [UserAuthType](arkts-userauthentication-userauthtype-e.md#userauthtype)

**System capability:** SystemCapability.UserIAM.UserAuth.Core

| Type | Description |
| --- | --- |
| 'ALL' | Reserved and not supported by the current version. |
| 'FACE_ONLY' | Facial authentication. |
