# NoticeType

```TypeScript
enum NoticeType
```

用户身份认证的通知类型枚举。该枚举定义了系统支持的通知类型，用于标识通知的来源。

**Since:** 10

**System capability:** SystemCapability.UserIAM.UserAuth.Core

**System API:** This is a system API.

## WIDGET_NOTICE

```TypeScript
WIDGET_NOTICE = 1
```

表示该通知由系统统一身份认证控件发出，用于通知用户认证框架相关事件。

**Since:** 10

**System capability:** SystemCapability.UserIAM.UserAuth.Core

**System API:** This is a system API.
