# FingerprintTips

```TypeScript
enum FingerprintTips
```

表示指纹认证过程中提示码的枚举。

**Since:** 8

**Deprecated since:** 11

**System capability:** SystemCapability.UserIAM.UserAuth.Core

## FINGERPRINT_AUTH_TIP_GOOD

```TypeScript
FINGERPRINT_AUTH_TIP_GOOD = 0
```

获取的指纹图像良好。

**Since:** 8

**Deprecated since:** 11

**System capability:** SystemCapability.UserIAM.UserAuth.Core

## FINGERPRINT_AUTH_TIP_DIRTY

```TypeScript
FINGERPRINT_AUTH_TIP_DIRTY = 1
```

由于传感器上可疑或检测到的污垢，指纹图像噪音过大。

**Since:** 8

**Deprecated since:** 11

**System capability:** SystemCapability.UserIAM.UserAuth.Core

## FINGERPRINT_AUTH_TIP_INSUFFICIENT

```TypeScript
FINGERPRINT_AUTH_TIP_INSUFFICIENT = 2
```

由于检测到的情况，指纹图像噪声太大，无法处理。

**Since:** 8

**Deprecated since:** 11

**System capability:** SystemCapability.UserIAM.UserAuth.Core

## FINGERPRINT_AUTH_TIP_PARTIAL

```TypeScript
FINGERPRINT_AUTH_TIP_PARTIAL = 3
```

仅检测到部分指纹图像。

**Since:** 8

**Deprecated since:** 11

**System capability:** SystemCapability.UserIAM.UserAuth.Core

## FINGERPRINT_AUTH_TIP_TOO_FAST

```TypeScript
FINGERPRINT_AUTH_TIP_TOO_FAST = 4
```

快速移动，指纹图像不完整。

**Since:** 8

**Deprecated since:** 11

**System capability:** SystemCapability.UserIAM.UserAuth.Core

## FINGERPRINT_AUTH_TIP_TOO_SLOW

```TypeScript
FINGERPRINT_AUTH_TIP_TOO_SLOW = 5
```

缺少运动，指纹图像无法读取。

**Since:** 8

**Deprecated since:** 11

**System capability:** SystemCapability.UserIAM.UserAuth.Core
