# AuthType

```TypeScript
type AuthType = 'ALL' | 'FACE_ONLY'
```

表示认证类型。

**Since:** 6

**Deprecated since:** 8

**Substitute:** userAuth.UserAuthType

**System capability:** SystemCapability.UserIAM.UserAuth.Core

| Type | Description |
| --- | --- |
| 'ALL' | 预留参数，当前版本暂不支持ALL类型的认证。 |
| 'FACE_ONLY' | 人脸认证。 |
