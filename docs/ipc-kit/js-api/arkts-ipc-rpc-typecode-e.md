# TypeCode

Since API version 12,[writeArrayBuffer](arkts-ipc-rpc-messagesequence-c.md#writearraybuffer-1) and [readArrayBuffer](arkts-ipc-rpc-messagesequence-c.md#readarraybuffer-1) are added to pass ArrayBuffer data. The specific TypedArray type is determined by the **TypeCode** defined as follows:

**Since:** 12

<!--Device-rpc-enum TypeCode--><!--Device-rpc-enum TypeCode-End-->

**System capability:** SystemCapability.Communication.IPC.Core

## INT8_ARRAY

```TypeScript
INT8_ARRAY = 0
```

The TypedArray type is **INT8_ARRAY**.

**Since:** 12
