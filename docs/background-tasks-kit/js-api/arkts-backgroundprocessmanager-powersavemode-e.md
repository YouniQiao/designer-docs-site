# PowerSaveMode

```TypeScript
export enum PowerSaveMode
```

能效模式。

**Since:** 20

**System capability:** SystemCapability.Resourceschedule.BackgroundProcessManager

## EFFICIENCY_MODE

```TypeScript
EFFICIENCY_MODE = 1
```

效率模式，不会跟随系统进入能效模式，进入能效模式后获取到的CPU资源更少。

**Since:** 20

**System capability:** SystemCapability.Resourceschedule.BackgroundProcessManager

## DEFAULT_MODE

```TypeScript
DEFAULT_MODE = 2
```

跟随系统，可能会进入能效模式。

**Since:** 20

**System capability:** SystemCapability.Resourceschedule.BackgroundProcessManager
