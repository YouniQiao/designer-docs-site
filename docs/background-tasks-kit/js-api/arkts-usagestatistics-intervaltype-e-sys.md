# IntervalType

```TypeScript
export enum IntervalType
```

应用使用时长的查询类型。

**Since:** 9

**System capability:** SystemCapability.ResourceSchedule.UsageStatistics.App

**System API:** This is a system API.

## BY_OPTIMIZED

```TypeScript
BY_OPTIMIZED = 0
```

表示系统自行判断最合适的查询类型（天、周、月、年）去查询指定时间段间隔的应用使用时长信息。

**Since:** 9

**System capability:** SystemCapability.ResourceSchedule.UsageStatistics.App
