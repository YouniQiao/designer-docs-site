# notificationActionButton

## Summary

### Interfaces

| Name | Description |
| --- | --- |
| [NotificationActionButton](arkts-notification-notificationactionbutton-i.md) | The **NotificationActionButton** module defines the action buttons displayed in a notification. It is used to addinteractive action buttons in NotificationRequest, allowing users to trigger a **WantAgent** action by tapping thebutton. This module is used when you need to provide interactive action buttons (such as **Reply** and**Mark as read**) in a notification. |
