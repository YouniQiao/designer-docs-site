# BundleOption

> **NOTE** > This API is supported since API version 7 and deprecated since API version 9. You are advised to use [notificationManager.BundleOption](arkts-notification-bundleoption-i.md) instead.

**Since:** 7

**Deprecated since:** 9

**Substitutes:** BundleOption

**System capability:** SystemCapability.Notification.Notification

## bundle

```TypeScript
bundle: string
```

Bundle information of the application.

**Type:** string

**Since:** 7

**Deprecated since:** 9

**Substitutes:** BundleOption

**System capability:** SystemCapability.Notification.Notification

## uid

```TypeScript
uid?: number
```

User ID. The default value is 0.

**Type:** number

**Since:** 7

**Deprecated since:** 9

**Substitutes:** BundleOption

**System capability:** SystemCapability.Notification.Notification
