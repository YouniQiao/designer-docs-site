# NotificationTime

描述通知计时信息。

**Since:** 11

**System capability:** SystemCapability.Notification.Notification

## isPaused

```TypeScript
isPaused?: boolean
```

是否暂停。默认为false。 - true：是。 - false：否。

**Type:** boolean

**Since:** 11

**System capability:** SystemCapability.Notification.Notification

## initialTime

```TypeScript
initialTime?: int
```

计时起始时间。单位：ms。

**Type:** int

**Since:** 11

**System capability:** SystemCapability.Notification.Notification

## isInTitle

```TypeScript
isInTitle?: boolean
```

时间是否展示在title中。默认为false。 - true：是。 - false：否。

**Type:** boolean

**Since:** 11

**System capability:** SystemCapability.Notification.Notification

## isCountDown

```TypeScript
isCountDown?: boolean
```

是否倒计时。默认为false。 - true：是。 - false：否。

**Type:** boolean

**Since:** 11

**System capability:** SystemCapability.Notification.Notification
