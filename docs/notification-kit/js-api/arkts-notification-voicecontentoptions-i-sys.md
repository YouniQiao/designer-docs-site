# VoiceContentOptions (System API)

Describes the voice content options for notification subscriptions.

**Since:** 26.0.0

**System capability:** SystemCapability.Notification.Notification

**System API:** This is a system API.

## enabled

```TypeScript
enabled?: boolean
```

Whether to subscribe to the voice broadcast content of a notification. - **true**: yes. - **false**: no. The default value is **false**.

**Type:** boolean

**Default:** false

**Since:** 26.0.0

**Model restriction:** This API can be used only in the stage model.

**System capability:** SystemCapability.Notification.Notification

**System API:** This is a system API.
