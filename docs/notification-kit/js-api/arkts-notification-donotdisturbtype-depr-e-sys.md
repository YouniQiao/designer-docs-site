# DoNotDisturbType (System API)

> **NOTE**<br> > This API is supported since API version 8 and deprecated since API version 9. You are advised to use > [notificationManager.DoNotDisturbType](arkts-notification-donotdisturbtype-e-sys.md#donotdisturbtype) instead.

**Since:** 8

**Deprecated since:** 9

**Substitutes:** DoNotDisturbType

**System capability:** SystemCapability.Notification.Notification

**System API:** This is a system API.

## TYPE_NONE

```TypeScript
TYPE_NONE = 0
```

Non-DND.

**Since:** 8

**Deprecated since:** 9

**Substitutes:** TYPE_NONE

**System capability:** SystemCapability.Notification.Notification

**System API:** This is a system API.

## TYPE_ONCE

```TypeScript
TYPE_ONCE = 1
```

One-shot DND at the specified time segment (only considering the hour and minute).

**Since:** 8

**Deprecated since:** 9

**Substitutes:** TYPE_ONCE

**System capability:** SystemCapability.Notification.Notification

**System API:** This is a system API.

## TYPE_DAILY

```TypeScript
TYPE_DAILY = 2
```

Daily DND at the specified time segment (only considering the hour and minute).

**Since:** 8

**Deprecated since:** 9

**Substitutes:** TYPE_DAILY

**System capability:** SystemCapability.Notification.Notification

**System API:** This is a system API.

## TYPE_CLEARLY

```TypeScript
TYPE_CLEARLY = 3
```

DND at the specified time segment (considering the year, month, day, hour, and minute).

**Since:** 8

**Deprecated since:** 9

**Substitutes:** TYPE_CLEARLY

**System capability:** SystemCapability.Notification.Notification

**System API:** This is a system API.
