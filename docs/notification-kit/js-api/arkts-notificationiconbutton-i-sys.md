# NotificationIconButton

描述系统通知按钮。

**Since:** 18

**System capability:** SystemCapability.Notification.Notification

**System API:** This is a system API.

## hidePanel

```TypeScript
hidePanel?: boolean
```

点击按钮时，是否隐藏通知中心。默认为false。 - true：是。 - false：否。

**Type:** boolean

**Since:** 18

**System capability:** SystemCapability.Notification.Notification

**System API:** This is a system API.

## iconResource

```TypeScript
iconResource: IconType
```

按钮的背景图。

**Type:** IconType

**Since:** 18

**System capability:** SystemCapability.Notification.Notification

**System API:** This is a system API.

## name

```TypeScript
name: string
```

按钮标识，用于区分同一通知的多个不同按钮。

**Type:** string

**Since:** 18

**System capability:** SystemCapability.Notification.Notification

**System API:** This is a system API.

## text

```TypeScript
text?: string
```

按钮展示的信息。默认为空。

**Type:** string

**Since:** 18

**System capability:** SystemCapability.Notification.Notification

**System API:** This is a system API.
