# DoNotDisturbType

```TypeScript
export enum DoNotDisturbType
```

免打扰设置的时间类型。

**Since:** 9

**System capability:** SystemCapability.Notification.Notification

**System API:** This is a system API.

## TYPE_NONE

```TypeScript
TYPE_NONE = 0
```

非通知勿扰类型。

**Since:** 9

**System capability:** SystemCapability.Notification.Notification

**System API:** This is a system API.

## TYPE_ONCE

```TypeScript
TYPE_ONCE = 1
```

以设置时间段(只看小时和分钟)一次执行勿扰。

**Since:** 9

**System capability:** SystemCapability.Notification.Notification

**System API:** This is a system API.

## TYPE_DAILY

```TypeScript
TYPE_DAILY = 2
```

以设置时间段(只看小时和分钟)每天执行勿扰。

**Since:** 9

**System capability:** SystemCapability.Notification.Notification

**System API:** This is a system API.

## TYPE_CLEARLY

```TypeScript
TYPE_CLEARLY = 3
```

以设置时间段(明确月日时)执行勿扰。

**Since:** 9

**System capability:** SystemCapability.Notification.Notification

**System API:** This is a system API.
