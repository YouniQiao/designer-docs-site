# VideoStateType (System API)

Indicates the type of video state.

**Since:** 7

**System capability:** SystemCapability.Telephony.CallManager

**System API:** This is a system API.

## TYPE_VOICE

```TypeScript
TYPE_VOICE = 0
```

Indicates the call is in voice state.

**Since:** 7

**System capability:** SystemCapability.Telephony.CallManager

**System API:** This is a system API.

## TYPE_VIDEO

```TypeScript
TYPE_VIDEO = 1
```

Indicates the call is in video state.

**Since:** 7

**Deprecated since:** 11

**Substitutes:** TYPE_VIDEO_BIDIRECTIONAL

**System capability:** SystemCapability.Telephony.CallManager

**System API:** This is a system API.

## TYPE_VIDEO_SEND_ONLY

```TypeScript
TYPE_VIDEO_SEND_ONLY = 1
```
