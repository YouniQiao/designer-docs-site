# MmsConfig (System API)

Defines the MMS message config.

**Since:** 11

**System capability:** SystemCapability.Telephony.SmsMms

**System API:** This is a system API.

## Modules to Import

```TypeScript
import { sms } from '@ohos.telephony.sms';
```
