# ConferenceState (System API)

Indicates the state of conference call.

**Since:** 7

**System capability:** SystemCapability.Telephony.CallManager

**System API:** This is a system API.

## TEL_CONFERENCE_IDLE

```TypeScript
TEL_CONFERENCE_IDLE = 0
```

Indicates the state is idle.

**Since:** 7

**System capability:** SystemCapability.Telephony.CallManager

**System API:** This is a system API.
