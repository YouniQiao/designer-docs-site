# LockType (System API)

Indicates the lock types.

**Since:** 8

**System capability:** SystemCapability.Telephony.CoreService

**System API:** This is a system API.

## PIN_LOCK

```TypeScript
PIN_LOCK = 1
```

Indicates the SIM card password lock.

**Since:** 8

**System capability:** SystemCapability.Telephony.CoreService

**System API:** This is a system API.

## FDN_LOCK

```TypeScript
FDN_LOCK = 2
```

Indicates the fixed dialing lock.

**Since:** 8

**System capability:** SystemCapability.Telephony.CoreService

**System API:** This is a system API.
