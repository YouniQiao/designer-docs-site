# MmiCodeResult (System API)

Indicates the MMI code result.

**Since:** 9

**System capability:** SystemCapability.Telephony.CallManager

**System API:** This is a system API.

```TypeScript

```

Indicates the result of MMI code with successfully.

**System API:** This is a system API.
