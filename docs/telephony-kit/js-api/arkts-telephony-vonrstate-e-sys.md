# VoNRState (System API)

Indicates the VoNR state.

**Since:** 10

**System capability:** SystemCapability.Telephony.CallManager

**System API:** This is a system API.

## VONR_STATE_OFF

```TypeScript
VONR_STATE_OFF = 0
```

Indicates the VoNR switch is off.

**Since:** 10

**System capability:** SystemCapability.Telephony.CallManager

**System API:** This is a system API.

## VONR_STATE_ON

```TypeScript
VONR_STATE_ON = 1
```

Indicates the VoNR switch is on.

**Since:** 10

**System capability:** SystemCapability.Telephony.CallManager

**System API:** This is a system API.
