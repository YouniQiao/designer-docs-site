# XCallType (System API)

Indicates the type of the XCall.

**Since:** 26.0.0

**System capability:** SystemCapability.Telephony.CallManager

**System API:** This is a system API.

## XCALL_ECALL_TYPE

```TypeScript
XCALL_ECALL_TYPE = 0
```

Indicates the xcall is Ecall.

**Since:** 26.0.0

**System capability:** SystemCapability.Telephony.CallManager

**System API:** This is a system API.
