# RttState (System API)

Indicates the state of the rtt.

**Since:** 22

**System capability:** SystemCapability.Telephony.CallManager

**System API:** This is a system API.

## RTT_STATE_NO

```TypeScript
RTT_STATE_NO = 0
```

Indicates the rtt is disable.

**Since:** 22

**System capability:** SystemCapability.Telephony.CallManager

**System API:** This is a system API.

## RTT_STATE_YES

```TypeScript
RTT_STATE_YES = 1
```

Indicates the rtt is enable.

**Since:** 22

**System capability:** SystemCapability.Telephony.CallManager

**System API:** This is a system API.

RTT_STATE_REMOTE_TTY = 2
