# AudioDeviceType (System API)

Indicates the device type of the audio device.

**Since:** 10

**System capability:** SystemCapability.Telephony.CallManager

**System API:** This is a system API.

## DEVICE_EARPIECE

```TypeScript
DEVICE_EARPIECE = 0
```

Indicates the audio device is earpiece.

**Since:** 10

**System capability:** SystemCapability.Telephony.CallManager

**System API:** This is a system API.
