# ReportType (System API)

Enumerates report types.

**Since:** 8

**System capability:** SystemCapability.Telephony.SmsMms

**System API:** This is a system API.

## MMS_YES

```TypeScript
MMS_YES = 128
```

Indicates that a report is required.

**Since:** 8

**System capability:** SystemCapability.Telephony.SmsMms

**System API:** This is a system API.

## MMS_NO

```TypeScript
MMS_NO
```
