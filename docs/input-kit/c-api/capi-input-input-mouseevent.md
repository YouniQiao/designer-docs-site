# Input_MouseEvent

```c
typedef struct Input_MouseEvent Input_MouseEvent
```

## Overview

Defines the mouse event object, which is used to represent input events generated by user mouse operations,including click information, coordinates, and click action events. It can be used to process mouse event input andimplement mouse event response.
