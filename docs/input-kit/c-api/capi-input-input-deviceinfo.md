# Input_DeviceInfo

```c
typedef struct Input_DeviceInfo Input_DeviceInfo
```

## Overview

Defines input device information, which is used to describe the basic information and capabilitycharacteristics of an input device, including attributes such as the device type and device ID. You can use thisstruct to obtain and manage detailed information about input devices, facilitating device identification andconfiguration management.
