## LASER_CURSOR

```TypeScript
LASER_CURSOR = 49
```

悬浮光标。手写笔进入空鼠模式时使用该光标，无法直接设置 。 空鼠模式支持通过手写笔在空中转动来控制屏幕上虚拟光标的移动，并借助笔身按键实现上下翻页功能，用于演示PPT、隔空操作等场景。

**Since:** 22

**System capability:** SystemCapability.MultimodalInput.Input.Pointer

**Since:** 22

**System capability:** SystemCapability.MultimodalInput.Input.Pointer
