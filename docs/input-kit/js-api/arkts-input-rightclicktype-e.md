# RightClickType

Enumerates shortcut menu triggering modes.

**Since:** 10

**System capability:** SystemCapability.MultimodalInput.Input.Pointer

## TOUCHPAD_RIGHT_BUTTON

```TypeScript
TOUCHPAD_RIGHT_BUTTON = 1
```

Tapping the right-button area of the touchpad.

**Since:** 10

**System capability:** SystemCapability.MultimodalInput.Input.Pointer
