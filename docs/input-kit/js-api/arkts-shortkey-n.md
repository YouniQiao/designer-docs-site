# shortKey

通过本模块接口，可以设置快捷键拉起Ability的延迟时间，如设置长按快捷键3s后再截屏等。 > **说明：** > > - 本模块接口为系统接口。

**Since:** 10

**System capability:** SystemCapability.MultimodalInput.Input.ShortKey

**System API:** This is a system API.

## Modules to Import

```TypeScript
import { shortKey } from '@kit.InputKit';
```
