# Pinch

Defines a pinch event.

**Since:** 10

**System capability:** SystemCapability.MultimodalInput.Input.Core

## Modules to Import

```TypeScript
import { SwipeInward, FourFingersSwipe, Pinch, ActionType, Rotate, ThreeFingersTap, ThreeFingersSwipe, TouchGestureEvent } from '@ohos.multimodalInput.gestureEvent';
```

## scale

```TypeScript
scale: number
```

Pinch scale factor. The value is greater than or equal to 0.

**Type:** number

**Since:** 10

**System capability:** SystemCapability.MultimodalInput.Input.Core

## type

```TypeScript
type: ActionType
```

Gesture event type, for example, gesture start, gesture update, or gesture end.

**Type:** ActionType

**Since:** 10

**System capability:** SystemCapability.MultimodalInput.Input.Core
