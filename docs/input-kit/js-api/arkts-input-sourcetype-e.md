# SourceType

Enumerates touch sources. Currently, only the touchscreen and touchpad are supported.

**Since:** 9

**System capability:** SystemCapability.MultimodalInput.Input.Core

## TOUCH_SCREEN

```TypeScript
TOUCH_SCREEN = 0
```

Touchscreen.

**Since:** 9

**System capability:** SystemCapability.MultimodalInput.Input.Core

## PEN

```TypeScript
PEN = 1
```

Stylus.

**Since:** 9

**System capability:** SystemCapability.MultimodalInput.Input.Core

## TOUCH_PAD

```TypeScript
TOUCH_PAD = 2
```

Touchpad.

**Since:** 9

**System capability:** SystemCapability.MultimodalInput.Input.Core
