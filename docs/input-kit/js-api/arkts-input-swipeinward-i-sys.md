# SwipeInward (System API)

Defines an inward swipe event.

**Since:** 12

**System capability:** SystemCapability.MultimodalInput.Input.Core

**System API:** This is a system API.

## Modules to Import

```TypeScript
import { SwipeInward, FourFingersSwipe, Pinch, ActionType, Rotate, ThreeFingersTap, ThreeFingersSwipe, TouchGestureEvent } from '@ohos.multimodalInput.gestureEvent';
```

## type

```TypeScript
type: ActionType
```

Type of the inward swipe event. The value is fixed at **SwipeInward**.

**Type:** ActionType

**Since:** 12

**System capability:** SystemCapability.MultimodalInput.Input.Core

**System API:** This is a system API.
