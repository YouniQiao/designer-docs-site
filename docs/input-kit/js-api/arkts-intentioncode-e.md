# IntentionCode

```TypeScript
export declare enum IntentionCode
```

将键盘输入设备的原始事件映射为归一化交互的意图事件，如键盘上空格键映射后的事件为INTENTION_SELECT，意图为选中。

**Since:** 10

**System capability:** SystemCapability.MultimodalInput.Input.Core
