# PrinterCapabilities

定义打印机能力的接口。

**Since:** 14

**System capability:** SystemCapability.Print.PrintFramework

## Modules to Import

```TypeScript
import { print } from '@kit.BasicServicesKit';
```

## supportedMediaTypes

```TypeScript
supportedMediaTypes?: Array<string>
```

表示打印机支持的纸张类型列表。

**Type:** Array<string>

**Since:** 14

**System capability:** SystemCapability.Print.PrintFramework
