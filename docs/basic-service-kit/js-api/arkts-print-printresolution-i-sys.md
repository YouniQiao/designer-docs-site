# PrintResolution

定义打印分辨率的接口。

**Since:** 10

**System capability:** SystemCapability.Print.PrintFramework

**System API:** This is a system API.

## Modules to Import

```TypeScript
import { print } from '@kit.BasicServicesKit';
```

## horizontalDpi

```TypeScript
horizontalDpi: int
```

表示水平DPI。单位：DPI。

**Type:** int

**Since:** 10

**System capability:** SystemCapability.Print.PrintFramework

**System API:** This is a system API.

## verticalDpi

```TypeScript
verticalDpi: int
```

表示垂直DPI。单位：DPI。

**Type:** int

**Since:** 10

**System capability:** SystemCapability.Print.PrintFramework

**System API:** This is a system API.
