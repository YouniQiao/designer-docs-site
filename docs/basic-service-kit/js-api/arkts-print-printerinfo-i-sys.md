# PrinterInfo

定义打印信息的接口。

**Since:** 10

**System capability:** SystemCapability.Print.PrintFramework

**System API:** This is a system API.

## Modules to Import

```TypeScript
import { print } from '@kit.BasicServicesKit';
```

## capability

```TypeScript
capability?: PrinterCapability
```

表示打印机功能。

**Type:** PrinterCapability

**Since:** 10

**System capability:** SystemCapability.Print.PrintFramework

**System API:** This is a system API.

## printerState

```TypeScript
printerState: PrinterState
```

表示当前打印机状态。

**Type:** PrinterState

**Since:** 10

**System capability:** SystemCapability.Print.PrintFramework

**System API:** This is a system API.
