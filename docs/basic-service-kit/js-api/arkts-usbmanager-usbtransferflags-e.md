# UsbTransferFlags

```TypeScript
export enum UsbTransferFlags
```

Enumerates USB transfer flags.

**Since:** 18

**System capability:** SystemCapability.USB.USBManager

## USB_TRANSFER_SHORT_NOT_OK

```TypeScript
USB_TRANSFER_SHORT_NOT_OK = 0
```

将短帧报告为错误。

**Since:** 18

**System capability:** SystemCapability.USB.USBManager

## USB_TRANSFER_FREE_BUFFER

```TypeScript
USB_TRANSFER_FREE_BUFFER = 1
```

自动释放传输缓冲区。

**Since:** 18

**System capability:** SystemCapability.USB.USBManager

## USB_TRANSFER_FREE_TRANSFER

```TypeScript
USB_TRANSFER_FREE_TRANSFER = 2
```

完成回调后自动传输。

**Since:** 18

**System capability:** SystemCapability.USB.USBManager

## USB_TRANSFER_ADD_ZERO_PACKET

```TypeScript
USB_TRANSFER_ADD_ZERO_PACKET = 3
```

传输将增加一个额外的数据包。

**Since:** 18

**System capability:** SystemCapability.USB.USBManager
