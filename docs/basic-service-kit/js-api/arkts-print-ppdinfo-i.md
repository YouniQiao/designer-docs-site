# PpdInfo

定义打印机所使用驱动的PPD文件信息的接口。

**Since:** 24

**System capability:** SystemCapability.Print.PrintFramework

## Modules to Import

```TypeScript
import { print } from '@kit.BasicServicesKit';
```

## ppdName

```TypeScript
ppdName: string
```

表示当前PPD文件的名称。

**Type:** string

**Since:** 24

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.Print.PrintFramework

## nickName

```TypeScript
nickName: string
```

表示当前PPD文件内的打印机别名。

**Type:** string

**Since:** 24

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.Print.PrintFramework

## manufacturer

```TypeScript
manufacturer: string
```

表示当前PPD文件内的打印机厂商名称。

**Type:** string

**Since:** 24

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.Print.PrintFramework
