# Constants

## estimatedRemainingChargeTime

```TypeScript
const estimatedRemainingChargeTime: number
```

表示当前设备充满电的预估时间，单位毫秒。此接口为系统接口。

**Since:** 9

**System capability:** SystemCapability.PowerManager.BatteryManager.Core

**System API:** This is a system API.
