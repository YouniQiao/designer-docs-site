# PrintColorMode

```TypeScript
enum PrintColorMode
```

打印色彩模式的枚举。

**Since:** 11

**System capability:** SystemCapability.Print.PrintFramework

## COLOR_MODE_MONOCHROME

```TypeScript
COLOR_MODE_MONOCHROME = 0
```

表示黑白打印。

**Since:** 11

**System capability:** SystemCapability.Print.PrintFramework

## COLOR_MODE_COLOR

```TypeScript
COLOR_MODE_COLOR = 1
```

表示彩色打印。

**Since:** 11

**System capability:** SystemCapability.Print.PrintFramework
