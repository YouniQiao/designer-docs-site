# PrintDocumentAdapterState

Enumerates the print job states.

**Since:** 11

**System capability:** SystemCapability.Print.PrintFramework

## PREVIEW_DESTROY

```TypeScript
PREVIEW_DESTROY = 0
```

The preview fails.

**Since:** 11

**System capability:** SystemCapability.Print.PrintFramework

## PRINT_TASK_SUCCEED

```TypeScript
PRINT_TASK_SUCCEED = 1
```

The print job is successful.

**Since:** 11

**System capability:** SystemCapability.Print.PrintFramework

## PRINT_TASK_FAIL

```TypeScript
PRINT_TASK_FAIL = 2
```

The print job is failed.

**Since:** 11

**System capability:** SystemCapability.Print.PrintFramework

## PRINT_TASK_CANCEL

```TypeScript
PRINT_TASK_CANCEL = 3
```

The print job is canceled.

**Since:** 11

**System capability:** SystemCapability.Print.PrintFramework

## PRINT_TASK_BLOCK

```TypeScript
PRINT_TASK_BLOCK = 4
```

**Since:** 11

**System capability:** SystemCapability.Print.PrintFramework
