# PrintJobState

Enumerates the print job states.

**Since:** 14

**System capability:** SystemCapability.Print.PrintFramework

## PRINT_JOB_PREPARE

```TypeScript
PRINT_JOB_PREPARE = 0
```

The printer is prepared for the print job.

**Since:** 14

**System capability:** SystemCapability.Print.PrintFramework

## PRINT_JOB_QUEUED

```TypeScript
PRINT_JOB_QUEUED = 1
```

The print job is on the print queue of the printer.

**Since:** 14

**System capability:** SystemCapability.Print.PrintFramework

## PRINT_JOB_RUNNING

```TypeScript
PRINT_JOB_RUNNING = 2
```

The print job is being executed.

**Since:** 14

**System capability:** SystemCapability.Print.PrintFramework

## PRINT_JOB_BLOCKED

```TypeScript
PRINT_JOB_BLOCKED = 3
```

The print job is blocked.

**Since:** 14

**System capability:** SystemCapability.Print.PrintFramework

## PRINT_JOB_COMPLETED

```TypeScript
PRINT_JOB_COMPLETED = 4
```

The print job is complete.

**Since:** 14

**System capability:** SystemCapability.Print.PrintFramework
