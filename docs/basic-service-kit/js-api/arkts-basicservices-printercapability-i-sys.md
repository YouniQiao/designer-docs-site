# PrinterCapability (System API)

Defines the printer capabilities.

**Since:** 24

**System capability:** SystemCapability.Print.PrintFramework

**System API:** This is a system API.

## Modules to Import

```TypeScript
import { print } from '@ohos.print';
```

## colorMode

```TypeScript
colorMode: number
```

Color mode.

**Type:** number

**Since:** 24

**System capability:** SystemCapability.Print.PrintFramework

**System API:** This is a system API.

## duplexMode

```TypeScript
duplexMode: number
```

Simplex or duplex mode.

**Type:** number

**Since:** 24

**System capability:** SystemCapability.Print.PrintFramework

**System API:** This is a system API.
