# PrinterStatus

Enumerates the printer states.

**Since:** 14

**System capability:** SystemCapability.Print.PrintFramework

## PRINTER_IDLE

```TypeScript
PRINTER_IDLE = 0
```

The printer is idle.

**Since:** 14

**System capability:** SystemCapability.Print.PrintFramework

## PRINTER_BUSY

```TypeScript
PRINTER_BUSY = 1
```

The printer is busy.

**Since:** 14

**System capability:** SystemCapability.Print.PrintFramework

## PRINTER_UNAVAILABLE

```TypeScript
PRINTER_UNAVAILABLE = 2
```

The printer is unavailable.

**Since:** 14

**System capability:** SystemCapability.Print.PrintFramework
