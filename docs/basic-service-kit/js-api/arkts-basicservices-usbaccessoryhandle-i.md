# USBAccessoryHandle

Describes the USB accessory handle.

**Since:** 14

**System capability:** SystemCapability.USB.USBManager

## Modules to Import

```TypeScript
import { usbManager } from '@ohos.usbManager';
```

## accessoryFd

```TypeScript
accessoryFd: number
```

Accessory file descriptor. A valid **accessoryFd** is a positive integer.

**Type:** number

**Since:** 14

**System capability:** SystemCapability.USB.USBManager
