# Parity

Enumerates the parity check modes.

**Since:** 19

**System capability:** SystemCapability.USB.USBManager.Serial

## PARITY_NONE

```TypeScript
PARITY_NONE = 0
```

No parity.

**Since:** 19

**System capability:** SystemCapability.USB.USBManager.Serial

## PARITY_ODD

```TypeScript
PARITY_ODD = 1
```

Odd parity.

**Since:** 19

**System capability:** SystemCapability.USB.USBManager.Serial

## PARITY_EVEN

```TypeScript
PARITY_EVEN = 2
```

Even parity.

**Since:** 19

**System capability:** SystemCapability.USB.USBManager.Serial

## PARITY_MARK

```TypeScript
PARITY_MARK = 3
```

Mark parity, whose parity bit is fixed at **1**.

**Since:** 19

**System capability:** SystemCapability.USB.USBManager.Serial

## PARITY_SPACE

```TypeScript
PARITY_SPACE = 4
```

Space parity, whose parity bit is fixed at **0**.

**Since:** 19

**System capability:** SystemCapability.USB.USBManager.Serial
