# Parity

```TypeScript
enum Parity
```

**Since:** 26.0.0

**System capability:** SystemCapability.BusManager.Serial

## NONE

```TypeScript
NONE = 'none'
```

无校验位

**Since:** 26.0.0

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.BusManager.Serial
