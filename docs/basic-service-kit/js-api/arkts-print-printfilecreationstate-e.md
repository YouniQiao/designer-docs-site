# PrintFileCreationState

```TypeScript
enum PrintFileCreationState
```

打印文件创建状态的枚举。

**Since:** 11

**System capability:** SystemCapability.Print.PrintFramework

## PRINT_FILE_CREATED

```TypeScript
PRINT_FILE_CREATED = 0
```

表示打印文件创建成功。

**Since:** 11

**System capability:** SystemCapability.Print.PrintFramework

## PRINT_FILE_CREATION_FAILED

```TypeScript
PRINT_FILE_CREATION_FAILED = 1
```

表示打印文件创建失败。

**Since:** 11

**System capability:** SystemCapability.Print.PrintFramework

## PRINT_FILE_CREATED_UNRENDERED

```TypeScript
PRINT_FILE_CREATED_UNRENDERED = 2
```

表示打印文件创建成功但未渲染。

**Since:** 11

**System capability:** SystemCapability.Print.PrintFramework
