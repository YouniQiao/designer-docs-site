# PreviewAttribute

定义打印预览属性的接口。

**Since:** 10

**System capability:** SystemCapability.Print.PrintFramework

**System API:** This is a system API.

## Modules to Import

```TypeScript
import { print } from '@kit.BasicServicesKit';
```

## result

```TypeScript
result?: int
```

表示预览文件结果。默认值为-1。

**Type:** int

**Since:** 10

**System capability:** SystemCapability.Print.PrintFramework

**System API:** This is a system API.

## previewRange

```TypeScript
previewRange: PrinterRange
```

表示预览页面范围。

**Type:** PrinterRange

**Since:** 10

**System capability:** SystemCapability.Print.PrintFramework

**System API:** This is a system API.
