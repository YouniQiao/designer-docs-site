# PrintPageRange

Defines the print range.

**Since:** 11

**System capability:** SystemCapability.Print.PrintFramework

## Modules to Import

```TypeScript
import { print } from '@ohos.print';
```

## endPage

```TypeScript
endPage?: number
```

End page. The default value is the maximum number of pages of the file to be printed.

**Type:** number

**Since:** 11

**System capability:** SystemCapability.Print.PrintFramework

## pages

```TypeScript
pages?: Array<number>
```

Page range set of the file to print. The default value is empty.

**Type:** Array<number>

**Since:** 11

**System capability:** SystemCapability.Print.PrintFramework

## startPage

```TypeScript
startPage?: number
```

Start page. The default value is **1**.

**Type:** number

**Since:** 11

**System capability:** SystemCapability.Print.PrintFramework
