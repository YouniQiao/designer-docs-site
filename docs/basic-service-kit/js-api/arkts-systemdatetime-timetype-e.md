# TimeType

```TypeScript
enum TimeType
```

定义获取时间的枚举类型。

**Since:** 10

**System capability:** SystemCapability.MiscServices.Time

## STARTUP

```TypeScript
STARTUP = 0
```

自系统启动以来经过的毫秒数，包括深度睡眠时间。

**Since:** 10

**System capability:** SystemCapability.MiscServices.Time

## ACTIVE

```TypeScript
ACTIVE = 1
```

自系统启动以来经过的毫秒数，不包括深度睡眠时间。

**Since:** 10

**System capability:** SystemCapability.MiscServices.Time
