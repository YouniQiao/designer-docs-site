# PanelInfo (System API)

Describes the properties of the word selection panel.

**Since:** 24

**System capability:** SystemCapability.SelectionInput.Selection

**System API:** This is a system API.

## Modules to Import

```TypeScript
import { PanelInfo, PanelType } from '@ohos.selectionInput.SelectionPanel';
```

## height

```TypeScript
height: number
```

Height of the word selection panel, in px.

**Type:** number

**Since:** 24

**Model restriction:** This API can be used only in the stage model.

**System capability:** SystemCapability.SelectionInput.Selection

**System API:** This is a system API.

## panelType

```TypeScript
panelType: PanelType
```

Type of the word selection panel.

**Type:** PanelType

**Default:** MENU_PANEL

**Since:** 24

**Model restriction:** This API can be used only in the stage model.

**System capability:** SystemCapability.SelectionInput.Selection

**System API:** This is a system API.
