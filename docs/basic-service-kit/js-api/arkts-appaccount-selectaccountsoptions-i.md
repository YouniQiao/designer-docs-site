# SelectAccountsOptions

表示用于选择账号的选项。

**Since:** 9

**System capability:** SystemCapability.Account.AppAccount

## Modules to Import

```TypeScript
import { appAccount } from '@kit.BasicServicesKit';
```

## allowedAccounts

```TypeScript
allowedAccounts?: Array<AppAccountInfo>
```

允许的账号数组，默认为空。

**Type:** Array<AppAccountInfo>

**Since:** 9

**System capability:** SystemCapability.Account.AppAccount

## requiredLabels

```TypeScript

```

**Type:** Array<string>

**Since:** 9

**System capability:** SystemCapability.Account.AppAccount
