Task information.

**Since:** 9

**System capability:** SystemCapability.Update.UpdateService

**System API:** This is a system API.

## existTask

```TypeScript
existTask: boolean
```

Whether a task exists. The value **true** indicates that the task exists, and the value **false** indicates the opposite.

**Type:** boolean

**Since:** 9

**System capability:** SystemCapability.Update.UpdateService

**System API:** This is a system API.
