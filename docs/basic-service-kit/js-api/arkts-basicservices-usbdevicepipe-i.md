# USBDevicePipe

Represents a USB device pipe, which is used to determine a USB device.

**Since:** 9

**System capability:** SystemCapability.USB.USBManager

## Modules to Import

```TypeScript
import { usbManager } from '@ohos.usbManager';
```

## busNum

```TypeScript
busNum: number
```

Bus address.

**Type:** number

**Since:** 9

**System capability:** SystemCapability.USB.USBManager

## devAddress

```TypeScript
devAddress: number
```

Device address.

**Type:** number

**Since:** 9

**System capability:** SystemCapability.USB.USBManager
