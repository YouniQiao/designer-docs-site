# SharedHost

定义共享设备信息的接口。

**Since:** 24

**System capability:** SystemCapability.Print.PrintFramework

## Modules to Import

```TypeScript
import { print } from '@kit.BasicServicesKit';
```

## ip

```TypeScript
ip: string
```

表示共享设备的IP地址。

**Type:** string

**Since:** 24

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.Print.PrintFramework

## workgroupName

```TypeScript
workgroupName: string
```

表示共享设备的工作组名称。

**Type:** string

**Since:** 24

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.Print.PrintFramework

## shareName

```TypeScript
shareName: string
```

表示共享设备的主机名称。

**Type:** string

**Since:** 24

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.Print.PrintFramework
