# RemoteAuthOptions

表示远程认证的可选参数集合。

**Since:** 12

**System capability:** SystemCapability.Account.OsAccount

**System API:** This is a system API.

## Modules to Import

```TypeScript
import { osAccount } from '@kit.BasicServicesKit';
```

## collectorNetworkId

```TypeScript
collectorNetworkId?: string
```

凭据收集者的网络标识，默认为空。

**Type:** string

**Since:** 12

**System capability:** SystemCapability.Account.OsAccount

**System API:** This is a system API.

## verifierNetworkId

```TypeScript
verifierNetworkId?: string
```

凭据验证者的网络标识，默认为空。

**Type:** string

**Since:** 12

**System capability:** SystemCapability.Account.OsAccount

**System API:** This is a system API.

## collectorTokenId

```TypeScript
collectorTokenId?: int
```

凭据收集者的令牌标识，默认为undefined。

**Type:** int

**Since:** 12

**System capability:** SystemCapability.Account.OsAccount

**System API:** This is a system API.
