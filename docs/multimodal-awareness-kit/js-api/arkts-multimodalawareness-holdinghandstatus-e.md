# HoldingHandStatus

Represents the holding hand status. The holding hand status is returned if listening for holding hand status
changes is enabled.

**Since:** 20

**System capability:** SystemCapability.MultimodalAwareness.Motion

## NOT_HELD

```TypeScript
NOT_HELD = 0
```

No holding.

**Since:** 20

**System capability:** SystemCapability.MultimodalAwareness.Motion

## LEFT_HAND_HELD

```TypeScript
LEFT_HAND_HELD = 1
```

Holding with the left hand.

**Since:** 20

**System capability:** SystemCapability.MultimodalAwareness.Motion

## RIGHT_HAND_HELD

```TypeScript
RIGHT_HAND_HELD = 2
```

Holding with the right hand.

**Since:** 20

**System capability:** SystemCapability.MultimodalAwareness.Motion

## BOTH_HANDS_HELD

```TypeScript
BOTH_HANDS_HELD = 3
```

Holding with both hands.

**Since:** 20

**System capability:** SystemCapability.MultimodalAwareness.Motion

## UNKNOWN_STATUS

```TypeScript
UNKNOWN_STATUS = 16
```

Unknown status.

**Since:** 20

**System capability:** SystemCapability.MultimodalAwareness.Motion
