# SymKey

Provides APIs for symmetric key operations. It is a child class of [Key](arkts-cryptoarchitecture-key-i.md).
Its objects need to be passed to
[init()](arkts-cryptoarchitecture-cipher-i.md#init-4) of the
[Cipher](arkts-cryptoarchitecture-cipher-i.md) instance in symmetric encryption and decryption.

Symmetric keys can be generated by a [SymKeyGenerator](arkts-cryptoarchitecture-symkeygenerator-i.md).

**Inheritance/Implementation:** SymKey extends [Key](arkts-cryptoarchitecture-key-i.md)

**Since:** 9

**System capability:** 
- API version 12 and later: SystemCapability.Security.CryptoFramework.Key.SymKey
- API version 9 to 11: SystemCapability.Security.CryptoFramework

## Modules to Import

```TypeScript
import { cryptoFramework } from '@kit.CryptoArchitectureKit';
```

## clearMem

```TypeScript
clearMem(): void
```

Clears the key data in memory. This API returns the result synchronously. Call this API when the symmetric key
instance is no longer required.

**Since:** 9

**Atomic service API:** This API can be used in atomic services since API version 12.

**System capability:** 
- API version 12 and later: SystemCapability.Security.CryptoFramework.Key.SymKey
- API version 9 to 11: SystemCapability.Security.CryptoFramework

**Example**

```TypeScript
import { cryptoFramework } from '@kit.CryptoArchitectureKit';

async function testGenerateAesKeyFun() {
  let symKeyGenerator = cryptoFramework.createSymKeyGenerator('AES256');
  let key = await symKeyGenerator.generateSymKey();
  let encodedKey = key.getEncoded();
  console.info('key blob: '+ encodedKey.data);
  key.clearMem();
  encodedKey = key.getEncoded();
  console.info('key blob: ' + encodedKey.data);
}

```
