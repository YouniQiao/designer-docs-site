# SymKey

Provides APIs for symmetric key operations. It is a child class of [Key](arkts-cryptoarchitecture-cryptoframework-key-i.md).Its objects need to be passed to [init()](arkts-cryptoarchitecture-cryptoframework-cipher-i.md#init-1) of the [Cipher](arkts-cryptoarchitecture-cryptoframework-cipher-i.md) instance in symmetric encryption and decryption.

Symmetric keys can be generated by a [SymKeyGenerator](arkts-cryptoarchitecture-cryptoframework-symkeygenerator-i.md).

**Inheritance/Implementation:** SymKey extends [Key](arkts-cryptoarchitecture-cryptoframework-key-i.md)

**Since:** 9

## clearMem

```TypeScript
clearMem(): void
```

Clears the key data in memory. This API returns the result synchronously. Call this API when the symmetric key instance is no longer required.

**Since:** 9

**Atomic service API:** This API can be used in atomic services since API version 12.

<!--Device-SymKey-clearMem(): void--><!--Device-SymKey-clearMem(): void-End-->

**System capability:** 
- API version 12 and later: SystemCapability.Security.CryptoFramework.Key.SymKey
- API version 9 to 11: SystemCapability.Security.CryptoFramework

**Example**

```TypeScript
import { cryptoFramework } from '@kit.CryptoArchitectureKit';

async function testGenerateAesKeyFun() {
  let symKeyGenerator = cryptoFramework.createSymKeyGenerator('AES256');
  let key = await symKeyGenerator.generateSymKey();
  let encodedKey = key.getEncoded();
  console.info('key blob: '+ encodedKey.data);
  key.clearMem();
  encodedKey = key.getEncoded();
  console.info('key blob: ' + encodedKey.data);
}

```
