# CryptoMode

```TypeScript
enum CryptoMode
```

枚举加密和解密的密码操作模式。

**Since:** 9

**System capability:** SystemCapability.Security.CryptoFramework.Cipher

## ENCRYPT_MODE

```TypeScript
ENCRYPT_MODE = 0
```

表示进行加密操作。

**Since:** 9

**Atomic service API:** From API version 12 this API can be used in atomic services.

**System capability:** SystemCapability.Security.CryptoFramework.Cipher

## DECRYPT_MODE

```TypeScript
DECRYPT_MODE = 1
```

表示进行解密操作。

**Since:** 9

**Atomic service API:** From API version 12 this API can be used in atomic services.

**System capability:** SystemCapability.Security.CryptoFramework.Cipher
