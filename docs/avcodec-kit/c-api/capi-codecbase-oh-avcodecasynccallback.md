# OH_AVCodecAsyncCallback

```c
typedef struct OH_AVCodecAsyncCallback {...} OH_AVCodecAsyncCallback
```

## Overview

The struct defines all the asynchronous callback function pointers of an OH_AVCodec instance. To ensure thenormal running of OH_AVCodec, you must register the instance of this struct with the OH_AVCodec instance and processthe information reported by the callback function.
