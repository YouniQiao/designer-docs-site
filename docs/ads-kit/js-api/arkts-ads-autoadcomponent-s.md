# AutoAdComponent

The AutoAdComponent module provides the capability of displaying carousel ads.

**Since:** 11

**System capability:** SystemCapability.Advertising.Ads

## Modules to Import

```TypeScript
import { AutoAdComponent } from '@ohos.advertising.AutoAdComponent';
```

## build

```TypeScript
build(): void
```
