# OH_ImageSourceNative

```c
struct OH_ImageSourceNative
```

## Overview

The struct describes the image source, which is encapsulated at the native layer and is used to create imagedata. The struct cannot be directly operated. Instead, functions must be called to create and release the struct andoperate the fields in the struct.
