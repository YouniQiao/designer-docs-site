# Size

表示图片尺寸。 继承自[lang.ISendable](docroot://arkts-utils/arkts-sendable.md#isendable)。

**Inheritance:** Sizeextends: lang.ISendable.

**Since:** 12

**System capability:** SystemCapability.Multimedia.Image.Core

## Modules to Import

```TypeScript
import { sendableImage } from '@kit.ImageKit';
```

## width

```TypeScript
width: number
```

输出图片的宽，单位：像素。

**Type:** number

**Since:** 12

**Atomic service API:** This API can be used in atomic services.
