# ImageMetadata

图像的元数据集。

**Since:** 23

**System capability:** SystemCapability.Multimedia.Image.Core

## Modules to Import

```TypeScript
import { image } from '@kit.ImageKit';
```

## avisMetadata

```TypeScript
avisMetadata?: AvisMetadata
```

AVIS图像元数据。

**Type:** AvisMetadata

**Since:** 26.0.0

**Model restriction:** This API can be used only in the Stage model.
