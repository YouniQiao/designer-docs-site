# Orientation

```TypeScript
enum Orientation
```

表示图像方向类型的枚举。

**Since:** 23

**System capability:** SystemCapability.Multimedia.Image.Core

## TOP_LEFT

```TypeScript
TOP_LEFT = 1
```

图像未旋转。

**Since:** 23

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.Multimedia.Image.Core

## TOP_RIGHT

```TypeScript
TOP_RIGHT = 2
```

图像是水平镜像的。

**Since:** 23

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.Multimedia.Image.Core

## BOTTOM_RIGHT

```TypeScript
BOTTOM_RIGHT = 3
```

图像旋转180度。

**Since:** 23

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.Multimedia.Image.Core

## BOTTOM_LEFT

```TypeScript
BOTTOM_LEFT = 4
```

图像垂直镜像。

**Since:** 23

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.Multimedia.Image.Core

## LEFT_TOP

```TypeScript
LEFT_TOP = 5
```

图像水平镜像，然后顺时针旋转270度。

**Since:** 23

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.Multimedia.Image.Core
