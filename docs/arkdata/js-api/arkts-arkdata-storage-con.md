# Constants

## MAX_KEY_LENGTH

```TypeScript
const MAX_KEY_LENGTH: 80
```

Indicates the maximum length of a key (80 characters).

**Since:** 6

**Deprecated since:** 9

## MAX_VALUE_LENGTH

```TypeScript
const MAX_VALUE_LENGTH: 8192
```

Indicates the maximum length of a string (8192 characters).

**Since:** 6

**Deprecated since:** 9
