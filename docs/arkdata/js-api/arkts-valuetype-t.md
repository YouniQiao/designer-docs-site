# ValueType

```TypeScript
export type ValueType = long | double | string | boolean
```

Defines the value types allowed in a **ValuesBucket** instance.

**Since:** 10

**Model restriction:** This API can be used only in the Stage model.

**Atomic service API:** From API version 20 this API can be used in atomic services.

**System capability:** SystemCapability.DistributedDataManager.DataShare.Core

| Type | Description |
| --- | --- |
| long | [since 20] |
| double | [since 20] |
| string | The value is a string. [since 12] |
| boolean | The value is true or false. [since 12] |
