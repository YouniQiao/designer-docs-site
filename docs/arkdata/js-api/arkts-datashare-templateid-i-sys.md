# TemplateId

Defines the **TemplateId** struct. **TemplateId** is generated by [**addTemplate**]dataShare.DataShareHelper.addTemplate to identify a template.

**Since:** 10

**System capability:** SystemCapability.DistributedDataManager.DataShare.Consumer

**System API:** This is a system API.

## Modules to Import

```TypeScript
import { dataShare } from '@kit.ArkData';
```

## subscriberId

```TypeScript
subscriberId: string
```

ID of the subscriber who handles the callback. The value must the same as the **subscriberId** in [**addTemplate**]dataShare.DataShareHelper.addTemplate. The ID of each subscriber must be unique.

**Type:** string

**Since:** 10

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.DistributedDataManager.DataShare.Consumer

**System API:** This is a system API.

## bundleNameOfOwner

```TypeScript
bundleNameOfOwner: string
```

Bundle name of the template owner. The value must be the same as the **bundleName** in [**addTemplate**]dataShare.DataShareHelper.addTemplate.

**Type:** string

**Since:** 10

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.DistributedDataManager.DataShare.Consumer

**System API:** This is a system API.
