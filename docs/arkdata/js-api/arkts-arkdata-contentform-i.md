# ContentForm

Represents data of the content widget type.

**Since:** 14

**System capability:** SystemCapability.DistributedDataManager.UDMF.Core

## Modules to Import

```TypeScript
import { uniformDataStruct } from '@ohos.data.uniformDataStruct';
```


**Type:** Uint8Array

**Since:** 14
