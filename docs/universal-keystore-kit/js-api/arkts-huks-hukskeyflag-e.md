# HuksKeyFlag

```TypeScript
export enum HuksKeyFlag
```

表示密钥的产生方式。

**Since:** 8

**System capability:** SystemCapability.Security.Huks.Core

## HUKS_KEY_FLAG_IMPORT_KEY

```TypeScript

```

表示通过导入公钥接口导入的密钥。

**Since:** 8

**Atomic service API:** From API version 12 this API can be used in atomic services.
