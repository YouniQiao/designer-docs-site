# HiCollie

## Overview

Provides the ability to detect thread stuck or jank of your own business threadpls note it should not be the same as main thread.You can use it in these two scenario:(1)The Business thread stuck for 6 seconds;(2)The Business thread jank for a short time, usually it is less than one second.
