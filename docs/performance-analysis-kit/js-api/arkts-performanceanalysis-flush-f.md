## flush

function flush(): void

Flush hilog logs in the sandbox.

**Since:** 26.0.0

**Atomic service API:** This API can be used in atomic services since API version 26.0.0.

**System capability:** SystemCapability.HiviewDFX.HiLog

**Example**

```TypeScript
hilog.flush();

```
