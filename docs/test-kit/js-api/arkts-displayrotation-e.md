# DisplayRotation

```TypeScript
declare enum DisplayRotation
```

描述设备的显示旋转方向。

**Since:** 9

**System capability:** SystemCapability.Test.UiTest

## ROTATION_0

```TypeScript
ROTATION_0 = 0
```

设备显示未旋转，处于原始竖屏方向。

**Since:** 9

**Atomic service API:** From API version 11 this API can be used in atomic services.

**System capability:** SystemCapability.Test.UiTest
