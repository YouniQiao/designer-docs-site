# CollaborationValues

Enumerates application collaboration key values.

**Since:** 18

**System capability:** SystemCapability.DistributedSched.AppCollaboration

## ABILITY_COLLABORATION_TYPE_DEFAULT


Default collaboration.

**Since:** 18
