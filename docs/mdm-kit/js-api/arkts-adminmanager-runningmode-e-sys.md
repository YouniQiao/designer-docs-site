# RunningMode

```TypeScript
export enum RunningMode
```

设备管理的运行模式。

**Since:** 19

**System capability:** SystemCapability.Customization.EnterpriseDeviceManager

**System API:** This is a system API.

## DEFAULT

```TypeScript
DEFAULT = 0
```

默认用户运行模式，表示应用在首次开机后的用户下运行。

**Since:** 19

**System capability:** SystemCapability.Customization.EnterpriseDeviceManager

**System API:** This is a system API.

## MULTI_USER

```TypeScript
MULTI_USER = 1
```

多用户运行模式，表示应用能够在多个用户下同时运行。

**Since:** 19

**System capability:** SystemCapability.Customization.EnterpriseDeviceManager

**System API:** This is a system API.
