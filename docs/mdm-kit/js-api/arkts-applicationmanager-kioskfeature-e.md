# KioskFeature

```TypeScript
enum KioskFeature
```

Kiosk模式的特征。

**Since:** 20

**System capability:** SystemCapability.Customization.EnterpriseDeviceManager

## ALLOW_NOTIFICATION_CENTER

```TypeScript
ALLOW_NOTIFICATION_CENTER = 1
```

允许进入通知中心（通过单指左上方下滑进入）。

**Since:** 20

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.Customization.EnterpriseDeviceManager

## ALLOW_CONTROL_CENTER

```TypeScript
ALLOW_CONTROL_CENTER = 2
```

允许进入控制中心（通过单指右上方下滑进入）。

**Since:** 20

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.Customization.EnterpriseDeviceManager

## ALLOW_GESTURE_CONTROL

```TypeScript
ALLOW_GESTURE_CONTROL = 3
```

允许进入最近任务栏（通过单指底部上滑停留进入）。

**Since:** 24

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.Customization.EnterpriseDeviceManager

## ALLOW_SIDE_DOCK

```TypeScript
ALLOW_SIDE_DOCK = 4
```

允许进入侧边DOCK栏（通过单指边缘内滑停留进入）。

**Since:** 24

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.Customization.EnterpriseDeviceManager
