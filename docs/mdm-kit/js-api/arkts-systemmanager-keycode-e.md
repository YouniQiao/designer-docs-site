# KeyCode

```TypeScript
enum KeyCode
```

按键编码。[添加按键事件策略]systemManager.addKeyEventPolicies、[删除按键事件策略]systemManager.removeKeyEventPolicies、 [获取按键事件策略]systemManager.getKeyEventPolicies和 [按键事件回调](arkts-enterpriseadminextensionability-c.md#onKeyEvent)接口通过按键编码 映射到设备对应实际按键。

**Since:** 23

**System capability:** SystemCapability.Customization.EnterpriseDeviceManager
