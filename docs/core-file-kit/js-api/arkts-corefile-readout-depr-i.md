# ReadOut

Obtains the file read result. This class applies only to the **read()** method.

**Since:** 6

**Deprecated since:** 9

**System capability:** SystemCapability.FileManagement.File.FileIO

## buffer

```TypeScript
buffer: ArrayBuffer
```

Buffer for storing the data read.

**Type:** ArrayBuffer

**Since:** 6

**Deprecated since:** 9

**System capability:** SystemCapability.FileManagement.File.FileIO

## bytesRead

```TypeScript
bytesRead: number
```

Length of the data read, in bytes.

**Type:** number

**Since:** 6

**Deprecated since:** 9

**System capability:** SystemCapability.FileManagement.File.FileIO

## offset

```TypeScript
offset: number
```

Position of the buffer to which the data will be read relative to the start address of the buffer, in bytes.

**Type:** number

**Since:** 6

**Deprecated since:** 9

**System capability:** SystemCapability.FileManagement.File.FileIO
