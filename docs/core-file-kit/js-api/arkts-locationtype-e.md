# LocationType

```TypeScript
declare enum LocationType
```

枚举，文件位置，表示该文件是否在本地或者云端存在。

**Since:** 11

**System capability:** SystemCapability.FileManagement.File.FileIO

## LOCAL

```TypeScript
LOCAL = 1 << 0
```

文件在本地存在。

**Since:** 11

**System capability:** SystemCapability.FileManagement.File.FileIO

## CLOUD

```TypeScript
CLOUD = 1 << 1
```

文件在云端存在。

**Since:** 11

**System capability:** SystemCapability.FileManagement.File.FileIO
