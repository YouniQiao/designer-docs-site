# State

```TypeScript
enum State
```

云文件下载状态，为枚举类型。

**Since:** 11

**System capability:** SystemCapability.FileManagement.DistributedFileService.CloudSync.Core

## RUNNING

```TypeScript
RUNNING = 0
```

云文件正在下载中。

**Since:** 11

**System capability:** SystemCapability.FileManagement.DistributedFileService.CloudSync.Core

## COMPLETED

```TypeScript
COMPLETED = 1
```

云文件下载完成。

**Since:** 11

**System capability:** SystemCapability.FileManagement.DistributedFileService.CloudSync.Core

## FAILED

```TypeScript
FAILED = 2
```

云文件下载失败。

**Since:** 11

**System capability:** SystemCapability.FileManagement.DistributedFileService.CloudSync.Core

## STOPPED

```TypeScript
STOPPED = 3
```

云文件下载已停止。

**Since:** 11

**System capability:** SystemCapability.FileManagement.DistributedFileService.CloudSync.Core

## MISSING

```TypeScript
MISSING = 4
```

Indicates that the download task is missing.

**Since:** 26.0.0

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.FileManagement.DistributedFileService.CloudSync.Core

**System API:** This is a system API.
