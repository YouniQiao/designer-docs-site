# DocumentFlag

组成FileInfo中的mode字段，通过与运算判断是否具备某种能力。

**Since:** 9

**Deprecated since:** 23

**System capability:** SystemCapability.FileManagement.UserFileService

**System API:** This is a system API.

import { fileExtensionInfo } from '@kit.CoreFileKit';
