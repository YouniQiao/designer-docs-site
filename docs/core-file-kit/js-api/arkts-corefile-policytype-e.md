# PolicyType

Indicates the policy type of the path.

**Since:** 15

**System capability:** SystemCapability.FileManagement.AppFileService.FolderAuthorization

## TEMPORARY_TYPE

```TypeScript
TEMPORARY_TYPE = 0
```

Indicates that the policy is temporary.

**Since:** 15

**System capability:** SystemCapability.FileManagement.AppFileService.FolderAuthorization

## PERSISTENT_TYPE

```TypeScript
PERSISTENT_TYPE = 1
```

Indicates that the policy is persistent.

**Since:** 15

**System capability:** SystemCapability.FileManagement.AppFileService.FolderAuthorization
