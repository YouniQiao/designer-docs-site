# @ohos.statfs

The **statfs** module provides APIs for obtaining file system information, including the total size and free size of a file system, in bytes.

**Since:** 8

**Deprecated since:** 9

**Substitutes:** [statvfs:statfs](arkts-file-statvfs.md)

**System capability:** SystemCapability.FileManagement.File.FileIO

## Summary

### Functions

| Name | Description |
| --- | --- |
| [getFreeBytes](arkts-corefile-getfreebytes-depr-f.md#getfreebytes-1) | Obtains the free size of the specified file system, in bytes. This API uses an asynchronous callback to return the result. |
| [getFreeBytes](arkts-corefile-getfreebytes-depr-f.md#getfreebytes-2) | Obtains the free size of the specified file system, in bytes. This API uses a promise to return the result. |
| [getTotalBytes](arkts-corefile-gettotalbytes-depr-f.md#gettotalbytes-1) | Obtains the total size of the specified file system, in bytes. This API uses an asynchronous callback to return the result. |
| [getTotalBytes](arkts-corefile-gettotalbytes-depr-f.md#gettotalbytes-2) | Obtains the total size of the specified file system, in byte. This API uses a promise to return the result. |
