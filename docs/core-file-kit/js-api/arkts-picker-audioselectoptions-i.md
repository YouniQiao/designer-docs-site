## maxSelectNumber

```TypeScript
maxSelectNumber?: int
```

选择文件最大个数，默认值为1，上限为500个，有效值范围1-500。

**Type:** int

**Since:** 23

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.FileManagement.UserFileService
