# GyroscopeResponse

Describes the gyroscope sensor data. It extends from [Response]sensor.Response.

**Inheritance:** GyroscopeResponseextends: Response.

**Since:** 8

**System capability:** SystemCapability.Sensors.Sensor

## x

```TypeScript
x: double
```

Angular velocity of rotation around the x-axis of the device, in rad/s. The value is equal to the reported physical quantity.

**Type:** double

**Since:** 8
