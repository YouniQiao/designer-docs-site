# SocketStateBase

Defines the status of the socket connection.

**Since:** 10

**System capability:** SystemCapability.Communication.NetStack

## Modules to Import

```TypeScript
import { socket } from '@ohos.net.socket';
```

## isBound

```TypeScript
isBound: boolean
```
