## onDoubleClick

```TypeScript
onDoubleClick?: Callback<GestureEvent>
```

[GestureEvent](arkts-arkui-gestureevent-i.md#gestureevent) indicates the double-tap event. Callback event when the double-tap is complete.

**Type:** Callback<GestureEvent>

**Since:** 14

**Model restriction:** This API can be used only in the stage model.
