# OcclusionState

```TypeScript
enum OcclusionState
```

窗口可见性状态枚举。

**Since:** 22

**System capability:** SystemCapability.Window.SessionManager

## NO_OCCLUSION

```TypeScript
NO_OCCLUSION = 0
```

窗口完全可见（没有任何部分被其他非透明窗口遮挡）。

**Since:** 22

**System capability:** SystemCapability.Window.SessionManager

## PARTIAL_OCCLUSION

```TypeScript
PARTIAL_OCCLUSION = 1
```

窗口部分可见（部分被其他非透明窗口遮挡）。

**Since:** 22

**System capability:** SystemCapability.Window.SessionManager

## FULL_OCCLUSION

```TypeScript
FULL_OCCLUSION = 2
```

窗口完全不可见（完全被其他非透明窗口遮挡，或窗口最小化，或窗口隐藏）。

**Since:** 22

**System capability:** SystemCapability.Window.SessionManager
