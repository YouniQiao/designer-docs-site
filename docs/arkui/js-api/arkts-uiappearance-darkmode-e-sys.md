# DarkMode

```TypeScript
enum DarkMode
```

深色模式枚举。 | 名称 | 值 | 说明 | | -- | -- | -- | | ALWAYS_DARK | 0 | 系统始终为深色。 | | ALWAYS_LIGHT | 1 | 系统始终为浅色。 |

**Since:** 10

**System capability:** SystemCapability.ArkUI.UiAppearance

**System API:** This is a system API.

## ALWAYS_DARK

```TypeScript
ALWAYS_DARK = 0
```

Always display with dark mode.

**Since:** 10

**System capability:** SystemCapability.ArkUI.UiAppearance

**System API:** This is a system API.

## ALWAYS_LIGHT

```TypeScript
ALWAYS_LIGHT = 1
```

**System capability:** SystemCapability.ArkUI.UiAppearance

**System API:** This is a system API.
