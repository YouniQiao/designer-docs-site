# Size

表示截取图像的大小。

**Since:** 7

**System capability:** SystemCapability.WindowManager.WindowManager.Core

**System API:** This is a system API.

## Modules to Import

```TypeScript
import { screenshot } from '@kit.ArkUI';
```

## width

```TypeScript
width: long
```

表示截取图像的宽度，单位为px，该参数应为整数。

**Type:** long

**Since:** 7

**System capability:** SystemCapability.WindowManager.WindowManager.Core

**System API:** This is a system API.

## height

```TypeScript
height: long
```

表示截取图像的高度，单位为px，该参数应为整数。

**Type:** long

**Since:** 7

**System capability:** SystemCapability.WindowManager.WindowManager.Core

**System API:** This is a system API.
