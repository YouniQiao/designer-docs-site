# List

```TypeScript
type List = TypedFrameNode<ListInterface, ListAttribute>
```

Represents a FrameNode of the **List** type. This type of node only allows child components of the [ListItem](arkts-arkui-listitem-t.md#listitem) and [ListItemGroup](arkts-arkui-listitemgroup-t.md#listitemgroup) types.

**Since:** 12

**Model restriction:** This API can be used only in the stage model.

**Atomic service API:** This API can be used in atomic services since API version 12.

**System capability:** SystemCapability.ArkUI.ArkUI.Full

**Property type:** TypedFrameNode<ListInterface, ListAttribute>
