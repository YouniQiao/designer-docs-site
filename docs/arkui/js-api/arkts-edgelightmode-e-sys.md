# EdgeLightMode

```TypeScript
declare enum EdgeLightMode
```

边缘光效动画模式枚举。

**Since:** 26.0.0

**System capability:** SystemCapability.ArkUI.ArkUI.Full

**System API:** This is a system API.

## EDGELIGHT_AUTO

```TypeScript
EDGELIGHT_AUTO = 0
```

自适应边缘光效动画。 根据设备运算能力，在低算力设备上关闭，在中高算力设备上开启。

**Since:** 26.0.0

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.ArkUI.ArkUI.Full

**System API:** This is a system API.

## EDGELIGHT_ENABLED

```TypeScript
EDGELIGHT_ENABLED = 1
```

开启边缘光效动画。

**Since:** 26.0.0

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.ArkUI.ArkUI.Full

**System API:** This is a system API.

## EDGELIGHT_DISABLED

```TypeScript
EDGELIGHT_DISABLED = 2
```

关闭边缘光效动画。

**Since:** 26.0.0

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.ArkUI.ArkUI.Full

**System API:** This is a system API.
