# TimePickerResult

Describes a time in 24-hour format.

**Since:** 8

**System capability:** SystemCapability.ArkUI.ArkUI.Full

## hour

```TypeScript
hour: number
```

Hour portion of the selected time.

Value range: [0-23]

**Type:** number

**Since:** 8

**Atomic service API:** This API can be used in atomic services since API version 11.

**System capability:** SystemCapability.ArkUI.ArkUI.Full

**System capability:** SystemCapability.ArkUI.ArkUI.Full
