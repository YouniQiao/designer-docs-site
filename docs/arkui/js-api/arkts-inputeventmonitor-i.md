# InputEventMonitor

输入事件监听器标识对象。 此对象由系统创建并返回，作为监听器的唯一标识。 > **说明：** > > - 对象为空对象，不包含任何可访问的成员。 > > - 开发者无法主动构造此对象，只能通过[addLocalInputEventMonitor]UIContext:UIContext#addLocalInputEventMonitor接口注册获取。 > > - 用于后续解除注册时验证身份。

**Since:** 26.0.0

**Model restriction:** This API can be used only in the Stage model.

**Atomic service API:** This API can be used in atomic services.

**System capability:** SystemCapability.ArkUI.ArkUI.Full
