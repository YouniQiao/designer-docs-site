# ThemeColorMode

```TypeScript
declare enum ThemeColorMode
```

设置颜色模式。

**Since:** 11

**System capability:** SystemCapability.ArkUI.ArkUI.Full

## SYSTEM

```TypeScript
SYSTEM = 0
```

跟随系统深浅色模式。

**Since:** 11

**Model restriction:** This API can be used only in the Stage model.

**Atomic service API:** This API can be used in atomic services.

**System capability:** SystemCapability.ArkUI.ArkUI.Full

## LIGHT

```TypeScript
LIGHT = 1
```

固定使用浅色模式。

**Since:** 11

**Model restriction:** This API can be used only in the Stage model.

**Atomic service API:** This API can be used in atomic services.

**System capability:** SystemCapability.ArkUI.ArkUI.Full

## DARK

```TypeScript
DARK = 2
```

固定使用深色模式。

**Since:** 11

**Model restriction:** This API can be used only in the Stage model.

**Atomic service API:** This API can be used in atomic services.

**System capability:** SystemCapability.ArkUI.ArkUI.Full
