# cursorControl

控制鼠标光标的显示样式。

**Since:** 11

**System capability:** SystemCapability.ArkUI.ArkUI.Full

## Summary

### Functions

| Name | Description |
| --- | --- |
| [restoreDefault](arkts-cursorcontrol-restoredefault-f.md#restoreDefault-1) | 方法语句中可使用的全局接口，调用此接口可将鼠标光标恢复成默认箭头样式。 |
| [setCursor](arkts-cursorcontrol-setcursor-f.md#setCursor-1) | 方法语句中可使用的全局接口，调用该接口可更改当前的鼠标光标样式。 |
