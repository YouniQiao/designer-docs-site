# StatusBarProperty

状态栏的属性。在获取状态栏属性信息时返回。

**Since:** 18

**System capability:** SystemCapability.Window.SessionManager

## Modules to Import

```TypeScript
import { window } from '@kit.ArkUI';
```

## contentColor

```TypeScript
contentColor: string
```

状态栏文字颜色，固定为ARGB格式，如：`#E5FFFFFF`。

**Type:** string

**Since:** 18

**Atomic service API:** This API can be used in atomic services.

**System capability:** SystemCapability.Window.SessionManager
