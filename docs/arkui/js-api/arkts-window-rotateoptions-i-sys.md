# RotateOptions

旋转参数。

**Since:** 9

**System capability:** SystemCapability.WindowManager.WindowManager.Core

**System API:** This is a system API.

## Modules to Import

```TypeScript
import { window } from '@kit.ArkUI';
```

## x

```TypeScript
x?: double
```

绕X轴的旋转角度。该参数为浮点数，默认值为0.0。

**Type:** double

**Since:** 9

**System capability:** SystemCapability.WindowManager.WindowManager.Core

**System API:** This is a system API.

## pivotY

```TypeScript
pivotY?: double
```

**Type:** double
