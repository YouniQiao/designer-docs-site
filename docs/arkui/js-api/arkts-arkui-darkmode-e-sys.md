# DarkMode (System API)

Enumerates the color modes.

**Since:** 20

**System capability:** SystemCapability.ArkUI.UiAppearance

**System API:** This is a system API.

## ALWAYS_DARK

```TypeScript
ALWAYS_DARK = 0
```

Always display with dark mode.

**Since:** 20

**System capability:** SystemCapability.ArkUI.UiAppearance

**System API:** This is a system API.

## ALWAYS_LIGHT

```TypeScript
ALWAYS_LIGHT = 1
```

Always display with light mode.

**Since:** 20

**System capability:** SystemCapability.ArkUI.UiAppearance

**System API:** This is a system API.
