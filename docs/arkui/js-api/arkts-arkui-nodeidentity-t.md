# NodeIdentity

```TypeScript
export declare type NodeIdentity = string | number
```

Defines the type can be used for identiting the node, for the string type, it's the inspector id
set through .[id](../arkts-components/arkts-arkui-commonmethod-c.md#id-1) attribute, and for the number type, it's the unique ID got from the FrameNode by
[getUniqueId](arkts-arkui-framenode-c.md#getuniqueid-1) method.

**Since:** 20

**Model restriction:** This API can be used only in the stage model.

**Atomic service API:** This API can be used in atomic services since API version 20.

**System capability:** SystemCapability.ArkUI.ArkUI.Full
