# Rect

表示截取图像的区域。

**Since:** 12

**System capability:** SystemCapability.WindowManager.WindowManager.Core

## Modules to Import

```TypeScript
import { screenshot } from '@kit.ArkUI';
```

## top

```TypeScript
top: long
```

表示截取图像区域的上边界，单位为px，该参数应为整数。

**Type:** long

**Since:** 12

**Atomic service API:** This API can be used in atomic services.

**System capability:** SystemCapability.WindowManager.WindowManager.Core
