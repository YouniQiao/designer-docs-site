# SceneOptions

3D场景控制使用的场景选项

**Since:** 12

**System capability:** SystemCapability.ArkUi.Graphics3D

## modelType

```TypeScript
modelType?: ModelType
```

3D渲染时的场景类型

**Type:** ModelType

**Since:** 12

**Atomic service API:** This API can be used in atomic services.

**System capability:** SystemCapability.ArkUi.Graphics3D

## scene

```TypeScript
scene?: ResourceStr | Scene
```

3D渲染的资源类型，3D场景控制的Scene类型

**Type:** ResourceStr | Scene

**Since:** 12

**Atomic service API:** This API can be used in atomic services.

**System capability:** SystemCapability.ArkUi.Graphics3D
