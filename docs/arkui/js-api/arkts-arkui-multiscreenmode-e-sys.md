# MultiScreenMode (System API)

Enumerates the display modes of secondary screens.

**Since:** 13

**System capability:** SystemCapability.WindowManager.WindowManager.Core

**System API:** This is a system API.

## SCREEN_MIRROR

```TypeScript
SCREEN_MIRROR = 0
```

Mirror mode.

**Since:** 13

**System capability:** SystemCapability.WindowManager.WindowManager.Core

**System API:** This is a system API.
