# StartLineInfo (System API)

Define start line info used in GridLayoutOptions.

**Since:** 23

**System capability:** SystemCapability.ArkUI.ArkUI.Full

**System API:** This is a system API.

## startIndex

```TypeScript
startIndex: number
```

Define the start index of the row where the target index or offset is located.

**Type:** number

**Since:** 23

**Model restriction:** This API can be used only in the stage model.

**System capability:** SystemCapability.ArkUI.ArkUI.Full

**System API:** This is a system API.

## startLine

```TypeScript
startLine: number
```

Define the start row of the item with startIndex.

**Type:** number

**Since:** 23

**Model restriction:** This API can be used only in the stage model.

**System capability:** SystemCapability.ArkUI.ArkUI.Full

**System API:** This is a system API.

## startOffset

```TypeScript
startOffset: number
```

Offset between the top of the **GridItem** corresponding to **startIndex** and the top of the **Grid**.
<br>Unit: vp

**Type:** number

**Since:** 23

**Model restriction:** This API can be used only in the stage model.

**System capability:** SystemCapability.ArkUI.ArkUI.Full

**System API:** This is a system API.

## totalOffset

```TypeScript
totalOffset: number
```

Total scrolling offset,
that is, the offset between the top of the first **GridItem** in the **Grid** component
and the top of the **Grid** component.<br>Unit: vp

**Type:** number

**Since:** 23

**Model restriction:** This API can be used only in the stage model.

**System capability:** SystemCapability.ArkUI.ArkUI.Full

**System API:** This is a system API.
