# ToolBarOptions

Declare ToolBarOptions use in ToolBar

**Inheritance/Implementation:** ToolBarOptions extends [Array<ToolBarOption>](Array<ToolBarOption>)

**Since:** 18

**Model restriction:** This API can be used only in the stage model.

**Atomic service API:** This API can be used in atomic services since API version 18.

**System capability:** SystemCapability.ArkUI.ArkUI.Full

## Modules to Import

```TypeScript
import { ToolBarOption, ItemState, ToolBar, ToolBarOptions, ToolBarModifier } from '@ohos.arkui.advanced.ToolBar';
```
