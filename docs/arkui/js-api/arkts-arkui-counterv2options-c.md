# CounterV2Options

Defines the counter options.

**Since:** 26.0.0

**System capability:** SystemCapability.ArkUI.ArkUI.Full

## Modules to Import

```TypeScript
import { CounterV2Component, CounterV2Options, CounterV2Type, CounterV2DateData } from '@ohos.arkui.advanced.CounterV2';
```

## dateOptions

```TypeScript
dateOptions?: CounterV2DateStyleOptions
```

Set the counter attribute of the INLINE_DATE counter component.

**Type:** CounterV2DateStyleOptions

**Since:** 26.0.0

**Model restriction:** This API can be used only in the stage model.

**Atomic service API:** This API can be used in atomic services since API version 26.0.0.

**System capability:** SystemCapability.ArkUI.ArkUI.Full
