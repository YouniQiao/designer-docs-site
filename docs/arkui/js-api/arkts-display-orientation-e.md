# Orientation

```TypeScript
enum Orientation
```

显示设备当前显示的方向枚举。

**Since:** 10

**System capability:** SystemCapability.WindowManager.WindowManager.Core

## PORTRAIT

```TypeScript
PORTRAIT = 0
```

表示设备当前以竖屏方式显示。

**Since:** 10

**Atomic service API:** From API version 12 this API can be used in atomic services.

**System capability:** SystemCapability.WindowManager.WindowManager.Core
