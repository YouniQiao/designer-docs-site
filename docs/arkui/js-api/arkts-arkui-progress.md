# Progress

The **Progress** component represents a progress indicator that displays the progress of content loading or an
operation.


## Progress

```TypeScript
Progress(options: ProgressOptions<Type>)
```

Creates a progress indicator.

**Since:** 7

**Atomic service API:** This API can be used in atomic services since API version 11.

**Widget capability:** This API can be used in ArkTS widgets since API version 9.

**System capability:** SystemCapability.ArkUI.ArkUI.Full

**Parameters:**

| Name | Type | Mandatory | Description |
| --- | --- | --- | --- |
| options | ProgressOptions&lt;Type&gt; | Yes | Options of the progress indicator, which vary by progress indicatortype. |
