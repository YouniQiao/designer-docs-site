# Size

窗口大小，单位为px。

**Since:** 7

**System capability:** SystemCapability.WindowManager.WindowManager.Core

## Modules to Import

```TypeScript
import { window } from '@kit.ArkUI';
```

## width

```TypeScript
width: int
```

窗口宽度，单位为px，该参数应为整数。

**Type:** int

**Since:** 7

**Atomic service API:** From API version 11 this API can be used in atomic services.

**System capability:** SystemCapability.WindowManager.WindowManager.Core

## height

```TypeScript
height: int
```

窗口高度，单位为px，该参数应为整数。

**Type:** int

**Since:** 7

**Atomic service API:** From API version 11 this API can be used in atomic services.

**System capability:** SystemCapability.WindowManager.WindowManager.Core
