# Constants

## TextTimer

```TypeScript
declare const TextTimer: TextTimerInterface
```

通过文本显示计时信息并控制其计时器状态的组件。 组件不可见（非锁屏状态和应用后台状态）时，UI时间变动将停止（即该组件此时不会绘制），[onTimer](arkts-texttimerattribute-c.md#onTimer)仍然会正常触发。 ###### 子组件 无

**Since:** 8

**Atomic service API:** From API version 11 this API can be used in atomic services.

**Widget capability:** From API version 10 this API can be used in ArkTS widgets.

**System capability:** SystemCapability.ArkUI.ArkUI.Full
