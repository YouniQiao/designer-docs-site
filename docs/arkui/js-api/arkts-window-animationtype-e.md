# AnimationType

```TypeScript
enum AnimationType
```

窗口动画类型枚举。

**Since:** 20

**System capability:** SystemCapability.Window.SessionManager

## FADE_IN_OUT

```TypeScript
FADE_IN_OUT = 0
```

表示窗口动画类型为淡入淡出。淡入动画在窗口显示过程中生效，淡出动画在窗口隐藏过程中生效。

**Since:** 20

**System capability:** SystemCapability.Window.SessionManager

## FADE_IN

```TypeScript
FADE_IN = 1
```

表示窗口动画类型为淡入。淡入动画在窗口显示过程中生效。

**Since:** 20

**System capability:** SystemCapability.Window.SessionManager

**System API:** This is a system API.
