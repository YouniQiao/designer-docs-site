# SystemBarTintState

当前系统栏回调信息集合。

**Since:** 8

**System capability:** SystemCapability.WindowManager.WindowManager.Core

**System API:** This is a system API.

## Modules to Import

```TypeScript
import { window } from '@kit.ArkUI';
```

## regionTint

```TypeScript
regionTint: Array<SystemBarRegionTint>
```

当前已改变的所有系统栏信息。

**Type:** Array<SystemBarRegionTint>

**Since:** 8

**System capability:** SystemCapability.WindowManager.WindowManager.Core

**System API:** This is a system API.
