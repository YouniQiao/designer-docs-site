# menu_item_group

## menu_item_group

```TypeScript
menu_item_group(value?: MenuItemGroupOptions)
```

**Since:** 9

**Atomic service API:** From API version 11 this API can be used in atomic services.

**System capability:** SystemCapability.ArkUI.ArkUI.Full

**Parameters:**

| Name | Type | Mandatory | Description |
| --- | --- | --- | --- |
| value | MenuItemGroupOptions | No | 包含设置MenuItemGroup的标题和尾部显示信息。 未设置时，不显示标题和尾部信息。 |
