# CounterOptions

Defines the counter options.

**Since:** 11

**System capability:** SystemCapability.ArkUI.ArkUI.Full

## Modules to Import

```TypeScript
import { CounterType, DateData, CounterComponent, CounterOptions } from '@ohos.arkui.advanced.Counter';
```

## dateOptions

```TypeScript
dateOptions?: DateStyleOptions
```

Set the counter attribute of the INLINE_DATE counter component.

**Type:** DateStyleOptions

**Since:** 11

**Model restriction:** This API can be used only in the stage model.

**Atomic service API:** This API can be used in atomic services since API version 12.

**System capability:** SystemCapability.ArkUI.ArkUI.Full
