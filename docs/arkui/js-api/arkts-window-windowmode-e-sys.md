# WindowMode

```TypeScript
enum WindowMode
```

窗口模式枚举。

**Since:** 7

**System capability:** SystemCapability.WindowManager.WindowManager.Core

**System API:** This is a system API.

## UNDEFINED

```TypeScript
UNDEFINED = 1
```

表示APP未定义窗口模式。

**Since:** 7

**System capability:** SystemCapability.WindowManager.WindowManager.Core

**System API:** This is a system API.

## FULLSCREEN

```TypeScript
FULLSCREEN = 2
```

表示APP全屏模式。

**Since:** 7

**System capability:** SystemCapability.WindowManager.WindowManager.Core

**System API:** This is a system API.

## PRIMARY

```TypeScript
PRIMARY = 3
```

表示APP分屏多窗口主要模式。上下分屏时，上分屏为主要模式。左右分屏时，左分屏为主要模式。

**Since:** 7

**System capability:** SystemCapability.WindowManager.WindowManager.Core

**System API:** This is a system API.

## SECONDARY

```TypeScript
SECONDARY = 4
```

表示APP分屏多窗口次要模式。上下分屏时，下分屏为次要模式。左右分屏时，右分屏为次要模式。

**Since:** 7

**System capability:** SystemCapability.WindowManager.WindowManager.Core

**System API:** This is a system API.

## FLOATING

```TypeScript
FLOATING = 5
```

表示APP自由悬浮形式窗口模式。

**Since:** 7

**System capability:** SystemCapability.WindowManager.WindowManager.Core

**System API:** This is a system API.
