# focusControl

Implements focus control.

**Since:** 9

**System capability:** SystemCapability.ArkUI.ArkUI.Full

## Summary

### Functions

| Name | Description |
| --- | --- |
| [requestFocus](arkts-arkui-requestfocus-f.md#requestfocus-1) | Requests focus transfer to the specified component during the next frame rendering. This global API provides asynchronous focus control. For scenarios requiring immediate focus changes, it is recommended that you use the focus synchronization transfer API [requestFocus](../arkts-apis/arkts-arkui-focuscontroller-c.md#requestfocus-1) in **FocusController**. |
