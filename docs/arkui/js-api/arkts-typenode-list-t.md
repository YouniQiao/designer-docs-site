# List

```TypeScript
type List = TypedFrameNode<ListInterface, ListAttribute>
```

List类型的FrameNode节点类型。只允许添加[ListItem]typeNode.ListItem、[ListItemGroup]typeNode.ListItemGroup类型子组件。

**Since:** 12

**Model restriction:** This API can be used only in the Stage model.

**Atomic service API:** This API can be used in atomic services.

**System capability:** SystemCapability.ArkUI.ArkUI.Full

**Property type:** TypedFrameNode<ListInterface, ListAttribute>
