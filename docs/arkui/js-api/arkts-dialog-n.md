# dialog

提供统一的Dialog API。

**Since:** 26.1.0

**System capability:** SystemCapability.ArkUI.ArkUI.Full

## Modules to Import

```TypeScript
import { DialogButtonOrientation,DialogState,DialogResult,DialogBaseController,DialogBaseAlignment,DialogDismissal } from '@kit.ArkUI';
```

## Summary

### Interfaces

| Name | Description |
| --- | --- |
| [DialogBaseOptions](arkts-dialog-dialogbaseoptions-i.md) | 所有Dialog类型共享的基本选项。 |
| [DialogButton](arkts-dialog-dialogbutton-i.md) | 固定样式对话框的按钮配置。 |
| [DialogCustomOptions](arkts-dialog-dialogcustomoptions-i.md) | 自定义样式对话框的选项。 对话内容作为present()方法的第一个参数提供。 不在此选项对象中。 |
| [DialogMessage](arkts-dialog-dialogmessage-i.md) | 对话框的消息选项。 |
| [DialogSheet](arkts-dialog-dialogsheet-i.md) | Dialog sheet的配置项。 |
| [DialogStyleOptions](arkts-dialog-dialogstyleoptions-i.md) | 固定样式对话框的选项。 |
| [DialogTextStyleOptions](arkts-dialog-dialogtextstyleoptions-i.md) | 在Dialog定义文本样式属性。 |
