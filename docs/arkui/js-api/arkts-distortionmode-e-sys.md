# DistortionMode

```TypeScript
declare enum DistortionMode
```

非线性形变动画模式的枚举。

**Since:** 26.0.0

**System capability:** SystemCapability.ArkUI.ArkUI.Full

**System API:** This is a system API.

## DISTORTION_AUTO

```TypeScript
DISTORTION_AUTO = 0
```

distortion动画自适应实现

**Since:** 26.0.0

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.ArkUI.ArkUI.Full

**System API:** This is a system API.

## DISTORTION_ENABLED

```TypeScript
DISTORTION_ENABLED = 1
```

distortion动画一直使能

**Since:** 26.0.0

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.ArkUI.ArkUI.Full

**System API:** This is a system API.

## DISTORTION_DISABLED

```TypeScript
DISTORTION_DISABLED = 2
```

禁用distortion动画。

**Since:** 26.0.0

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.ArkUI.ArkUI.Full

**System API:** This is a system API.
