# UIEnvWindowAvoidAreaInfoVP

Describes [environment variable](docroot://ui/arkts-env-system-property.md) data types for window avoidance areas of different types. All types of window avoidance areas are measured in vp.

**Since:** 23

**System capability:** SystemCapability.Window.SessionManager


## cutout

```TypeScript
cutout: UIEnvAvoidAreaVP
```

Avoidance area whose [AvoidAreaType](arkts-arkui-window-avoidareatype-e.md) is **TYPE_CUTOUT** type, in vp.
