# LocalizedBarrierDirection

```TypeScript
declare enum LocalizedBarrierDirection
```

定义支持镜像模式的屏障线的方向。

**Since:** 12

**System capability:** SystemCapability.ArkUI.ArkUI.Full

## START

```TypeScript
START = 0
```

屏障在其所有[referencedId](arkts-localizedbarrierstyle-i.md#LocalizedBarrierStyle)的最左/右侧，LTR模式时为最左侧，RTL模式时为最右侧。

**Since:** 12

**Model restriction:** This API can be used only in the Stage model.

**Atomic service API:** This API can be used in atomic services.

**System capability:** SystemCapability.ArkUI.ArkUI.Full

## END

```TypeScript
END = 1
```

屏障在其所有[referencedId](arkts-localizedbarrierstyle-i.md#LocalizedBarrierStyle)的最左/右侧, LTR模式时为最右侧，RTL模式时为最左侧。

**Since:** 12

**Model restriction:** This API can be used only in the Stage model.

**Atomic service API:** This API can be used in atomic services.

**System capability:** SystemCapability.ArkUI.ArkUI.Full

## TOP

TOP = 2
