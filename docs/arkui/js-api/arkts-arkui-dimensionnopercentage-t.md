# DimensionNoPercentage

```TypeScript
declare type DimensionNoPercentage = PX | VP | FP | LPX | Resource
```

The percentage length union type is not supported.

**Since:** 11

**Model restriction:** This API can be used only in the stage model.

**Atomic service API:** This API can be used in atomic services since API version 12.

**System capability:** SystemCapability.ArkUI.ArkUI.Full

| Type | Description |
| --- | --- |
| PX | Length in px. |
| VP | Length in vp. |
| FP | Length in fp. |
| LPX | Length in lpx. |
| Resource | Resource reference type, which is used to set the value of a component attribute. |
