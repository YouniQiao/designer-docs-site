# TextClockController

TextClock容器组件的控制器，可以将该控制器绑定到TextClock组件，通过它控制文本时钟的启动与停止。一个TextClock组件仅支持绑定一个控制器。 ###### 导入对象 ```ts controller: TextClockController = new TextClockController() ```

**Since:** 8

**System capability:** SystemCapability.ArkUI.ArkUI.Full
