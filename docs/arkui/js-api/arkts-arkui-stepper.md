# Stepper

The **Stepper** component provides a step navigator, suitable for guiding users through a step-by-step task completion process. > **NOTE**

## Stepper

```TypeScript
Stepper(value?: { index?: number })
```

Creates a **Stepper** component. > **NOTE**
