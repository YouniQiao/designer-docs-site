# InputEventSubTypeMask

```TypeScript
declare enum InputEventSubTypeMask
```

输入事件子类型掩码枚举，用于标识不同类型的输入事件子类型。

**Since:** 26.0.0

**System capability:** SystemCapability.ArkUI.ArkUI.Full
