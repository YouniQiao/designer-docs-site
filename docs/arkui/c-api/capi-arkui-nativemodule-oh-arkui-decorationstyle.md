# OH_ArkUI_DecorationStyle

```c
typedef struct OH_ArkUI_DecorationStyle OH_ArkUI_DecorationStyle
```

## Overview

Defines a text decoration style.<br>        [OH_ArkUI_DecorationStyle_Create](capi-styled-string-h.md#oh_arkui_decorationstyle_create) can be used to create atext decoration style object.<br>        [OH_ArkUI_DecorationStyle_Destroy](capi-styled-string-h.md#oh_arkui_decorationstyle_destroy) can be used to destroy the textdecoration style object.<br>        After the object is created, the **OH_ArkUI_DecorationStyle_SetXXX** series APIscan be used to set the specific styles that take effect. For example, you can use[OH_ArkUI_DecorationStyle_SetTextDecorationType](capi-styled-string-h.md#oh_arkui_decorationstyle_settextdecorationtype) to set the decoration type.
