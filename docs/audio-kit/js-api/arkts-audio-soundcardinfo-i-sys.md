# SoundCardInfo (System API)

Describes sound card information.

**Since:** 26.0.0

**System capability:** SystemCapability.Multimedia.Audio.Core

**System API:** This is a system API.

## Modules to Import

```TypeScript
import { audio } from '@ohos.multimedia.audio';
```
