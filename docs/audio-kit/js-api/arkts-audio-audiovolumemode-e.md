# AudioVolumeMode

Enumerates the audio volume modes.

**Since:** 19

**System capability:** SystemCapability.Multimedia.Audio.Volume

## SYSTEM_GLOBAL

```TypeScript
SYSTEM_GLOBAL = 0
```

System-level volume (default mode).

**Since:** 19

**System capability:** SystemCapability.Multimedia.Audio.Volume

## APP_INDIVIDUAL

```TypeScript
APP_INDIVIDUAL = 1
```

Application-level volume.

**Since:** 19

**System capability:** SystemCapability.Multimedia.Audio.Volume
