# AsrVoiceMuteMode (System API)

ASR voice mute mode.

**Since:** 12

**System capability:** SystemCapability.Multimedia.Audio.Capturer

**System API:** This is a system API.

## OUTPUT_MUTE

```TypeScript
OUTPUT_MUTE = 0
```

Mute the local output stream.

**Since:** 12

**System capability:** SystemCapability.Multimedia.Audio.Capturer

**System API:** This is a system API.

## INPUT_MUTE

```TypeScript
INPUT_MUTE = 1
```

Mute the local MIC input stream.

**Since:** 12

**System capability:** SystemCapability.Multimedia.Audio.Capturer

**System API:** This is a system API.

## TTS_MUTE

```TypeScript
TTS_MUTE = 2
```

Send tts output stream to TX and mute the local output stream.

**Since:** 12

**System capability:** SystemCapability.Multimedia.Audio.Capturer

**System API:** This is a system API.

## CALL_MUTE

```TypeScript
CALL_MUTE = 3
```

Mute the voice call stream.

**Since:** 12

**System capability:** SystemCapability.Multimedia.Audio.Capturer

**System API:** This is a system API.

## OUTPUT_MUTE_EX

```TypeScript
OUTPUT_MUTE_EX = 4
```

Based on the OUTPUT_MUTE, send output stream to voice call record.

**Since:** 12

**System capability:** SystemCapability.Multimedia.Audio.Capturer

**System API:** This is a system API.
