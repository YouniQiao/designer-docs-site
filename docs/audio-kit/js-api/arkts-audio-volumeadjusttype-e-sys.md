# VolumeAdjustType (System API)

Enumerates volume adjustment types.

**Since:** 10

**System capability:** SystemCapability.Multimedia.Audio.Volume

**System API:** This is a system API.

## VOLUME_UP

```TypeScript
VOLUME_UP = 0
```

Adjust volume up.

**Since:** 10

**System capability:** SystemCapability.Multimedia.Audio.Volume

**System API:** This is a system API.

## VOLUME_DOWN

```TypeScript
VOLUME_DOWN = 1
```

Adjust volume down.

**Since:** 10

**System capability:** SystemCapability.Multimedia.Audio.Volume

**System API:** This is a system API.
