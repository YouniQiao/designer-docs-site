# DeviceBlockStatus

Enumerates the blocked statuses of audio devices.

**Since:** 13

**System capability:** SystemCapability.Multimedia.Audio.Device

## UNBLOCKED

```TypeScript
UNBLOCKED = 0
```

The audio device is not blocked.

**Since:** 13

**System capability:** SystemCapability.Multimedia.Audio.Device

## BLOCKED

```TypeScript
BLOCKED = 1
```

The audio device is blocked.

**Since:** 13

**System capability:** SystemCapability.Multimedia.Audio.Device
