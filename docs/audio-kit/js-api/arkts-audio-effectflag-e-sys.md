# EffectFlag (System API)

Enumerates audio effect flags.

**Since:** 18

**System capability:** SystemCapability.Multimedia.Audio.Core

**System API:** This is a system API.

## RENDER_EFFECT_FLAG

```TypeScript
RENDER_EFFECT_FLAG = 0
```

Audio render effect.

**Since:** 18

**System capability:** SystemCapability.Multimedia.Audio.Core

**System API:** This is a system API.
