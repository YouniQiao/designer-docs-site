# OH_AVScreenCapture

```c
typedef struct OH_AVScreenCapture OH_AVScreenCapture
```

## Overview

Defines all the asynchronous callback function pointers of an **OH_AVScreenCapture** instance. To ensure thenormal running of **OH_AVScreenCapture**, you must register the instance of this struct with the OH_AVScreenCapture** instance to process the information reported by the callback functions.

**Since**: 10
