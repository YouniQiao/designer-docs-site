# OH_Rect

```c
typedef struct OH_Rect {...} OH_Rect
```

## Overview

The struct describes the width, height, and image information of the rectangle used for screen capture.

**Since**: 10

**Related module**: [AVScreenCapture](capi-avscreencapture.md)

**Header file**: [native_avscreen_capture_base.h](capi-native-avscreen-capture-base-h.md)

### Member variables

| Name | Description |
| -- | -- |

| int32_t y | y-coordinate of screen recording |
| int32_t width | Width of screen recording, in px |
| int32_t height | Height of screen recording, in px |
