# AVScreenCaptureFillMode

```TypeScript
enum AVScreenCaptureFillMode
```

进行屏幕录制时视频填充模式的枚举。

**Since:** 18

**System capability:** SystemCapability.Multimedia.Media.AVScreenCapture

## PRESERVE_ASPECT_RATIO

```TypeScript
PRESERVE_ASPECT_RATIO = 0
```

保持与原始图像相同的宽高比例，即与物理屏幕宽高比例一致。

**Since:** 18

**System capability:** SystemCapability.Multimedia.Media.AVScreenCapture

## SCALE_TO_FILL

```TypeScript
SCALE_TO_FILL = 1
```

进行图像拉伸填充，适配设置的宽度和高度。

**Since:** 18

**System capability:** SystemCapability.Multimedia.Media.AVScreenCapture
