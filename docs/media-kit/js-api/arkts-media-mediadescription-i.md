# MediaDescription

Provides the container definition for media description key-value pairs.

**Since:** 8

**System capability:** SystemCapability.Multimedia.Media.Core

## Modules to Import

```TypeScript
import { media } from '@ohos.multimedia.media';
```

```TypeScript
[key: string]: Object
```
