# Name

**Since:** 7

**System capability:** SystemCapability.Applications.ContactsData

## Modules to Import

```TypeScript
import { contact } from '@kit.ContactsKit';
```

## hasName

```TypeScript
hasName?: boolean
```

联系人信息中是否包含姓名。true表示包含，false表示不包含。

**Type:** boolean

**Since:** 22

**Atomic service API:** This API can be used in atomic services.

**System capability:** SystemCapability.Applications.ContactsData
