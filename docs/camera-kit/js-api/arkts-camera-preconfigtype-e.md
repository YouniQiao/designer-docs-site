# PreconfigType

Enumerates the preconfigured resolution types.

**Since:** 12

**System capability:** SystemCapability.Multimedia.Camera.Core

## PRECONFIG_720P

```TypeScript

```

**Atomic service API:** This API can be used in atomic services since API version 19.
