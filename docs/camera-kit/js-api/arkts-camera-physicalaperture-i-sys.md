# PhysicalAperture

Describes the physical aperture object.

**Since:** 11

**System capability:** SystemCapability.Multimedia.Camera.Core

**System API:** This is a system API.

## Modules to Import

```TypeScript
import { camera } from '@kit.CameraKit';
```

## apertures

```TypeScript
apertures: Array<double>
```

Supported physical aperture.

**Type:** Array<double>

**Since:** 11

**Atomic service API:** From API version 24 this API can be used in atomic services.

**System capability:** SystemCapability.Multimedia.Camera.Core

**System API:** This is a system API.

## zoomRange

```TypeScript
zoomRange: ZoomRange
```

Zoom range of a given physical aperture.

**Type:** ZoomRange

**Since:** 11

**Atomic service API:** From API version 24 this API can be used in atomic services.

**System capability:** SystemCapability.Multimedia.Camera.Core

**System API:** This is a system API.
