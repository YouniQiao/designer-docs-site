# SmoothZoomInfo

Describes the smooth zoom information.

**Since:** 11

**System capability:** SystemCapability.Multimedia.Camera.Core

## Modules to Import

```TypeScript
import { camera } from '@kit.CameraKit';
```

## duration

```TypeScript
duration: int
```

Total duration of smooth zoom, in ms.

**Type:** int

**Since:** 11
