# MetadataObjectType

```TypeScript
enum MetadataObjectType
```

Enumerates the types of metadata objects used for camera detection.

**Since:** 10

**System capability:** SystemCapability.Multimedia.Camera.Core

```TypeScript
FACE_DETECTION = 0
```
