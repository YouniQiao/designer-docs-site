# DisplayTag

The pre-defined display tag by system.

**Since:** 11

**System capability:** SystemCapability.Multimedia.AVSession.Core

## TAG_AUDIO_VIVID

```TypeScript
TAG_AUDIO_VIVID = 1
```

Indicate the AUDIO VIVID property of current media resource.

**Since:** 11

**System capability:** SystemCapability.Multimedia.AVSession.Core
