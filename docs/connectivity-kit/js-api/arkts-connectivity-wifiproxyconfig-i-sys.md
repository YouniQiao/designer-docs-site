# WifiProxyConfig (System API)

Wi-Fi Proxy config.

**Since:** 10

**System capability:** SystemCapability.Communication.WiFi.STA

**System API:** This is a system API.

## Modules to Import

```TypeScript
import { wifiManager } from '@ohos.wifiManager';
```
