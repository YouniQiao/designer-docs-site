创建SSAP服务端实例。

**Since:** 26.0.0

**Required permissions:**

**System capability:** SystemCapability.Communication.NearLink.Base

**Return value:**

| Type | Description |
| --- | --- |
| Server | 返回一个SSAP服务端实例{@code Server}。 |
