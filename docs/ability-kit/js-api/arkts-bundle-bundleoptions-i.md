## userId

```TypeScript
userId?: number
```

用户ID。默认值：调用方所在用户，取值范围：大于等于0。

**Type:** number

**Since:** 7

**Deprecated since:** 9

**System capability:** SystemCapability.BundleManager.BundleFramework
