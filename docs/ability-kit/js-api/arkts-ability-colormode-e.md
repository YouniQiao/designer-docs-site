# ColorMode

> **NOTE**
>
> This API has been supported since API version 7 and deprecated since API version 9. No substitute is provided.

Enumerates the color modes of applications and widgets.

**Since:** 7

**Deprecated since:** 9

**Substitutes:** [null]

**System capability:** SystemCapability.BundleManager.BundleFramework

## AUTO_MODE

```TypeScript
AUTO_MODE = -1
```

Auto mode.

**Since:** 7

**Deprecated since:** 9

**Substitutes:** COLOR_MODE_NOT_SET

**System capability:** SystemCapability.BundleManager.BundleFramework

## DARK_MODE

```TypeScript
DARK_MODE = 0
```

Dark mode.

**Since:** 7

**Deprecated since:** 9

**Substitutes:** COLOR_MODE_DARK

**System capability:** SystemCapability.BundleManager.BundleFramework

## LIGHT_MODE

```TypeScript
LIGHT_MODE = 1
```

Light mode.

**Since:** 7

**Deprecated since:** 9

**System capability:** SystemCapability.BundleManager.BundleFramework
