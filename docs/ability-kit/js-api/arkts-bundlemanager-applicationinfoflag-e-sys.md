# ApplicationInfoFlag

```TypeScript
export enum ApplicationInfoFlag
```

标识应用和用户之间的各种状态类型。

**Since:** 12

**System capability:** SystemCapability.BundleManager.BundleFramework.Core

**System API:** This is a system API.

## FLAG_INSTALLED

```TypeScript
FLAG_INSTALLED = 0x00000001
```

表示指定用户安装应用的状态为已安装状态。

**Since:** 12

**System capability:** SystemCapability.BundleManager.BundleFramework.Core

**System API:** This is a system API.

## FLAG_OTHER_INSTALLED

```TypeScript
FLAG_OTHER_INSTALLED = 0x00000010
```

表示除指定用户外，其他用户的应用安装状态为已安装。

**Since:** 15

**System capability:** SystemCapability.BundleManager.BundleFramework.Core

**System API:** This is a system API.

## FLAG_PREINSTALLED_APP

```TypeScript
FLAG_PREINSTALLED_APP = 0x00000020
```

表示应用的预置属性为预置应用。

**Since:** 15

**System capability:** SystemCapability.BundleManager.BundleFramework.Core

**System API:** This is a system API.

## FLAG_PREINSTALLED_APP_UPDATE

```TypeScript
FLAG_PREINSTALLED_APP_UPDATE = 0x00000040
```

表示该预置应用的更新状态为已更新。

**Since:** 15

**System capability:** SystemCapability.BundleManager.BundleFramework.Core

**System API:** This is a system API.
