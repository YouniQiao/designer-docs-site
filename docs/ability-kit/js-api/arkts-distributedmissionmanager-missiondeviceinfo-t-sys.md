# MissionDeviceInfo

```TypeScript
export type MissionDeviceInfo = _MissionDeviceInfo
```

Defines the parameters required for registering a listener. It is used as an input parameter in [registerMissionListener](arkts-distributedmissionmanager-registermissionlistener-f-sys.md#registerMissionListener-2) .

**Since:** 10

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.Ability.AbilityRuntime.Mission

**System API:** This is a system API.

**Property type:** _MissionDeviceInfo
