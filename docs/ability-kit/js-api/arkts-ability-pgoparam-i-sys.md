# PGOParam (System API)

Defines the parameters of the PGO configuration file.

**Since:** 11

**System capability:** SystemCapability.BundleManager.BundleFramework.Core

**System API:** This is a system API.

## Modules to Import

```TypeScript
import { installer } from '@ohos.bundle.installer';
```
