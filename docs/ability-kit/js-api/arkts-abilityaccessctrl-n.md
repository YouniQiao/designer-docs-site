## Summary

### Functions

| Name | Description |
| --- | --- |
| [createAtManager](arkts-abilityaccessctrl-createatmanager-f.md#createAtManager-1) | 创建程序访问控制管理实例，用于权限校验、运行时权限申请、设置页授权引导和权限状态变化监听等场景。调用成功后返回AtManager实例，可用于后续的权限管理操作。 |
