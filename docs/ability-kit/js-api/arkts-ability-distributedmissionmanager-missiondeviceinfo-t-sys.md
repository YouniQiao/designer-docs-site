# MissionDeviceInfo (System API)

```TypeScript
export type MissionDeviceInfo = _MissionDeviceInfo
```

Defines the parameters required for registering a listener. It is used as an input parameter in [registerMissionListener](arkts-ability-distributedmissionmanager-registermissionlistener-f-sys.md#registermissionlistener-2).

**Since:** 10

**Model restriction:** This API can be used only in the stage model.
