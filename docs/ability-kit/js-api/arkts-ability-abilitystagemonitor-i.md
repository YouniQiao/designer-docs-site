# AbilityStageMonitor

The module provides the capability to listen for a specified [AbilityStage](arkts-ability-app-ability-abilitystage-abilitystage-c.md) object. You can use AbilityStageMonitor as an input parameter of [abilityDelegator.waitAbilityStageMonitor](arkts-ability-abilitydelegator-i.md#waitabilitystagemonitor-1)to register a listener.

**Since:** 9
