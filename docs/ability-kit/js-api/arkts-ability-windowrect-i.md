# WindowRect

Defines the location attributes of a modal dialog box.

**Since:** 10

**System capability:** SystemCapability.Ability.AbilityRuntime.Core

## Modules to Import

```TypeScript
import { dialogRequest } from '@ohos.app.ability.dialogRequest';
```

## height

```TypeScript
height: number
```

Height of the dialog box, in px.

**Type:** number

**Since:** 10

**Model restriction:** This API can be used only in the stage model.

**System capability:** SystemCapability.Ability.AbilityRuntime.Core

## left

```TypeScript
left: number
```

X-coordinate of the upper left corner of the dialog box.

**Type:** number

**Since:** 10

**Model restriction:** This API can be used only in the stage model.

**System capability:** SystemCapability.Ability.AbilityRuntime.Core

## top

```TypeScript
top: number
```

Y-coordinate of the upper left corner of the dialog box.

**Type:** number

**Since:** 10

**Model restriction:** This API can be used only in the stage model.

**System capability:** SystemCapability.Ability.AbilityRuntime.Core

## width

```TypeScript
width: number
```

Width of the dialog box, in px.

**Type:** number

**Since:** 10

**Model restriction:** This API can be used only in the stage model.

**System capability:** SystemCapability.Ability.AbilityRuntime.Core
