# OperationType

```TypeScript
export enum OperationType
```

表示WantAgent支持的操作类型。

**Since:** 9

**System capability:** SystemCapability.Ability.AbilityRuntime.Core

## UNKNOWN_TYPE

```TypeScript
UNKNOWN_TYPE = 0
```

不识别的类型。

**Since:** 9

**Atomic service API:** From API version 12 this API can be used in atomic services.

**System capability:** SystemCapability.Ability.AbilityRuntime.Core

## START_ABILITY

```TypeScript
START_ABILITY
```

开启一个有页面的Ability。

**Since:** 9

**Atomic service API:** From API version 12 this API can be used in atomic services.

**System capability:** SystemCapability.Ability.AbilityRuntime.Core

## START_ABILITIES

```TypeScript
START_ABILITIES
```

开启多个有页面的Ability。

**Since:** 9

**Atomic service API:** From API version 12 this API can be used in atomic services.

**System capability:** SystemCapability.Ability.AbilityRuntime.Core

## START_SERVICE

```TypeScript
START_SERVICE
```

开启一个无页面的Ability（仅在FA模型下生效）。

**Since:** 9

**Atomic service API:** From API version 12 this API can be used in atomic services.

**System capability:** SystemCapability.Ability.AbilityRuntime.Core

## SEND_COMMON_EVENT

```TypeScript
SEND_COMMON_EVENT
```

发送一个公共事件。

**Since:** 9

**Atomic service API:** From API version 12 this API can be used in atomic services.

**System capability:** SystemCapability.Ability.AbilityRuntime.Core

## START_SERVICE_EXTENSION

```TypeScript
START_SERVICE_EXTENSION = 6
```

开启一个ServiceExtension。

**Since:** 12

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.Ability.AbilityRuntime.Core

**System API:** This is a system API.
