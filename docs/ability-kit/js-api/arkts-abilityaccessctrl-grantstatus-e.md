# GrantStatus

```TypeScript
export enum GrantStatus
```

表示授权状态的枚举。

**Since:** 8

**System capability:** SystemCapability.Security.AccessToken

## PERMISSION_DENIED

```TypeScript
PERMISSION_DENIED = -1
```

表示未授权。

**Since:** 8

**Atomic service API:** From API version 11 this API can be used in atomic services.

**System capability:** SystemCapability.Security.AccessToken

## PERMISSION_GRANTED

```TypeScript
PERMISSION_GRANTED = 0
```

表示已授权。

**Since:** 8

**Atomic service API:** From API version 11 this API can be used in atomic services.

**System capability:** SystemCapability.Security.AccessToken
