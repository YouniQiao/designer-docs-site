# AbilityStageMonitor

本模块提供监听指定[AbilityStage]./../@ohos.app.ability.AbilityStage:AbilityStage对象的能力。开发者可以将AbilityStageMonitor作为 [abilityDelegator.waitAbilityStageMonitor](arkts-abilitydelegator-i.md#waitAbilityStageMonitor) 的入参来注册监听。

**Since:** 9

**System capability:** SystemCapability.Ability.AbilityRuntime.Core

## srcEntrance

```TypeScript
srcEntrance: string
```

被监听的AbilityStage的源路径。

**Type:** string

**Since:** 9

**Atomic service API:** From API version 11 this API can be used in atomic services.

**System capability:** SystemCapability.Ability.AbilityRuntime.Core

## moduleName

```TypeScript
moduleName: string
```

被监听的AbilityStage的模块名。

**Type:** string

**Since:** 9

**Atomic service API:** From API version 11 this API can be used in atomic services.

**System capability:** SystemCapability.Ability.AbilityRuntime.Core
