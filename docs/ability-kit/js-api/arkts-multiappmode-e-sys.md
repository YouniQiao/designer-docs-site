# MultiAppMode

```TypeScript
export enum MultiAppMode
```

定义应用是否支持多开模式。

**Since:** 12

**System capability:** SystemCapability.Ability.AbilityRuntime.AbilityCore

**System API:** This is a system API.

## NOT_SUPPORTED

```TypeScript
NOT_SUPPORTED = 0
```

应用不支持多开模式。

**Since:** 12

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.Ability.AbilityRuntime.AbilityCore

**System API:** This is a system API.

## MULTI_INSTANCE

```TypeScript
MULTI_INSTANCE = 1
```

应用支持多实例模式。 **说明：** 只支持2in1设备。

**Since:** 14

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.Ability.AbilityRuntime.AbilityCore

**System API:** This is a system API.

## APP_CLONE

```TypeScript
APP_CLONE = 2
```

应用支持分身模式。

**Since:** 12

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.Ability.AbilityRuntime.AbilityCore

**System API:** This is a system API.
