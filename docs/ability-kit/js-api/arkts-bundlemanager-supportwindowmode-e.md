# SupportWindowMode

```TypeScript
export enum SupportWindowMode
```

标识该组件所支持的窗口模式。

**Since:** 9

**System capability:** SystemCapability.BundleManager.BundleFramework.Core

## FULL_SCREEN

```TypeScript
FULL_SCREEN = 0
```

窗口支持全屏显示。

**Since:** 9

**Atomic service API:** From API version 11 this API can be used in atomic services.

**System capability:** SystemCapability.BundleManager.BundleFramework.Core

## SPLIT

```TypeScript
SPLIT = 1
```

窗口支持分屏显示。

**Since:** 9

**Atomic service API:** From API version 11 this API can be used in atomic services.

**System capability:** SystemCapability.BundleManager.BundleFramework.Core

## FLOATING

```TypeScript
FLOATING = 2
```

支持窗口化显示，即显示悬浮窗口。

**Since:** 9

**Atomic service API:** From API version 11 this API can be used in atomic services.

**System capability:** SystemCapability.BundleManager.BundleFramework.Core
