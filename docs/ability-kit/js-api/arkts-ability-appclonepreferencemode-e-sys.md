# AppClonePreferenceMode (System API)

Enumerates the application clone preference modes.

**Since:** 26.1.0

**System capability:** SystemCapability.BundleManager.BundleFramework.Core

**System API:** This is a system API.

## ALWAYS_ASK

```TypeScript
ALWAYS_ASK = 0
```

Always prompts the user to select an application.

**Since:** 26.1.0

**Model restriction:** This API can be used only in the stage model.

**System capability:** SystemCapability.BundleManager.BundleFramework.Core

**System API:** This is a system API.

## MAIN_APP

```TypeScript
MAIN_APP = 1
```

Uses the main application by default.

**Since:** 26.1.0

**Model restriction:** This API can be used only in the stage model.

**System capability:** SystemCapability.BundleManager.BundleFramework.Core

**System API:** This is a system API.

## CLONE_APP

```TypeScript
CLONE_APP = 2
```

Uses the application clone by default.

**Since:** 26.1.0

**Model restriction:** This API can be used only in the stage model.

**System capability:** SystemCapability.BundleManager.BundleFramework.Core

**System API:** This is a system API.
