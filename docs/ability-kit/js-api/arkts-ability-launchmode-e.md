# LaunchMode

> **NOTE**
>
> This API has been supported since API version 7 and deprecated since API version 9. You are advised to use
> [bundleManager.LaunchType](arkts-ability-launchtype-e.md) instead.

Enumerates the ability launch modes.

**Since:** 7

**Deprecated since:** 9

**Substitutes:** [LaunchType](arkts-ability-launchtype-e.md)

**System capability:** SystemCapability.BundleManager.BundleFramework

## SINGLETON

```TypeScript
SINGLETON = 0
```

The ability has only one instance.

**Since:** 7

**Deprecated since:** 9

**Substitutes:** SINGLETON

**System capability:** SystemCapability.BundleManager.BundleFramework

## STANDARD

```TypeScript
STANDARD = 1
```

The ability can have multiple instances.

**Since:** 7

**Deprecated since:** 9

**Substitutes:** MULTITON

**System capability:** SystemCapability.BundleManager.BundleFramework
