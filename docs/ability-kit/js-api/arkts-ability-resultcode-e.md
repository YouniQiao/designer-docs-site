# ResultCode

Enumerates the result codes of the request for the modal dialog box.

**Since:** 9

**System capability:** SystemCapability.Ability.AbilityRuntime.Core

## RESULT_OK

```TypeScript
RESULT_OK = 0
```

The request succeeds.

**Since:** 9

**System capability:** SystemCapability.Ability.AbilityRuntime.Core

## RESULT_CANCEL

```TypeScript
RESULT_CANCEL = 1
```

The request fails.

**Since:** 9

**System capability:** SystemCapability.Ability.AbilityRuntime.Core
