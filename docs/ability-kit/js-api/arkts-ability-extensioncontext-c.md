# ExtensionContext

ExtensionContext provides the context environment for an [ExtensionAbility](arkts-ability-extensionability-c.md). It inherits from [Context](../../../../reference/apis-ability-kit/js-apis-inner-application-context.md#context). This module provides APIs for accessing resources of a specific [ExtensionAbility](arkts-ability-extensionability-c.md).

**Inheritance/Implementation:** ExtensionContext extends [Context](arkts-ability-context-t.md)

**Since:** 9

**System capability:** SystemCapability.Ability.AbilityRuntime.Core

## config

```TypeScript
config: Configuration
```

Indicates configuration information.

**Type:** Configuration

**Since:** 9

**Model restriction:** This API can be used only in the stage model.

**Atomic service API:** This API can be used in atomic services since API version 11.

**System capability:** SystemCapability.Ability.AbilityRuntime.Core
