# ExtensionContext

ExtensionContext provides the context environment for an [ExtensionAbility](arkts-ability-app-ability-extensionability-extensionability-c.md). It inherits from [Context](../../../../reference/apis-ability-kit/js-apis-inner-application-context.md#context).This module provides APIs for accessing resources of a specific [ExtensionAbility](arkts-ability-app-ability-extensionability-extensionability-c.md).

**Inheritance/Implementation:** ExtensionContext extends [Context](arkts-ability-context-t.md)

**Since:** 9
