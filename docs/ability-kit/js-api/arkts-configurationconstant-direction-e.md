# Direction

```TypeScript
export enum Direction
```

表示屏幕方向的枚举，用于[Configuration.direction](arkts-configuration-i.md#Configuration)字段。开发者可以使用这些预置枚举设置或获取系统/应 用的显示方向。

**Since:** 9

**System capability:** SystemCapability.Ability.AbilityBase

## DIRECTION_NOT_SET

```TypeScript
DIRECTION_NOT_SET = -1
```

表示未设置方向。

**Since:** 9

**Atomic service API:** From API version 11 this API can be used in atomic services.

**System capability:** SystemCapability.Ability.AbilityBase

## DIRECTION_VERTICAL

```TypeScript
DIRECTION_VERTICAL = 0
```

表示垂直方向。

**Since:** 9

**Atomic service API:** From API version 11 this API can be used in atomic services.

**System capability:** SystemCapability.Ability.AbilityBase

## DIRECTION_HORIZONTAL

```TypeScript
DIRECTION_HORIZONTAL = 1
```

表示水平方向。

**Since:** 9

**Atomic service API:** From API version 11 this API can be used in atomic services.

**System capability:** SystemCapability.Ability.AbilityBase
