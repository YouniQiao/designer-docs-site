# connectOptions

## Summary

### Interfaces

| Name | Description |
| --- | --- |
| [ConnectOptions](arkts-connectoptions-i.md) | 在连接指定的后台服务时作为入参，用于接收连接过程中的状态变化，如作为 [connectServiceExtensionAbility]{@link ./application/UIAbilityContext:UIAbilityContext.connectServiceExtensionAbility} 的入参，连接指定的ServiceExtensionAbility。 |

### Types

| Name | Description |
| --- | --- |
| [OnConnectFn](arkts-onconnectfn-t.md) | 与指定的后台服务成功建立连接时，会触发该回调。 |
| [OnDisconnectFn](arkts-ondisconnectfn-t.md) | 与指定的后台服务成功断开连接时，会触发该回调。 |
| [OnFailedFn](arkts-onfailedfn-t.md) | 与指定的后台服务建立连接失败时，会触发该回调，返回连接失败的错误码。 |
