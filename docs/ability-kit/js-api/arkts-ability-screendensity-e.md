# ScreenDensity

Enumerates the pixel densities of the screen, which are used in the [Configuration.screenDensity](arkts-ability-configuration-i.md#configuration) field. You can use these predefined enumerated values to set or obtain the pixel density of the screen. The font size is positively correlated with the screen pixel density. By monitoring changes in the screen pixel density, you can detect adjustments in the font size. Typically, for the same physical size, the higher the screen pixel density, the larger the font display effect.

**Since:** 9

**System capability:** SystemCapability.Ability.AbilityBase

## SCREEN_DENSITY_NOT_SET

```TypeScript
SCREEN_DENSITY_NOT_SET = 0
```

The screen pixel density is not set.

**Since:** 9

**Atomic service API:** This API can be used in atomic services since API version 11.

**System capability:** SystemCapability.Ability.AbilityBase
