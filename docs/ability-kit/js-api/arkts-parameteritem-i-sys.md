# ParameterItem

快捷方式配置信息中的自定义数据。由开发者自行决定传入的键值对，可以通过key值获取对应的value值。

**Since:** 12

**System capability:** SystemCapability.BundleManager.BundleFramework.Launcher

**System API:** This is a system API.

## value

```TypeScript
value: string
```

快捷方式配置信息中的自定义数据的值。

**Type:** string

**Since:** 12

**System capability:** SystemCapability.BundleManager.BundleFramework.Launcher

**System API:** This is a system API.

## key

```TypeScript
key: string
```

快捷方式配置信息中的自定义数据的键。

**Type:** string

**Since:** 12

**System capability:** SystemCapability.BundleManager.BundleFramework.Launcher

**System API:** This is a system API.
