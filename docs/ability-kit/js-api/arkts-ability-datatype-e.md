# DataType

Enumerates the types of data under lock screen.

**Since:** 12

**System capability:** SystemCapability.Security.ScreenLockFileManager

## MEDIA_DATA

```TypeScript
MEDIA_DATA = 0x00000001
```

Media photo and video data.

**Since:** 12

**System capability:** SystemCapability.Security.ScreenLockFileManager

## ALL_DATA

```TypeScript
ALL_DATA = 0xffffffff
```

All data type.

**Since:** 12

**System capability:** SystemCapability.Security.ScreenLockFileManager
