# want

## Summary

### Interfaces

| Name | Description |
| --- | --- |
| [Want](arkts-ability-want-i.md) | Want is a carrier for information transfer between objects (application components). Want can be used as a parameter of [startAbility](arkts-ability-uiabilitycontext-c.md#startability-1) to specify a startup target and information that needs to be carried during startup, for example, **bundleName** and **abilityName**, which respectively indicate the bundle name of the target ability and the ability name in the bundle. When ability A needs to start ability B and transfer some data to ability B, it can use Want a carrier to transfer the data. |
