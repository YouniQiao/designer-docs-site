# WindowMode

Enumerates the window modes in which a UIAbility can be displayed at startup. It can be used in [startAbility](arkts-ability-uiabilitycontext-c.md#startability-1).

**Since:** 12

**System capability:** SystemCapability.Ability.AbilityRuntime.Core

## WINDOW_MODE_UNDEFINED

```TypeScript
WINDOW_MODE_UNDEFINED = 0
```

Undefined window mode.

**Since:** 12

**Model restriction:** This API can be used only in the stage model.

## WINDOW_MODE_FLOATING

```TypeScript
WINDOW_MODE_FLOATING = 102
```

The ability is displayed in a floating window.

**Since:** 12

**Model restriction:** This API can be used only in the stage model.
