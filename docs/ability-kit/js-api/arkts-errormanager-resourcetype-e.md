# ResourceType

```TypeScript
export enum ResourceType
```

**Since:** 24

**System capability:** SystemCapability.Ability.AbilityRuntime.AbilityCore

## PSS_MEMORY

```TypeScript
PSS_MEMORY = 1
```

表示应用当前超基线的资源是PSS的内存。

**Since:** 24

**Model restriction:** This API can be used only in the Stage model.
