# BaseContext

BaseContext抽象类用于表示继承的子类Context是Stage模型还是FA模型，是所有Context类型的父类。

**Since:** 8

**System capability:** SystemCapability.Ability.AbilityRuntime.Core

## stageMode

```TypeScript
stageMode: boolean
```

表示是否Stage模型。<br>true：[Stage模型](docroot://application-models/ability-terminology.md#stage模型)。<br>false： [FA模型](docroot://application-models/ability-terminology.md#fa模型)。

**Type:** boolean

**Since:** 8

**Atomic service API:** From API version 11 this API can be used in atomic services.

**System capability:** SystemCapability.Ability.AbilityRuntime.Core
