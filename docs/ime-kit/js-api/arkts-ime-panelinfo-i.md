# PanelInfo

Describes the attributes of the input method panel.

**Since:** 10

**System capability:** SystemCapability.MiscServices.InputMethodFramework

## Modules to Import

```TypeScript
import { inputMethodEngine } from '@ohos.inputMethodEngine';
```


**Type:** PanelFlag

**System capability:** SystemCapability.MiscServices.InputMethodFramework
