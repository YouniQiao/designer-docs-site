# ExtendAction

Describes the type of the extended edit action on the text box.

**Since:** 10

**System capability:** SystemCapability.MiscServices.InputMethodFramework

## SELECT_ALL

```TypeScript
SELECT_ALL = 0
```

**System capability:** SystemCapability.MiscServices.InputMethodFramework
