# SystemPanelStatus (System API)

System panel status.

**Since:** 26.0.0

**System capability:** SystemCapability.MiscServices.InputMethodFramework

**System API:** This is a system API.

## Modules to Import

```TypeScript
import { inputMethodSystemPanelManager } from '@ohos.inputMethodSystemPanelManager';
```

## inputType

```TypeScript
inputType: InputMethodInputType
```

The input type of the input method.

**Type:** InputMethodInputType

**Since:** 26.0.0

**Model restriction:** This API can be used only in the stage model.

**System capability:** SystemCapability.MiscServices.InputMethodFramework

**System API:** This is a system API.

## isPanelRaised

```TypeScript
isPanelRaised: boolean
```

Whether the system panel needs to be raised.

**Type:** boolean

**Since:** 26.0.0

**Model restriction:** This API can be used only in the stage model.

**System capability:** SystemCapability.MiscServices.InputMethodFramework

**System API:** This is a system API.
