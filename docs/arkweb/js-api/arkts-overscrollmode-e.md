# OverScrollMode

```TypeScript
declare enum OverScrollMode
```

设置Web的过滚动模式为关闭或开启。

**Since:** 11

**System capability:** SystemCapability.Web.Webview.Core

## NEVER

```TypeScript
NEVER = 0
```

Web过滚动模式关闭。

**Since:** 11

**Atomic service API:** This API can be used in atomic services.

**System capability:** SystemCapability.Web.Webview.Core

## ALWAYS

```TypeScript
ALWAYS = 1
```

Web过滚动模式开启。

**Since:** 11

**Atomic service API:** This API can be used in atomic services.

**System capability:** SystemCapability.Web.Webview.Core
