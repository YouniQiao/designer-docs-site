# PdfData

Implements the output data stream class of **createPdf()**. > **NOTE** > > - The sample effect is subject to the actual device. > > - When a PDF file is generated on a web page, a data stream is returned, which is encapsulated by the **PdfData** > class.

**Since:** 14

**System capability:** SystemCapability.Web.Webview.Core

## Modules to Import

```TypeScript
import { webview } from '@kit.ArkWeb';
```

## pdfArrayBuffer

```TypeScript
pdfArrayBuffer(): Uint8Array
```

Obtains the data stream generated by a web page. For details about the sample code, see [createPdf](arkts-arkweb-webviewcontroller-c.md#createpdf-1).

**Since:** 14

**Atomic service API:** This API can be used in atomic services since API version 14.

**System capability:** SystemCapability.Web.Webview.Core

**Return value:**

| Type | Description |
| --- | --- |
| Uint8Array | Data stream. |
