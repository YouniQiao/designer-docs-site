# NestedScrollOptionsExt

Implements a **NestedScrollOptionsExt** object to set up, down, left, and right nested scrolling options.

**Since:** 14

**System capability:** SystemCapability.Web.Webview.Core

## scrollDown

```TypeScript
scrollDown?: NestedScrollMode
```

Nested scrolling options when the component scrolls down. Default value: **NestedScrollMode.SELF_FIRST**.

**Type:** NestedScrollMode

**Since:** 14

**System capability:** SystemCapability.Web.Webview.Core
