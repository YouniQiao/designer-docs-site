# OnAudioStateChangedEvent

定义网页上的音频播放状态发生改变时的回调函数。

**Since:** 12

**System capability:** SystemCapability.Web.Webview.Core

## playing

```TypeScript
playing: boolean
```

当前页面的音频播放状态，true表示正在播放，false表示未播放。

**Type:** boolean

**Since:** 12

**Atomic service API:** This API can be used in atomic services.

**System capability:** SystemCapability.Web.Webview.Core
