# NativeEmbedParamDataInfo

提供同层渲染object标签内嵌param元素变化时同层标签的详细信息。

**Since:** 21

**System capability:** SystemCapability.Web.Webview.Core

## objectAttributeId

```TypeScript
objectAttributeId?: string
```

同层标签的id信息。

**Type:** string

**Since:** 21

**System capability:** SystemCapability.Web.Webview.Core

## paramItems

```TypeScript
paramItems?: Array<NativeEmbedParamItem>
```

发生变化的param元素的详细信息，包括每一个param元素的状态变化类型、id、参数名称和参数值。

**Type:** Array<NativeEmbedParamItem>

**Since:** 21

**System capability:** SystemCapability.Web.Webview.Core

## embedId

```TypeScript
embedId: string
```

同层标签的唯一id。

**Type:** string

**Since:** 21

**System capability:** SystemCapability.Web.Webview.Core
