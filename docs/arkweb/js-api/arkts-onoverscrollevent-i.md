# OnOverScrollEvent

定义网页过度滚动时触发的回调。

**Since:** 12

**System capability:** SystemCapability.Web.Webview.Core

## yOffset

```TypeScript
yOffset: number
```

以网页最上端为基准，竖直过度滚动的偏移量。 单位：vp。

**Type:** number

**Since:** 12

**Atomic service API:** This API can be used in atomic services.

**System capability:** SystemCapability.Web.Webview.Core

## xOffset

```TypeScript
xOffset: number
```

以网页最左端为基准，水平过度滚动的偏移量。 单位：vp。

**Type:** number

**Since:** 12

**Atomic service API:** This API can be used in atomic services.

**System capability:** SystemCapability.Web.Webview.Core
