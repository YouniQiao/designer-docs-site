# WebKeyboardAppearanceMode

```TypeScript
declare enum WebKeyboardAppearanceMode
```

提供给{@链接键盘外观}的枚举类型，用于设置Web键盘外观模式。

**Since:** 26.0.0

**System capability:** SystemCapability.Web.Webview.Core

## NONE_IMMERSIVE

```TypeScript
NONE_IMMERSIVE = 0
```

默认皮肤模式，没有沉浸式风格。

**Since:** 26.0.0

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.Web.Webview.Core

## IMMERSIVE

```TypeScript
IMMERSIVE = 1
```

没有沉浸式风格

**Since:** 26.0.0

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.Web.Webview.Core

## LIGHT_IMMERSIVE

```TypeScript
LIGHT_IMMERSIVE = 2
```

浅色沉浸式风格。

**Since:** 26.0.0

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.Web.Webview.Core

## DARK_IMMERSIVE

```TypeScript
DARK_IMMERSIVE = 3
```

深色沉浸式风格。

**Since:** 26.0.0

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.Web.Webview.Core
