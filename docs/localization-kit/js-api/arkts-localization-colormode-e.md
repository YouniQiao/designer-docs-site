# ColorMode

Defines the color mode of the current device.

**Since:** 12

**System capability:** SystemCapability.Global.ResourceManager

## DARK

```TypeScript
DARK = 0
```

Dark mode.

**Since:** 12

**Atomic service API:** This API can be used in atomic services since API version 12.

**System capability:** SystemCapability.Global.ResourceManager

```TypeScript
LIGHT = 1
```
