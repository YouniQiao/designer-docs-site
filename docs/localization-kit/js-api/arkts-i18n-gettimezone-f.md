# getTimeZone

## getTimeZone

```TypeScript
export function getTimeZone(zoneID?: string): TimeZone
```

Obtains the **TimeZone** object corresponding to the specified time zone ID.

| zoneID | string | No |  |

**Return value:**

| Type | Description |
