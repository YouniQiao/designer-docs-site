# DataMigrationProgress (System API)

Describes the data migration progress.

**Since:** 23

**System capability:** SystemCapability.Global.FontManager

**System API:** This is a system API.

## Modules to Import

```TypeScript
import { fontManager } from '@ohos.fontManager';
```

## progressPercentage

```TypeScript
progressPercentage: number
```

Data migration progress, in percentage. The value ranges from 0 to 100. The value range is all integers.

**Type:** number

**Since:** 23

**System capability:** SystemCapability.Global.FontManager

**System API:** This is a system API.

## timeRemaining

```TypeScript
timeRemaining: number
```

Estimated remaining time, in seconds. The value should be an integer.

**Type:** number

**Since:** 23

**System capability:** SystemCapability.Global.FontManager

**System API:** This is a system API.
