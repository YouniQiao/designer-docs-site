# RenderingMode (System API)

Enumerates the rendering modes supported by the widget.

**Since:** 18

**System capability:** SystemCapability.Ability.Form

**System API:** This is a system API.

## AUTO_COLOR

```TypeScript

```

Auto mode.

**System capability:** SystemCapability.Ability.Form

**System API:** This is a system API.
