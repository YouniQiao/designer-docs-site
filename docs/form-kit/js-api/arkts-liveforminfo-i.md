# LiveFormInfo

Provides information about a live form.

**Since:** 20

**System capability:** SystemCapability.Ability.Form

## Modules to Import

```TypeScript
import { LiveFormInfo } from '@kit.FormKit';
```

## formId

```TypeScript
formId: string
```

The form id of the live form.

**Type:** string

**Since:** 20

**Model restriction:** This API can be used only in the Stage model.

**Atomic service API:** This API can be used in atomic services.

**System capability:** SystemCapability.Ability.Form

## rect

```TypeScript
rect: formInfo.Rect
```

The live form display area.

**Type:** formInfo.Rect

**Since:** 20

**Model restriction:** This API can be used only in the Stage model.

**Atomic service API:** This API can be used in atomic services.

**System capability:** SystemCapability.Ability.Form

## borderRadius

```TypeScript
borderRadius: double
```

The form border radius. Unit: vp, The value must be greater than or equal to 0.

**Type:** double

**Since:** 20

**Model restriction:** This API can be used only in the Stage model.

**Atomic service API:** This API can be used in atomic services.

**System capability:** SystemCapability.Ability.Form
