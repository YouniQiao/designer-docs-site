# FormUsageState (System API)

Enumerates the usage statuses of a widget.

**Since:** 11

**System capability:** SystemCapability.Ability.Form

**System API:** This is a system API.

## USED

```TypeScript
USED = 0
```

The widget is in use.

**Since:** 11

**System capability:** SystemCapability.Ability.Form

**System API:** This is a system API.

## UNUSED

```TypeScript
UNUSED = 1
```

The widget is not in use.

**Since:** 11

**System capability:** SystemCapability.Ability.Form

**System API:** This is a system API.
