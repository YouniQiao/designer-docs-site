# Configuration

定义环境变化信息。Configuration是接口定义，仅做字段声明。

**Since:** 8

**Deprecated since:** 9

**Substitute:** ohos.app.ability.Configuration/Configuration

**System capability:** SystemCapability.Ability.AbilityBase

## colorMode

```TypeScript
colorMode?: ConfigurationConstant.ColorMode
```

表示深浅色模式，取值范围：浅色模式（COLOR_MODE_LIGHT），深色模式（COLOR_MODE_DARK）。默认为浅色。

**Type:** ConfigurationConstant.ColorMode

**Since:** 8

**Deprecated since:** 9

**Substitute:** ohos.app.ability.Configuration/Configuration#colorMode

**System capability:** SystemCapability.Ability.AbilityBase

## language

```TypeScript
language?: string
```

表示应用程序的当前语言。例如：zh。

**Type:** string

**Since:** 8

**Deprecated since:** 9

**Substitute:** ohos.app.ability.Configuration/Configuration#language

**System capability:** SystemCapability.Ability.AbilityBase
