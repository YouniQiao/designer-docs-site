## actualDelayTime

```TypeScript
actualDelayTime: number
```

应用的实际挂起延迟时间，单位：ms。 一般情况下默认值为180000，低电量（依据系统低电量广播）时默认值为60000。

**Type:** number

**Since:** 7

**Deprecated since:** 9
