# ReadOut

仅用于read方法，获取文件的读取结果。

**Since:** 6

**Deprecated since:** 9

**System capability:** SystemCapability.FileManagement.File.FileIO

## offset

```TypeScript
offset: number
```

读取数据相对于缓冲区首地址的偏移，单位为Byte。

**Type:** number

**Since:** 6

**Deprecated since:** 9

**System capability:** SystemCapability.FileManagement.File.FileIO

## bytesRead

```TypeScript
bytesRead: number
```

实际读取长度，单位为Byte。

**Type:** number

**Since:** 6

**Deprecated since:** 9

**System capability:** SystemCapability.FileManagement.File.FileIO

## buffer

```TypeScript
buffer: ArrayBuffer
```

保存读取数据的缓冲区。

**Type:** ArrayBuffer

**Since:** 6

**Deprecated since:** 9

**System capability:** SystemCapability.FileManagement.File.FileIO
