# TouchGuideType

```TypeScript
type TouchGuideType = 'touchBegin' | 'touchEnd'
```

Enumerates the touch guide event types.

**Since:** 9

**System capability:** SystemCapability.BarrierFree.Accessibility.Core

| Type | Description |
| --- | --- |
| 'touchBegin' | Start of touch. |
| 'touchEnd' | End of touch. |
