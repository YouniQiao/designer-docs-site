# RepeatClickInterval

```TypeScript
type RepeatClickInterval = 'Shortest' | 'Short' | 'Medium' | 'Long' | 'Longest'
```

Defines the interval between repeated clicks. The configuration of **RepeatClickInterval** takes effect when [ignoreRepeatClick](arkts-config-con-sys.md#ignoreRepeatClick) is set to **true**; the normal type is used when [ignoreRepeatClick](arkts-config-con-sys.md#ignoreRepeatClick) is set to **false**.

**Since:** 11

**System capability:** SystemCapability.BarrierFree.Accessibility.Core

**System API:** This is a system API.

| Type | Description |
| --- | --- |
| 'Shortest' | Shortest. |
| 'Short' | Short. |
| 'Medium' | Medium. |
| 'Long' | Long. |
| 'Longest' | Longest. |
