# WindowUpdateType

```TypeScript
type WindowUpdateType = 'add' | 'remove' | 'bounds' | 'active' | 'focus'
```

Window update type.

**Since:** 7

**Atomic service API:** From API version 23 this API can be used in atomic services.

**Widget capability:** From API version 23 this API can be used in ArkTS widgets.

**System capability:** SystemCapability.BarrierFree.Accessibility.Core

| Type | Description |
| --- | --- |
| 'add' | Window adding. |
| 'remove' | Window deletion. |
| 'bounds' | Window boundary change. |
| 'active' | Window activity change. |
| 'focus' | Window focus change. |
