# DataBlob

二进制数据的封装接口，核心字段data为Uint8Array类型。

**Since:** 9

**System capability:** SystemCapability.Security.Cert

## Modules to Import

```TypeScript
import { cert } from '@kit.DeviceCertificateKit';
```

## data

```TypeScript
data: Uint8Array
```

表示数据内容。

**Type:** Uint8Array

**Since:** 9

**Atomic service API:** From API version 12 this API can be used in atomic services.

**System capability:** SystemCapability.Security.Cert
