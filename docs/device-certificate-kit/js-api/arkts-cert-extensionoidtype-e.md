# ExtensionOidType

```TypeScript
enum ExtensionOidType
```

表示获取扩展域中对象标识符类型的枚举。

**Since:** 10

**System capability:** SystemCapability.Security.Cert

## EXTENSION_OID_TYPE_ALL

```TypeScript
EXTENSION_OID_TYPE_ALL = 0
```

表示获取扩展域中所有的对象标识符。

**Since:** 10

**Atomic service API:** From API version 12 this API can be used in atomic services.

**System capability:** SystemCapability.Security.Cert

## EXTENSION_OID_TYPE_CRITICAL

```TypeScript
EXTENSION_OID_TYPE_CRITICAL = 1
```

表示获取扩展域中critical为true的对象标识符。

**Since:** 10

**Atomic service API:** From API version 12 this API can be used in atomic services.

**System capability:** SystemCapability.Security.Cert

## EXTENSION_OID_TYPE_UNCRITICAL

```TypeScript
EXTENSION_OID_TYPE_UNCRITICAL = 2
```

表示获取扩展域中critical为false的对象标识符。

**Since:** 10

**Atomic service API:** From API version 12 this API can be used in atomic services.

**System capability:** SystemCapability.Security.Cert
