# UkeyInfo

提供USB Key证书凭据属性信息。

**Since:** 22

**System capability:** SystemCapability.Security.CertificateManager

## Modules to Import

```TypeScript
import { certificateManager } from '@kit.DeviceCertificateKit';
```

## certPurpose

```TypeScript
certPurpose?: CertificatePurpose
```

表示凭据用途。 默认值： PURPOSE_DEFAULT。

**Type:** CertificatePurpose

**Since:** 22

**System capability:** SystemCapability.Security.CertificateManager
