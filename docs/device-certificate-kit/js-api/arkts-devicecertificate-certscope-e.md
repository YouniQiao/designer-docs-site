# CertScope

Enumerates the certificate scopes.

**Since:** 18

**System capability:** SystemCapability.Security.CertificateManager

## CURRENT_USER

```TypeScript
CURRENT_USER = 1
```

The certificate is accessible only to the current user.

**Since:** 18

**System capability:** SystemCapability.Security.CertificateManager

## GLOBAL_USER

```TypeScript
GLOBAL_USER = 2
```

The certificate is accessible to all users.

**Since:** 18

**System capability:** SystemCapability.Security.CertificateManager
