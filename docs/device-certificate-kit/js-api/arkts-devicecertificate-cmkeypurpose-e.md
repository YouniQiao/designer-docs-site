# CmKeyPurpose

Enumerates the purposes of using the key.

**Since:** 11

**System capability:** SystemCapability.Security.CertificateManager

## CM_KEY_PURPOSE_SIGN

```TypeScript
CM_KEY_PURPOSE_SIGN = 4
```

Signs data.

**Since:** 11

**System capability:** SystemCapability.Security.CertificateManager

## CM_KEY_PURPOSE_VERIFY

```TypeScript
CM_KEY_PURPOSE_VERIFY = 8
```

Verifies a signature.

**Since:** 11

**System capability:** SystemCapability.Security.CertificateManager
