# CmsCertType

Enumerates certificate types obtained from CMS.

**Since:** 22

**System capability:** SystemCapability.Security.Cert

## SIGNER_CERTS

```TypeScript
SIGNER_CERTS = 0
```

Signer certificates.

**Since:** 22

**Atomic service API:** This API can be used in atomic services since API version 22.

**System capability:** SystemCapability.Security.Cert

## ALL_CERTS

```TypeScript
ALL_CERTS = 1
```

All certificates.

**Since:** 22

**Atomic service API:** This API can be used in atomic services since API version 22.

**System capability:** SystemCapability.Security.Cert
