# ExtensionEntryType

```TypeScript
enum ExtensionEntryType
```

表示获取扩展域中对象类型的枚举。

**Since:** 10

**System capability:** SystemCapability.Security.Cert

## EXTENSION_ENTRY_TYPE_ENTRY

```TypeScript
EXTENSION_ENTRY_TYPE_ENTRY = 0
```

表示获取整个对象。

**Since:** 10

**Atomic service API:** From API version 12 this API can be used in atomic services.

**System capability:** SystemCapability.Security.Cert

## EXTENSION_ENTRY_TYPE_ENTRY_CRITICAL

```TypeScript
EXTENSION_ENTRY_TYPE_ENTRY_CRITICAL = 1
```

表示获取对象的critical属性。

**Since:** 10

**Atomic service API:** From API version 12 this API can be used in atomic services.

**System capability:** SystemCapability.Security.Cert

## EXTENSION_ENTRY_TYPE_ENTRY_VALUE

```TypeScript
EXTENSION_ENTRY_TYPE_ENTRY_VALUE = 2
```

表示获取对象的数据。

**Since:** 10

**Atomic service API:** From API version 12 this API can be used in atomic services.

**System capability:** SystemCapability.Security.Cert
