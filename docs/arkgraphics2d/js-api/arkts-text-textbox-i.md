# TextBox

文本矩形区域，表示文本在布局时所占用的矩形空间。

**Since:** 12

**System capability:** SystemCapability.Graphics.Drawing

## Modules to Import

```TypeScript
import { text } from '@kit.ArkGraphics2D';
```

## rect

```TypeScript
rect: common2D.Rect
```

矩形区域信息，单位为物理像素px。

**Type:** common2D.Rect

**Since:** 12

**Atomic service API:** From API version 22 this API can be used in atomic services.

**System capability:** SystemCapability.Graphics.Drawing

## direction

```TypeScript
direction: TextDirection
```

文本方向。

**Type:** TextDirection

**Since:** 12

**Atomic service API:** From API version 22 this API can be used in atomic services.

**System capability:** SystemCapability.Graphics.Drawing
