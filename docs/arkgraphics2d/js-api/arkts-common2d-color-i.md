# Color

ARGB格式的颜色描述。

**Since:** 11

**System capability:** SystemCapability.Graphics.Drawing

## Modules to Import

```TypeScript
import { common2D } from '@kit.ArkGraphics2D';
```

## red

```TypeScript
red: int
```

颜色的R分量（红色），值是0~255的整数。

**Type:** int

**Since:** 11

**Atomic service API:** From API version 22 this API can be used in atomic services.

**System capability:** SystemCapability.Graphics.Drawing
