# ShadowFlag

```TypeScript
enum ShadowFlag
```

控制阴影绘制行为的枚举。

**Since:** 12

**System capability:** SystemCapability.Graphics.Drawing

## NONE

```TypeScript
NONE = 0
```

不使用任何阴影处理选项。

**Since:** 12

**System capability:** SystemCapability.Graphics.Drawing

## TRANSPARENT_OCCLUDER

```TypeScript
TRANSPARENT_OCCLUDER = 1
```

遮挡物是半透明的。

**Since:** 12

**System capability:** SystemCapability.Graphics.Drawing

## GEOMETRIC_ONLY

```TypeScript
GEOMETRIC_ONLY = 2
```

仅使用几何阴影效果。

**Since:** 12

**System capability:** SystemCapability.Graphics.Drawing

## ALL

```TypeScript
ALL = 3
```

使用所有可用的阴影处理选项，以生成组合阴影效果，包括半透明遮挡和几何阴影效果。

**Since:** 12

**System capability:** SystemCapability.Graphics.Drawing
