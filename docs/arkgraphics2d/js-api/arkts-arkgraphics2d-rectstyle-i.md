# RectStyle

Describes the style of a rectangle.

**Since:** 12

**System capability:** SystemCapability.Graphics.Drawing

## Modules to Import

```TypeScript
import { text } from '@ohos.graphics.text';
```

## color

```TypeScript
color: common2D.Color
```

**Type:** common2D.Color

**Since:** 12
