# RunMetrics

描述文本行中连续文本块的布局信息和度量数据。

**Since:** 12

**System capability:** SystemCapability.Graphics.Drawing

## Modules to Import

```TypeScript
import { text } from '@kit.ArkGraphics2D';
```

## textStyle

```TypeScript
textStyle: TextStyle
```

字体的样式信息。

**Type:** TextStyle

**Since:** 12

**Atomic service API:** From API version 22 this API can be used in atomic services.

**System capability:** SystemCapability.Graphics.Drawing

## fontMetrics

```TypeScript
fontMetrics: drawing.FontMetrics
```

字体度量信息。

**Type:** drawing.FontMetrics

**Since:** 12

**Atomic service API:** From API version 22 this API can be used in atomic services.

**System capability:** SystemCapability.Graphics.Drawing
