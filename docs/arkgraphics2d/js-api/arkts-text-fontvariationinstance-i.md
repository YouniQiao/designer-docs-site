# FontVariationInstance

字体可变实例信息，存放预设的可变字体样式信息。

**Since:** 24

**System capability:** SystemCapability.Graphics.Drawing

## Modules to Import

```TypeScript
import { text } from '@kit.ArkGraphics2D';
```

## localName

```TypeScript
localName: string
```

字体可变实例的本地化名称，可以为空。

**Type:** string

**Since:** 24

**Atomic service API:** This API can be used in atomic services.

**System capability:** SystemCapability.Graphics.Drawing

## coordinates

```TypeScript
coordinates: Array<FontVariation>
```

可变字体属性数组。

**Type:** Array<FontVariation>

**Since:** 24

**Atomic service API:** This API can be used in atomic services.

**System capability:** SystemCapability.Graphics.Drawing

## name

```TypeScript
name: string
```

字体可变实例的英文名称。

**Type:** string

**Since:** 24

**Atomic service API:** This API can be used in atomic services.

**System capability:** SystemCapability.Graphics.Drawing
