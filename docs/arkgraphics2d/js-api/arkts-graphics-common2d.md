# @ohos.graphics.common2D

This module defines some common data types in the 2D graphics field. > **NOTE** > > - This module uses the physical pixel unit, px.

**Since:** 11

**System capability:** SystemCapability.Graphics.Drawing

## Modules to Import

```TypeScript
import { common2D } from '@ohos.graphics.common2D';
```

## Summary

### Interfaces

| Name | Description |
| --- | --- |
| [Color](arkts-arkgraphics2d-color-i.md) | Describes a color in ARGB format. |
| [Color4f](arkts-arkgraphics2d-color4f-i.md) | Describes a color in ARGB format. |
| [Point](arkts-arkgraphics2d-point-i.md) | Describes a coordinate point. |
| [Point3d](arkts-arkgraphics2d-point3d-i.md) | Describes a 3D coordinate point. It inherits from [Point](arkts-arkgraphics2d-point-i.md#point). |
| [Rect](arkts-arkgraphics2d-rect-i.md) | Describes a rectangle, which can be defined by two coordinate points: upper left corner point and lower right corner point. |
