# RectType

```TypeScript
enum RectType
```

定义填充网格的矩形类型的枚举。仅在[Lattice](arkts-graphics-drawing.md#drawing)中使用。

**Since:** 12

**System capability:** SystemCapability.Graphics.Drawing

## DEFAULT

```TypeScript
DEFAULT = 0
```

将图像绘制到矩形网格中。

**Since:** 12

**System capability:** SystemCapability.Graphics.Drawing

## TRANSPARENT

```TypeScript
TRANSPARENT = 1
```

将矩形网格设置为透明的。

**Since:** 12

**System capability:** SystemCapability.Graphics.Drawing

## FIXEDCOLOR

```TypeScript
FIXEDCOLOR = 2
```

将[Lattice](arkts-graphics-drawing.md#drawing)中fColors数组的颜色绘制到矩形网格中。

**Since:** 12

**System capability:** SystemCapability.Graphics.Drawing
