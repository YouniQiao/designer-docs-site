# TextBadgeType

Enumerates the text badges.

**Since:** 20

**System capability:** SystemCapability.Graphics.Drawing

## TEXT_BADGE_NONE

```TypeScript
TEXT_BADGE_NONE = 0
```

Disables the superscript and subscript.

**Since:** 20

**Atomic service API:** This API can be used in atomic services since API version 22.

**System capability:** SystemCapability.Graphics.Drawing
