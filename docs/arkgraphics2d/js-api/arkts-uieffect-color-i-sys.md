# Color

RGBA格式的颜色描述。

**Since:** 20

**System capability:** SystemCapability.Graphics.Drawing

**System API:** This is a system API.

## Modules to Import

```TypeScript
import { uiEffect } from '@kit.ArkGraphics2D';
```

## red

```TypeScript
red: double
```

颜色的R分量（红色）。值大于等于0，当值小于0时无效。

**Type:** double

**Since:** 20

**System capability:** SystemCapability.Graphics.Drawing

**System API:** This is a system API.
