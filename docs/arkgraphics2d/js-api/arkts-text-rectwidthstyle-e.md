# RectWidthStyle

```TypeScript
enum RectWidthStyle
```

矩形区域宽度规格枚举。

**Since:** 12

**System capability:** SystemCapability.Graphics.Drawing

## TIGHT

```TypeScript
TIGHT = 0
```

不设置letterSpacing时，与字形紧贴，否则包含letterSpacing的宽度。

**Since:** 12

**Atomic service API:** From API version 22 this API can be used in atomic services.

**System capability:** SystemCapability.Graphics.Drawing

## MAX

```TypeScript
MAX = 1
```

扩展宽度，以匹配所有行上最宽矩形的位置。

**Since:** 12

**Atomic service API:** From API version 22 this API can be used in atomic services.

**System capability:** SystemCapability.Graphics.Drawing
