# TextEncoding

```TypeScript
enum TextEncoding
```

文本的编码类型枚举。

**Since:** 11

**System capability:** SystemCapability.Graphics.Drawing

## TEXT_ENCODING_UTF8

```TypeScript
TEXT_ENCODING_UTF8 = 0
```

使用1个字节表示UTF-8或ASCII。

**Since:** 11

**Atomic service API:** From API version 22 this API can be used in atomic services.

**System capability:** SystemCapability.Graphics.Drawing

## TEXT_ENCODING_UTF16

```TypeScript
TEXT_ENCODING_UTF16 = 1
```

使用2个字节表示大部分unicode。

**Since:** 11

**Atomic service API:** From API version 22 this API can be used in atomic services.

**System capability:** SystemCapability.Graphics.Drawing

## TEXT_ENCODING_UTF32

```TypeScript
TEXT_ENCODING_UTF32 = 2
```

使用4个字节表示全部unicode。

**Since:** 11

**Atomic service API:** From API version 22 this API can be used in atomic services.

**System capability:** SystemCapability.Graphics.Drawing

## TEXT_ENCODING_GLYPH_ID

```TypeScript
TEXT_ENCODING_GLYPH_ID = 3
```

使用2个字节表示glyph index。

**Since:** 11

**Atomic service API:** From API version 22 this API can be used in atomic services.

**System capability:** SystemCapability.Graphics.Drawing
