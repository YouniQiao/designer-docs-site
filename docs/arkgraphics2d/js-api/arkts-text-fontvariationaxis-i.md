**Since:** 24

**System capability:** SystemCapability.Graphics.Drawing


```TypeScript

```

## flags

```TypeScript
flags: int
```

字体可变轴的标志位，用于标记某个可变轴是否应该对用户隐藏，值为0或1。值为0时表示该轴对用户可见，值为1时表示该轴应隐藏。

**Type:** int

**Since:** 24

**Atomic service API:** This API can be used in atomic services.

**System capability:** SystemCapability.Graphics.Drawing
