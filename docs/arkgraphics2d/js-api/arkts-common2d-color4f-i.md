# Color4f

ARGB格式的颜色描述。

**Since:** 20

**System capability:** SystemCapability.Graphics.Drawing

## Modules to Import

```TypeScript
import { common2D } from '@kit.ArkGraphics2D';
```

## red

```TypeScript

```

颜色的R分量（红色），值是0.0~1.0的浮点数。

**Since:** 20

**System capability:** SystemCapability.Graphics.Drawing
