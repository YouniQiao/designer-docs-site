# TextHeightBehavior

```TypeScript
enum TextHeightBehavior
```

文本高度修饰符模式枚举。

**Since:** 12

**System capability:** SystemCapability.Graphics.Drawing

## ALL

```TypeScript
ALL = 0x0
```

高度修饰符设置为段落中第一行上升、最后一行下降。

**Since:** 12

**Atomic service API:** From API version 22 this API can be used in atomic services.

**System capability:** SystemCapability.Graphics.Drawing

## DISABLE_FIRST_ASCENT

```TypeScript
DISABLE_FIRST_ASCENT = 0x1
```

高度修饰符设置为禁止段落中第一行上升。

**Since:** 12

**Atomic service API:** From API version 22 this API can be used in atomic services.

**System capability:** SystemCapability.Graphics.Drawing

## DISABLE_LAST_ASCENT

```TypeScript
DISABLE_LAST_ASCENT = 0x2
```

高度修饰符设置为禁止段落中最后一行下降。

**Since:** 12

**Atomic service API:** From API version 22 this API can be used in atomic services.

**System capability:** SystemCapability.Graphics.Drawing

## DISABLE_ALL

```TypeScript
DISABLE_ALL = 0x1 | 0x2
```

高度修饰符设置为禁止段落中第一行上升、最后一行下降。

**Since:** 12

**Atomic service API:** From API version 22 this API can be used in atomic services.

**System capability:** SystemCapability.Graphics.Drawing
