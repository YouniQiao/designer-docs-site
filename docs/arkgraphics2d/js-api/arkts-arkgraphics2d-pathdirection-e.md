# PathDirection

Enumerates the directions of a closed contour.

**Since:** 12

**System capability:** SystemCapability.Graphics.Drawing

## CLOCKWISE

```TypeScript
CLOCKWISE = 0
```

Adds a closed contour clockwise.

**Since:** 12

**System capability:** SystemCapability.Graphics.Drawing

## COUNTER_CLOCKWISE

```TypeScript
COUNTER_CLOCKWISE = 1
```

Adds a closed contour counterclockwise.

**Since:** 12

**System capability:** SystemCapability.Graphics.Drawing
