# createFilter

## createFilter

```TypeScript
function createFilter(): Filter
```

创建Filter实例用于给组件添加多种filter效果。

**Since:** 12

**System capability:** SystemCapability.Graphics.Drawing

**Return value:**

| Type | Description |
| --- | --- |
| Filter | 返回Filter的头节点。 |

**Example**

```TypeScript
let filter : uiEffect.Filter = uiEffect.createFilter()

```
