# FontEdging

```TypeScript
enum FontEdging
```

字型边缘效果类型枚举。 > **说明：** > > FontEdging不支持位图字体（如点阵字体、emoji等）。

**Since:** 12

**System capability:** SystemCapability.Graphics.Drawing

## ALIAS

```TypeScript
ALIAS = 0
```

无抗锯齿处理。

**Since:** 12

**Atomic service API:** From API version 22 this API can be used in atomic services.

**System capability:** SystemCapability.Graphics.Drawing

## ANTI_ALIAS

```TypeScript
ANTI_ALIAS = 1
```

使用抗锯齿来平滑字型边缘。

**Since:** 12

**Atomic service API:** From API version 22 this API can be used in atomic services.

**System capability:** SystemCapability.Graphics.Drawing

## SUBPIXEL_ANTI_ALIAS

```TypeScript
SUBPIXEL_ANTI_ALIAS = 2
```

使用次像素级别的抗锯齿平滑字型边缘，可以获得更平滑的字型渲染效果。

**Since:** 12

**Atomic service API:** From API version 22 this API can be used in atomic services.

**System capability:** SystemCapability.Graphics.Drawing
