# PointMode

```TypeScript
enum PointMode
```

绘制数组点的方式的枚举。

**Since:** 12

**System capability:** SystemCapability.Graphics.Drawing

## POINTS

```TypeScript
POINTS = 0
```

分别绘制每个点。

**Since:** 12

**System capability:** SystemCapability.Graphics.Drawing

## LINES

```TypeScript
LINES = 1
```

将每对点绘制为线段。

**Since:** 12

**System capability:** SystemCapability.Graphics.Drawing

## POLYGON

```TypeScript
POLYGON = 2
```

将点阵列绘制为开放多边形。

**Since:** 12

**System capability:** SystemCapability.Graphics.Drawing
