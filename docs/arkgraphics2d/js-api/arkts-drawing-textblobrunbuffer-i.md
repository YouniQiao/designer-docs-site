# TextBlobRunBuffer

描述一行文字中具有相同属性的连续字形。

**Since:** 11

**System capability:** SystemCapability.Graphics.Drawing

## Modules to Import

```TypeScript
import { drawing } from '@kit.ArkGraphics2D';
```

## positionY

```TypeScript
positionY: double
```

文本的起点y轴坐标，该参数为浮点数。

**Type:** double

**Since:** 11

**System capability:** SystemCapability.Graphics.Drawing

## glyph

```TypeScript
glyph: int
```

存储文字的索引，该参数为整数，传入浮点类型时向下取整。

**Type:** int

**Since:** 11

**System capability:** SystemCapability.Graphics.Drawing

## positionX

```TypeScript
positionX: double
```

文本的起点x轴坐标，该参数为浮点数。

**Type:** double

**Since:** 11

**System capability:** SystemCapability.Graphics.Drawing
