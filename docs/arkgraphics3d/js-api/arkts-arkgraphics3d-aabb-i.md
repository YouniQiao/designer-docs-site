# Aabb

Axis aligned bounding box.

**Since:** 12

**System capability:** SystemCapability.ArkUi.Graphics3D

## aabbMax

```TypeScript
aabbMax: Vec3
```

Coordinates of the AABB maximum corner, the unit is the scene unit in the world coordinate system (e.g., cm, m, km).

**Type:** Vec3

**Since:** 12

**System capability:** SystemCapability.ArkUi.Graphics3D
