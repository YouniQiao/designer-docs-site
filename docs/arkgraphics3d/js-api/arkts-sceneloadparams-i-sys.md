## offset

```TypeScript
offset?: long
```

资源中3D模型数据的起始偏移量 单位：字节，值必须大于等于0. 默认值：0.

**Type:** long

**Since:** 26.0.0

**Model restriction:** This API can be used only in the Stage model.

**System capability:** SystemCapability.ArkUi.Graphics3D

**System API:** This is a system API.
