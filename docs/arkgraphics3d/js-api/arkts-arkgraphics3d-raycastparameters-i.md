# RaycastParameters

How a raycast should be performed.

**Since:** 20

**System capability:** SystemCapability.ArkUi.Graphics3D

## rootNode

```TypeScript
rootNode?: Node
```

If defined, search only the nodes in the hierarchy under this node
If undefined, search all the nodes in the scene

**Type:** Node

**Since:** 20

**System capability:** SystemCapability.ArkUi.Graphics3D
