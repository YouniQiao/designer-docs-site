# PolygonMode

The enum of polygon mode.

**Since:** 23

**System capability:** SystemCapability.ArkUi.Graphics3D

## FILL

```TypeScript
FILL = 0
```

Render the whole polygon

**Since:** 23

**System capability:** SystemCapability.ArkUi.Graphics3D

## LINE

```TypeScript
LINE = 1
```

Render only edges(wireframe) of the polygon

**Since:** 23

**System capability:** SystemCapability.ArkUi.Graphics3D

## POINT

```TypeScript
POINT = 2
```

Render only vertices of the polygon

**Since:** 23

**System capability:** SystemCapability.ArkUi.Graphics3D
