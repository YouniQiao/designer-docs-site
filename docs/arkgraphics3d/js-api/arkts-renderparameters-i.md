# RenderParameters

定义手动渲染的参数.

**Since:** 15

**System capability:** SystemCapability.ArkUi.Graphics3D

## alwaysRender

```TypeScript
alwaysRender?: boolean
```

如果为true，即使场景没有变化也始终渲染 自上一帧以来. 如果为false，则场景没有变化时可以省略渲染.

**Type:** boolean

**Since:** 15

**System capability:** SystemCapability.ArkUi.Graphics3D
