# ExposureInfo

Describes the exposure information object.

**起始版本：** 12

**系统能力：** SystemCapability.Multimedia.Camera.Core

**系统接口：** 此接口为系统接口。

## 导入模块

```TypeScript
import { camera } from '@kit.CameraKit';
```

## exposureTime

```TypeScript
readonly exposureTime?: int
```
