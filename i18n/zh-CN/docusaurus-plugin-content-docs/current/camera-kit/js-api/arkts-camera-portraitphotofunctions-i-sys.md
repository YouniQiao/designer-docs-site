# PortraitPhotoFunctions

Portrait Photo Functions object.

**继承实现关系：** PortraitPhotoFunctions继承自：FlashQuery， AutoExposureQuery， FocusQuery， ZoomQuery， BeautyQuery， ColorEffectQuery， ColorManagementQuery， PortraitQuery， ApertureQuery， SceneDetectionQuery。

**起始版本：** 13

**系统能力：** SystemCapability.Multimedia.Camera.Core

**系统接口：** 此接口为系统接口。

## 导入模块

```TypeScript
import { camera } from '@kit.CameraKit';
```
