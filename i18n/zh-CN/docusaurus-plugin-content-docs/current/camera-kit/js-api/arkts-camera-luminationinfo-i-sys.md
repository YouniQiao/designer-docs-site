# LuminationInfo

Describes the illumination information.

**起始版本：** 12

**系统能力：** SystemCapability.Multimedia.Camera.Core

**系统接口：** 此接口为系统接口。

```TypeScript
import { camera } from '@kit.CameraKit';
```

## lumination

