# VideoFunctions

Video Functions object.

**继承实现关系：** VideoFunctions继承自：FlashQuery， AutoExposureQuery， ManualExposureQuery， FocusQuery， ZoomQuery， StabilizationQuery， BeautyQuery， ColorEffectQuery， ColorManagementQuery， MacroQuery， SceneDetectionQuery。

**起始版本：** 13

**系统能力：** SystemCapability.Multimedia.Camera.Core

**系统接口：** 此接口为系统接口。

## 导入模块

```TypeScript
import { camera } from '@kit.CameraKit';
```
