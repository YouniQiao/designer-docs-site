# SmoothZoomInfo

Describes the smooth zoom information.

**起始版本：** 11

**系统能力：** SystemCapability.Multimedia.Camera.Core

## 导入模块

```TypeScript
import { camera } from '@kit.CameraKit';
```

## duration

```TypeScript
duration: int
```

Total duration of smooth zoom, in ms.

**类型：** int

**起始版本：** 11

**原子化服务API：** 从API version 19开始，该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.Multimedia.Camera.Core
