# USBEndpoint

通过USB发送和接收数据的端口。通过[USBInterface]usb.USBInterface获取。

**起始版本：** 8

**废弃版本：** 9

**替代接口：** @ohos.usbManager:usbManager.USBEndpoint

**系统能力：** SystemCapability.USB.USBManager

## 导入模块

```TypeScript
import { usb } from '@kit.BasicServicesKit';
```
