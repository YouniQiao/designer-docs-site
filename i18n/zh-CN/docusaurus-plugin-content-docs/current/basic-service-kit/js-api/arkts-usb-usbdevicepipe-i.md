# USBDevicePipe

USB设备消息传输通道，用于确定设备。

**起始版本：** 8

**废弃版本：** 9

**替代接口：** @ohos.usbManager:usbManager.USBDevicePipe

**系统能力：** SystemCapability.USB.USBManager

## 导入模块

```TypeScript
import { usb } from '@kit.BasicServicesKit';
```

## devAddress

```TypeScript
devAddress: number
```

设备地址。

**类型：** number

**废弃版本：** 9

**替代接口：** @ohos.usbManager:usbManager.USBDevicePipe.devAddress
