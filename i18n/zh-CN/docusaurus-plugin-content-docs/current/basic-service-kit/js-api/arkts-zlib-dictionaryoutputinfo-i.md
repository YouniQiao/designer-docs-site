# DictionaryOutputInfo

InflateGetDictionary和deflateGetDictionary这两个函数会返回值的相关信息。

**起始版本：** 12

**系统能力：** SystemCapability.BundleManager.Zlib

## 导入模块

```TypeScript
import { zlib } from '@kit.BasicServicesKit';
```

## dictionaryLength

```TypeScript
dictionaryLength: int
```

字典的长度。

**类型：** int

**起始版本：** 12

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.BundleManager.Zlib
