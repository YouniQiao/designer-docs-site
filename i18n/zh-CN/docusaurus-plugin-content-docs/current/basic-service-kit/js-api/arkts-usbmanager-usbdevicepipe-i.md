# USBDevicePipe

USB设备消息传输通道，用于确定设备。

**起始版本：** 9

**系统能力：** SystemCapability.USB.USBManager

## 导入模块

```TypeScript
import { usbManager } from '@kit.BasicServicesKit';
```

## devAddress

```TypeScript
devAddress: int
```

设备地址。

**类型：** int

**起始版本：** 9

**系统能力：** SystemCapability.USB.USBManager

## busNum

```TypeScript
busNum: int
```

总线地址。

**类型：** int

**起始版本：** 9

**系统能力：** SystemCapability.USB.USBManager
