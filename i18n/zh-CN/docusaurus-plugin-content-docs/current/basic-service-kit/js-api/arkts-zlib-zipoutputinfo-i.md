# ZipOutputInfo

压缩和解压缩的返回值信息。

**起始版本：** 12

**系统能力：** SystemCapability.BundleManager.Zlib

## 导入模块

```TypeScript
import { zlib } from '@kit.BasicServicesKit';
```

## destLen

```TypeScript
destLen: long
```

目标缓冲区的总长度。

**类型：** long

**起始版本：** 12

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.BundleManager.Zlib
