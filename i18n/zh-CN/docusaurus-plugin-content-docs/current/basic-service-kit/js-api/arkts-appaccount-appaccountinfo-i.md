# AppAccountInfo

表示应用账号信息。

**起始版本：** 7

**系统能力：** SystemCapability.Account.AppAccount

## 导入模块

```TypeScript
import { appAccount } from '@kit.BasicServicesKit';
```

## owner

```TypeScript
owner: string
```

应用账号所有者的包名。最大长度为1024个字符。

**类型：** string

**起始版本：** 7

**系统能力：** SystemCapability.Account.AppAccount

## name

```TypeScript
name: string
```

应用账号的名称。

**类型：** string

**起始版本：** 7

**系统能力：** SystemCapability.Account.AppAccount
