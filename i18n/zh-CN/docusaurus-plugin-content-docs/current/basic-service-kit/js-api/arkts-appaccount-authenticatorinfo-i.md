# AuthenticatorInfo

表示OAuth认证器信息。

**起始版本：** 8

**系统能力：** SystemCapability.Account.AppAccount

## 导入模块

```TypeScript
import { appAccount } from '@kit.BasicServicesKit';
```

## iconId

```TypeScript
iconId: long
```

认证器的图标标识。

**类型：** long

**起始版本：** 8

**系统能力：** SystemCapability.Account.AppAccount

## owner

```TypeScript
owner: string
```

认证器的所有者的包名。

**类型：** string

**起始版本：** 8

**系统能力：** SystemCapability.Account.AppAccount

## labelId

```TypeScript
labelId: long
```

认证器的标签标识。

**类型：** long

**起始版本：** 8

**系统能力：** SystemCapability.Account.AppAccount
