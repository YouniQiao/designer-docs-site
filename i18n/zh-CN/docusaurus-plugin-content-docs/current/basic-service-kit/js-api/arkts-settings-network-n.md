# network

Provides methods for setting network information, including the data roaming status, HTTP proxy configurations, and preferred networks.

**起始版本：** 7

**系统能力：** SystemCapability.Applications.Settings.Core

## 导入模块

```TypeScript
import { settings } from '@kit.BasicServicesKit';
```

## 汇总

### 常量

| 名称 | 描述 |
| --- | --- |
| [DATA_ROAMING_STATUS](arkts-network-con.md#DATA_ROAMING_STATUS) | Specifies whether data roaming is enabled. <p>If the value is {@code true}, data roaming is enabled. If the value is {@code false}, data roaming is disabled. |
| [HTTP_PROXY_CFG](arkts-network-con.md#HTTP_PROXY_CFG) | Indicates the host name and port number of the global HTTP proxy. The host name and port number are separated by a colon (:). |
| [NETWORK_PREFERENCE_USAGE](arkts-network-con.md#NETWORK_PREFERENCE_USAGE) | Indicates the user preferences of the network to use. |
