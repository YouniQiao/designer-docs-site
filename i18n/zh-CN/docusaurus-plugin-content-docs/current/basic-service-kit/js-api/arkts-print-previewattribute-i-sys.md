# PreviewAttribute

定义打印预览属性的接口。

**起始版本：** 10

**系统能力：** SystemCapability.Print.PrintFramework

**系统接口：** 此接口为系统接口。

## 导入模块

```TypeScript
import { print } from '@kit.BasicServicesKit';
```

## result

```TypeScript
result?: int
```

表示预览文件结果。默认值为-1。

**类型：** int

**起始版本：** 10

**系统能力：** SystemCapability.Print.PrintFramework

**系统接口：** 此接口为系统接口。

## previewRange

```TypeScript
previewRange: PrinterRange
```

表示预览页面范围。

**类型：** PrinterRange

**起始版本：** 10

**系统能力：** SystemCapability.Print.PrintFramework

**系统接口：** 此接口为系统接口。
