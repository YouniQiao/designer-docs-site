# PrintMargin

定义打印页边距的接口。

**起始版本：** 10

**系统能力：** SystemCapability.Print.PrintFramework

**系统接口：** 此接口为系统接口。

## 导入模块

```TypeScript
import { print } from '@kit.BasicServicesKit';
```

## top

```TypeScript
top?: int
```

表示页面上边距。默认值为0。单位：毫米。

**类型：** int

**起始版本：** 10

**系统能力：** SystemCapability.Print.PrintFramework

**系统接口：** 此接口为系统接口。
