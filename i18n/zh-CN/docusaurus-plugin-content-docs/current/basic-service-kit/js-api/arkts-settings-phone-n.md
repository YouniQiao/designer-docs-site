# phone

Provides methods for setting the answering mode of incoming and outgoing calls.

**起始版本：** 7

**系统能力：** SystemCapability.Applications.Settings.Core

## 导入模块

```TypeScript
import { settings } from '@kit.BasicServicesKit';
```

## 汇总

### 常量

| 名称 | 描述 |
| --- | --- |
| [RTT_CALLING_STATUS](arkts-phone-con.md#RTT_CALLING_STATUS) | Specifies whether real-time text (RTT) calling is enabled. If enabled, incoming and outgoing calls are answered as RTT calls when supported by the device and carrier. If the value is {@code 1}, RTT calling is enabled. If the value is {@code 0}, RTT calling is disabled. |
