# GetAuthInfoOptions

表示[查询认证凭据信息]osAccount.UserIdentityManager.getAuthInfo(options?: GetAuthInfoOptions)的可选参数集合。

**起始版本：** 12

**系统能力：** SystemCapability.Account.OsAccount

**系统接口：** 此接口为系统接口。

## 导入模块

```TypeScript
import { osAccount } from '@kit.BasicServicesKit';
```

**系统接口：** 此接口为系统接口。
