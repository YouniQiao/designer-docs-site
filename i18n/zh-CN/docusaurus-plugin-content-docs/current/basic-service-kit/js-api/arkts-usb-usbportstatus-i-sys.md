# USBPortStatus

**起始版本：** 9

**废弃版本：** 9

**替代接口：** @ohos.usbManager:usbManager.USBPortStatus

**系统能力：** SystemCapability.USB.USBManager

**系统接口：** 此接口为系统接口。
