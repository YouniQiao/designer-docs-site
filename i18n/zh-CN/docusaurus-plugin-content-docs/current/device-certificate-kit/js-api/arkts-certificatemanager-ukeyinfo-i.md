# UkeyInfo

提供USB Key证书凭据属性信息。

**起始版本：** 22

**系统能力：** SystemCapability.Security.CertificateManager

## 导入模块

```TypeScript
import { certificateManager } from '@kit.DeviceCertificateKit';
```

## certPurpose

```TypeScript
certPurpose?: CertificatePurpose
```

表示凭据用途。 默认值： PURPOSE_DEFAULT。

**类型：** CertificatePurpose

**起始版本：** 22

**系统能力：** SystemCapability.Security.CertificateManager
