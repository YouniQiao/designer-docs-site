# DataArray

buffer数组的列表。

**起始版本：** 9

**系统能力：** SystemCapability.Security.Cert

## 导入模块

```TypeScript
import { cert } from '@kit.DeviceCertificateKit';
```

## data

```TypeScript
data: Array<Uint8Array>
```

表示数据数组内容。

**类型：** Array<Uint8Array>

**起始版本：** 9

**原子化服务API：** 从API version 12开始，该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.Security.Cert
