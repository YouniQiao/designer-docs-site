# DeviceType

```TypeScript
enum DeviceType
```

**起始版本：** 10

**系统能力：** SystemCapability.Multimedia.AVSession.Core

## DEVICE_TYPE_LOCAL

```TypeScript
DEVICE_TYPE_LOCAL = 0
```

A device type indicating the route is on internal speakers or audio jack on the device itself.

**起始版本：** 10

**原子化服务API：** 从API version 12开始，该接口支持在原子化服务中使用。
