# PiPWindowInfo

画中画窗口信息。

**起始版本：** 15

**系统能力：** SystemCapability.Window.SessionManager

## 导入模块

```TypeScript
import { PiPWindow } from '@kit.ArkUI';
```

## size

```TypeScript
size: PiPWindowSize
```

画中画窗口大小。

**类型：** PiPWindowSize

**起始版本：** 15

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.Window.SessionManager

## windowId

```TypeScript
windowId: int
```

画中画窗口ID。

**类型：** int

**起始版本：** 15

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.Window.SessionManager
