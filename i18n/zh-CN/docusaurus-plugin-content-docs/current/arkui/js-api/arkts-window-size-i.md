# Size

窗口大小，单位为px。

**起始版本：** 7

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

## 导入模块

```TypeScript
import { window } from '@kit.ArkUI';
```

## width

```TypeScript
width: int
```

窗口宽度，单位为px，该参数应为整数。

**类型：** int

**起始版本：** 7

**原子化服务API：** 从API version 11开始，该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

## height

```TypeScript
height: int
```

窗口高度，单位为px，该参数应为整数。

**类型：** int

**起始版本：** 7

**原子化服务API：** 从API version 11开始，该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.WindowManager.WindowManager.Core
