# ExpandedMenuOptions

扩展下拉菜单。 继承于[MenuItemOptions](docroot://reference/apis-arkui/arkui-ts/ts-basic-components-menuitem.md#menuitemoptions对象说明)。

**继承实现关系：** ExpandedMenuOptions继承自：MenuItemOptions。

**起始版本：** 11

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## 导入模块

```TypeScript
import { EditorMenuOptions,SelectionMenuOptions,EditorEventInfo,SelectionMenu,ExpandedMenuOptions } from '@kit.ArkUI';
```

## action

```TypeScript
action?: () => void
```

点击菜单项的事件回调。

**类型：** () => void

**起始版本：** 11

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API version 12开始，该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
