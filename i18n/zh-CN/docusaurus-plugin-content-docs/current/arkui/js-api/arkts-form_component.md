# form_component

Defines FormComponent Component instance.

## form_component

```TypeScript
form_component(value: FormInfo)
```

Set a new value of form info.

**起始版本：** 7

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**系统接口：** 此接口为系统接口。

**参数：**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| value | FormInfo | 是 | [since 9 - 11] |
