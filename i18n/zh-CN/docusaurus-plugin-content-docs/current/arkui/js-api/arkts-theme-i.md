# Theme

Defines the struct of Theme.

**起始版本：** 12

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## 导入模块

```TypeScript
import { CustomColors,ThemeControl,Colors,CustomDarkColors,Theme,CustomTheme } from '@kit.ArkUI';
```
