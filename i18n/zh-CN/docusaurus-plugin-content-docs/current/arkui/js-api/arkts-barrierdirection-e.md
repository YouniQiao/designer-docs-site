# BarrierDirection

```TypeScript
declare enum BarrierDirection
```

定义屏障线的方向。

**起始版本：** 12

**系统能力：** SystemCapability.Test.UiTest

## LEFT

```TypeScript
LEFT = 0
```

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
