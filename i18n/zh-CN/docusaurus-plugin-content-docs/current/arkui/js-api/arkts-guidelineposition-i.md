# GuideLinePosition

guideLine位置参数，用于定义guideLine的位置。

**起始版本：** 12

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## start

```TypeScript
start? : Dimension
```

guideLine距离容器左侧或者顶部的距离。

**类型：** Dimension

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
