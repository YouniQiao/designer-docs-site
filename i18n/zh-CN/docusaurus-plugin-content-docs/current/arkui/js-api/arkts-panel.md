# panel

Defines Panel Component instance.

## panel

```TypeScript
panel(show: boolean)
```

Called when the panel slidable panel pops up.

**起始版本：** 11

**废弃版本：** 12

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**参数：**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| show | boolean | 是 |  |
