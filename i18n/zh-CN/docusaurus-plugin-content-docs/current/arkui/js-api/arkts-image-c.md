# Image

You can create an Image object by calling new Image().

**起始版本：** 11

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## onerror

```TypeScript
onerror?: () => void
```

Called when an image fails to be loaded. This function has no parameter.

**类型：** () => void

**起始版本：** 11

**模型约束：** 此接口仅可在FA模型下使用。

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## src

```TypeScript
src: string
```
