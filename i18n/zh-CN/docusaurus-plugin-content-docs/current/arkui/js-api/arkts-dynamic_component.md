# dynamic_component

Defines DynamicComponent Component instance.

## dynamic_component

```TypeScript
dynamic_component(options: DynamicOptions)
```

Construct the DynamicComponentInterface.<br/> Called when the DynamicComponentInterface is used.

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**系统接口：** 此接口为系统接口。

**参数：**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| options | DynamicOptions | 是 | Construction configuration of DynamicComponentAttribute |
