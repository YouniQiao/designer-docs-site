# OperateItemV2Options

OperateItemV2构造函数的参数选项。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## arrow

```TypeScript
arrow?: OperateIconV2
```

列表项右侧元素为箭头。

**类型：** OperateIconV2

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本26.0.0开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
