# UnionEffectContainerAttribute

支持通用事件。

**继承实现关系：** UnionEffectContainerAttribute继承自：CommonMethod<UnionEffectContainerAttribute>。

**起始版本：** 23

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**系统接口：** 此接口为系统接口。

## pointLight

```TypeScript
pointLight(light: PointLightStyle)
```

设置点光源样式。

**起始版本：** 23

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**系统接口：** 此接口为系统接口。

**参数：**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| light | PointLightStyle | 是 | 点光源样式。 |
