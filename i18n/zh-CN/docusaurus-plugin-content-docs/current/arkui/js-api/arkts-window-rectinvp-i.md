# RectInVP

窗口矩形区域，单位为vp。

**起始版本：** 23

**系统能力：** SystemCapability.Window.SessionManager

## 导入模块

```TypeScript
import { window } from '@kit.ArkUI';
```

## top

```TypeScript
top: double
```

矩形区域的上边界值，单位为vp。

**类型：** double

**起始版本：** 23

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Window.SessionManager
