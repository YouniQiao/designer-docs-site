# Rect

窗口矩形区域。

**起始版本：** 7

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

## 导入模块

```TypeScript
import { window } from '@kit.ArkUI';
```

## top

```TypeScript
top: int
```

矩形区域的上边界，单位为px，该参数应为整数。

**类型：** int

**起始版本：** 7

**原子化服务API：** 从API version 11开始，该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.WindowManager.WindowManager.Core
