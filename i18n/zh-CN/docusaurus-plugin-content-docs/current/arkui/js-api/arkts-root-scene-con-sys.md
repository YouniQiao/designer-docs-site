# Constants

## RootScene

```TypeScript
declare const RootScene: RootSceneInterface
```

Defines the RootScene component.

**起始版本：** 10

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**系统接口：** 此接口为系统接口。
