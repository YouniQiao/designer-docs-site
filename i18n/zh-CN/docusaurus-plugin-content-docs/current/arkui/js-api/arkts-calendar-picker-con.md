# Constants

## CalendarPicker

```TypeScript
declare const CalendarPicker: CalendarPickerInterface
```

日历选择器组件，提供下拉日历弹窗，可以让用户选择日期。 > **说明：** > - 该组件从API版本26.0.0开始支持[WithTheme]./with_theme。 子组件 无

**起始版本：** 10

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API version 11开始，该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## CalendarPickerInstance

```TypeScript
declare const CalendarPickerInstance: CalendarPickerAttribute
```
