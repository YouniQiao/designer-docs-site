# clearTimeout

## clearTimeout

```TypeScript
export declare function clearTimeout(timeoutID?: number): void
```

Cancel the timer set by "setTimeout()".

**起始版本：** 7

**原子化服务API：** 从API version 11开始，该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**参数：**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| timeoutID | number | 否 |  |
