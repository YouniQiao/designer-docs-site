# Constants

## SymbolSpan

```TypeScript
declare const SymbolSpan: SymbolSpanInterface
```

作为Text组件的子组件，用于显示图标小符号的组件。 > **说明：** > - 本模块接口仅可在Stage模型下使用。 > > - 该组件支持继承父组件Text的属性，即如果子组件未设置属性且父组件设置属性，则继承父组件设置的全部属性。 > > - SymbolSpan拖拽不会置灰显示。 ###### 子组件 不支持子组件。

**起始版本：** 11

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API version 12开始，该接口支持在原子化服务中使用。

**卡片能力：** 从API version 12开始，该接口支持在ArkTS卡片中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## SymbolSpanInstance

```TypeScript
declare const SymbolSpanInstance: SymbolSpanAttribute
```

**卡片能力：** 从API version 12开始，该接口支持在ArkTS卡片中使用。
