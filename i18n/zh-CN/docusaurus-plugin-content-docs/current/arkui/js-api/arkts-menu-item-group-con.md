# Constants

## MenuItemGroup

```TypeScript
declare const MenuItemGroup: MenuItemGroupInterface
```

该组件用来展示菜单MenuItem的分组。 > **说明：** > - 该组件从API版本26.0.0开始支持[WithTheme]with_theme。 ###### 子组件 包含[MenuItem]menu_item子组件。

**起始版本：** 9

**原子化服务API：** 从API version 11开始，该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## MenuItemGroupInstance

declare const MenuItemGroupInstance: MenuItemGroupAttribute
