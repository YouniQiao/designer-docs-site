# DepthCameraParams

**起始版本：** 26.0.0

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**系统接口：** 此接口为系统接口。

## quaternion

```TypeScript
quaternion: DepthVector4
```

Camera rotation as quaternion (x, y, z, w). Represents the orientation of the camera in 3D space.

**类型：** DepthVector4

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**系统接口：** 此接口为系统接口。
