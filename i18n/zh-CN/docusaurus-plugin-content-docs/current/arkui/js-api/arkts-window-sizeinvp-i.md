# SizeInVP

窗口大小，单位为vp。

**起始版本：** 23

**系统能力：** SystemCapability.Window.SessionManager

## 导入模块

```TypeScript
import { window } from '@kit.ArkUI';
```

## width

```TypeScript
width: double
```

窗口宽度，单位为vp，该参数为浮点数。

**类型：** double

**起始版本：** 23

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Window.SessionManager

## height

```TypeScript
height: double
```

窗口高度，单位为vp，该参数为浮点数。

**类型：** double

**起始版本：** 23

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Window.SessionManager
