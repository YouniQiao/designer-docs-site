# ToastShowMode

```TypeScript
export enum ToastShowMode
```

设置Toast的显示模式，默认显示在应用内，支持显示在子窗。

**起始版本：** 11

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## DEFAULT

```TypeScript
DEFAULT = 0
```

Toast显示在应用内。

**起始版本：** 11

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API version 12开始，该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## TOP_MOST

```TypeScript
TOP_MOST = 1
```

Toast显示在子窗。

**起始版本：** 11

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API version 12开始，该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## SYSTEM_TOP_MOST

```TypeScript
SYSTEM_TOP_MOST = 2
```

Toast shows in SYSTEM_TOAST window.

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**系统接口：** 此接口为系统接口。
