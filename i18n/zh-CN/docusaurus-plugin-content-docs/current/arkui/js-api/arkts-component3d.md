# component3d

定义Component3D实例.

## component3d

```TypeScript
component3d(sceneOptions?: SceneOptions)
```

构造函数使用的SceneOptions

**起始版本：** 12

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.ArkUi.Graphics3D

**参数：**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| sceneOptions | SceneOptions | 否 | 3D场景控制器 |
