# DateOptions

Defines date options for DatePickerComponent.

**继承实现关系：** DateOptions继承自：CommonOptions。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## 导入模块

```TypeScript
import { DisplayMode,TimeFormat,DatePickerComponent,DateMode,DatePickerComponentOptions,DatePickerComponentResult } from '@kit.ArkUI';
```

## mode

```TypeScript
mode?: DateMode
```

Defines mode of date picker.

**类型：** DateMode

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## lunar

```TypeScript
lunar?: boolean
```

Specifies whether to display lunar calendar.

**类型：** boolean

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
