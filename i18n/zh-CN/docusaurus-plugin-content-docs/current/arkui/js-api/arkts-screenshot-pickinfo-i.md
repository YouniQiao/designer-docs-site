# PickInfo

截取图像的信息。

**起始版本：** 12

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

## 导入模块

```TypeScript
import { screenshot } from '@kit.ArkUI';
```

## pickRect

```TypeScript
pickRect: Rect
```

表示截取图像的区域。

**类型：** Rect

**起始版本：** 12

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

## pixelMap

```TypeScript
pixelMap: image.PixelMap
```

表示截取的图像PixelMap对象。

**类型：** image.PixelMap

**起始版本：** 12

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.WindowManager.WindowManager.Core
