# SystemBarTintState

当前系统栏回调信息集合。

**起始版本：** 8

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

**系统接口：** 此接口为系统接口。

## 导入模块

```TypeScript
import { window } from '@kit.ArkUI';
```

## regionTint

```TypeScript
regionTint: Array<SystemBarRegionTint>
```

当前已改变的所有系统栏信息。

**类型：** Array<SystemBarRegionTint>

**起始版本：** 8

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

**系统接口：** 此接口为系统接口。

## displayId

```TypeScript
displayId: long
```

当前窗口所在屏幕id，该参数应为整数。

**类型：** long

**起始版本：** 8

**系统能力：** SystemCapability.WindowManager.WindowManager.Core

**系统接口：** 此接口为系统接口。
