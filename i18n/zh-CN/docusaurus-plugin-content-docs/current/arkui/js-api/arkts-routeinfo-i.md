# RouteInfo

路由信息。

**起始版本：** 10

**废弃版本：** 13

**替代接口：** Navigation#NavPathStack

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## param

```TypeScript
param?: unknown
```

点击NavRouter跳转到NavDestination页面时，传递的参数。

**类型：** unknown

**起始版本：** 10

**废弃版本：** 13

**替代接口：** Navigation.NavPathInfo#param

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API version 11开始，该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## name

name: string

**起始版本：** 10

**废弃版本：** 13

**原子化服务API：** 从API version 11开始，该接口支持在原子化服务中使用。
