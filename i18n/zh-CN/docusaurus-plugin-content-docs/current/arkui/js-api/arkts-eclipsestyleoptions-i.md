# EclipseStyleOptions

圆形样式选项。圆形样式的显示类似月圆月缺的进度展示效果，从月牙逐渐变化至满月。 继承自[CommonProgressStyleOptions](arkts-commonprogressstyleoptions-i.md#CommonProgressStyleOptions)。

**继承实现关系：** EclipseStyleOptions继承自：CommonProgressStyleOptions。

**起始版本：** 10

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API version 11开始，该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
