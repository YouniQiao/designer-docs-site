# ColorContent

指定颜色填充内容。

**起始版本：** 15

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## ORIGIN

```TypeScript
static readonly ORIGIN: ColorContent
```

重置[fillColor](arkts-imageattribute-c.md#fillColor)接口，效果上与不设置 [fillColor](arkts-imageattribute-c.md#fillColor)一致。

**类型：** ColorContent

**起始版本：** 15

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
