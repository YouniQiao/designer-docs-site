# Constants

## StepperItemInstance

```TypeScript
declare const StepperItemInstance: StepperItemAttribute
```

Defines StepperItem Component instance.

**起始版本：** 8

**废弃版本：** 22

**替代接口：** Swiper

**原子化服务API：** 从API version 11开始，该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## StepperItem

```TypeScript
declare const StepperItem: StepperItemInterface
```

用作[Stepper]stepper组件的页面子组件。 > **说明：** > - 从API version 8开始支持，从API version 22开始废弃，建议使用[Swiper]swiper替代。 ###### 子组件 支持单个子组件。

**起始版本：** 8

**废弃版本：** 22

**替代接口：** Swiper

**原子化服务API：** 从API version 11开始，该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
