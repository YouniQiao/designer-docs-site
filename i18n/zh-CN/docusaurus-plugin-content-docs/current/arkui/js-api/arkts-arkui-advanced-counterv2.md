```TypeScript
import { CounterV2Component,CounterV2Options,CounterV2Type,CounterV2DateData } from '@kit.ArkUI';
```

## 汇总

### 类

| 名称 | 描述 |
| --- | --- |
| [CounterV2CommonOptions](arkts-counterv2commonoptions-c.md) | CounterV2CommonOptions定义了CounterV2的共通属性和事件。 |
| [CounterV2DateData](arkts-counterv2datedata-c.md) | CounterV2DateData定义了日期通用属性和方法，包括年、月、日。 |
| [CounterV2DateStyleOptions](arkts-counterv2datestyleoptions-c.md) | CounterV2DateStyleOptions定义日期内联型CounterV2的属性和事件。 继承于[CounterV2CommonOptions]{@link CounterV2CommonOptions}。 |
| [CounterV2InlineStyleOptions](arkts-counterv2inlinestyleoptions-c.md) | CounterV2InlineStyleOptions定义了数值内联型CounterV2的属性和事件。 继承于[CounterV2CommonOptions]{@link CounterV2CommonOptions}。 |
| [CounterV2NumberStyleOptions](arkts-counterv2numberstyleoptions-c.md) | CounterV2NumberStyleOptions定义了列表型和紧凑型CounterV2的属性和事件。 继承于[CounterV2InlineStyleOptions]{@link CounterV2InlineStyleOptions}。 |
| [CounterV2Options](arkts-counterv2options-c.md) | CounterV2Options定义CounterV2类型及样式。 |
