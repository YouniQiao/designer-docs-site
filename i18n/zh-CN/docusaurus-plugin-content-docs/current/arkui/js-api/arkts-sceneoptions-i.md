# SceneOptions

3D场景控制使用的场景选项

**起始版本：** 12

**系统能力：** SystemCapability.ArkUi.Graphics3D

## modelType

```TypeScript
modelType?: ModelType
```

3D渲染时的场景类型

**类型：** ModelType

**起始版本：** 12

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.ArkUi.Graphics3D

## scene

```TypeScript
scene?: ResourceStr | Scene
```

3D渲染的资源类型，3D场景控制的Scene类型

**类型：** ResourceStr | Scene

**起始版本：** 12

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.ArkUi.Graphics3D
