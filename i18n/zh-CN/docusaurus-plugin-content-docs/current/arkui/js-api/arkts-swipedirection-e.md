# SwipeDirection

```TypeScript
declare enum SwipeDirection
```

定义滑动手势的触发方向。

**起始版本：** 8

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## None

```TypeScript
None
```

任何方向均不可触发。

**起始版本：** 8

**原子化服务API：** 从API version 11开始，该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## Horizontal

```TypeScript
Horizontal
```

水平方向，手指滑动方向与x轴夹角小于45度时触发。

**起始版本：** 8

**原子化服务API：** 从API version 11开始，该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## Vertical

```TypeScript
Vertical
```

竖直方向，手指滑动方向与y轴夹角小于45度时触发。

**起始版本：** 8

**原子化服务API：** 从API version 11开始，该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## All

```TypeScript
All
```

所有方向。

**起始版本：** 8

**原子化服务API：** 从API version 11开始，该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
