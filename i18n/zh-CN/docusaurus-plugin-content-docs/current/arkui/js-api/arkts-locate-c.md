## unicodeSetting

```TypeScript
unicodeSetting: object
```

The Unicode locale key set defined by the locale. If this locale does not have a specific key set, an empty set is returned. For example: {"nu": "arab"}, which means that the numbers in the current environment use Arabic numbers.

**类型：** object

**起始版本：** 5

**模型约束：** 此接口仅可在FA模型下使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
