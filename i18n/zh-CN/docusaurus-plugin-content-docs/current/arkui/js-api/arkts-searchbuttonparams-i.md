# SearchButtonParams

Defines the SearchButton parameters.

**起始版本：** 18

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## 导入模块

```TypeScript
import { SearchParams,AtomicServiceSearch,SearchButtonParams,OperationParams,SelectParams,InputFilterParams,MenuAlignParams } from '@kit.ArkUI';
```

## options

```TypeScript
options?: SearchButtonOptions
```

Indicates the fontSize and fontColor of the search button.

**类型：** SearchButtonOptions

**起始版本：** 18

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## searchButtonValue

```TypeScript
searchButtonValue: ResourceStr
```

Indicates the text of the search button.

**类型：** ResourceStr

**起始版本：** 18

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
