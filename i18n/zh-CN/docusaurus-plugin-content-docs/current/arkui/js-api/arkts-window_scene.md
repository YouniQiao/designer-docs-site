# window_scene

Defines the WindowScene instance.

## window_scene

```TypeScript
window_scene(persistentId: number)
```

Called when the WindowScene is used.

**起始版本：** 10

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**系统接口：** 此接口为系统接口。

**参数：**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| persistentId | number | 是 | indicates the persistent identifier of WindowScene. |
