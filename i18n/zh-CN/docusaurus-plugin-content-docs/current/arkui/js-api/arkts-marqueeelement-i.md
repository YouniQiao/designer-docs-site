# MarqueeElement

The <marquee> component inserts scrolling text, which is displayed in a single line by default. When the text length exceeds the display area of the component, the marquee effect is displayed.

**继承实现关系：** MarqueeElement继承自：Element。

**起始版本：** 4

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## start

```TypeScript
start(): void
```

Starts scrolling.

**起始版本：** 4

**模型约束：** 此接口仅可在FA模型下使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## stop

```TypeScript
stop(): void
```

Stops scrolling.

**起始版本：** 4

**模型约束：** 此接口仅可在FA模型下使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
