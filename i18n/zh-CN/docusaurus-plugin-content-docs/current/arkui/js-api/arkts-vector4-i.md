# Vector4

用于表示包含x、y、z、w四个值的向量。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## w

```TypeScript
w: double
```

向量w轴方向的值。 取值范围：(-∞, +∞)。

**类型：** double

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
