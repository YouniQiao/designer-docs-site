# TitleButtonRect

标题栏上的最小化、最大化、关闭按钮矩形区域，该区域位置坐标相对窗口右上角。

**起始版本：** 11

**系统能力：** SystemCapability.Window.SessionManager

## 导入模块

```TypeScript
import { window } from '@kit.ArkUI';
```

## top

```TypeScript
top: int
```

矩形区域的上边界，单位为vp，该参数为整数。

**类型：** int

**起始版本：** 11

**原子化服务API：** 从API version 12开始，该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.Window.SessionManager

## width

```TypeScript
width: int
```

矩形区域的宽度，单位为vp，该参数为整数。

**系统能力：** SystemCapability.Window.SessionManager
