# OrientationResult

设置窗口显示方向的执行结果。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.Window.SessionManager

## 导入模块

```TypeScript
import { window } from '@kit.ArkUI';
```

## executionResult

```TypeScript
executionResult : OrientationExecutionResult
```

窗口显示方向的执行结果枚举。

**类型：** OrientationExecutionResult

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.Window.SessionManager
