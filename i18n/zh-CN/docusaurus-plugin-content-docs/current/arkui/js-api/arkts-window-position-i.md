# Position

窗口或组件的位置。

**起始版本：** 20

**系统能力：** SystemCapability.Window.SessionManager

## 导入模块

```TypeScript
import { window } from '@kit.ArkUI';
```

## x

```TypeScript
x: int
```

x坐标，单位为px，该参数应为整数。

**类型：** int

**起始版本：** 20

**系统能力：** SystemCapability.Window.SessionManager

## y

```TypeScript
y: int
```

y坐标，单位为px，该参数应为整数。

**类型：** int

**起始版本：** 20

**系统能力：** SystemCapability.Window.SessionManager
