# closeGeneratorDialog

## closeGeneratorDialog

```TypeScript
function closeGeneratorDialog(uiContext: UIContext): Promise<void>
```

Close the AI image generation task popup.

**起始版本：** 24

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**系统接口：** 此接口为系统接口。

**参数：**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| uiContext | UIContext | 是 | the context of dialog for ui display. |

**返回值：**

| 类型 | 说明 |
| --- | --- |
| Promise&lt;void> | - Returns the result of close operation. |
