# StartAnimationParams

启动动画配置。 仅对同应用的不同ability间跳转生效。 仅对全屏应用生效。

**起始版本：** 20

**系统能力：** SystemCapability.Window.SessionManager

## 导入模块

```TypeScript
import { window } from '@kit.ArkUI';
```

## type

```TypeScript
type: AnimationType
```

The type of window animation

**类型：** AnimationType

**起始版本：** 20

**系统能力：** SystemCapability.Window.SessionManager
