# CameraElement

The <camera> component provides preview and photographing functions.

**继承实现关系：** CameraElement继承自：Element。

**起始版本：** 6

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## takePhoto

```TypeScript
takePhoto(options: CameraTakePhotoOptions): void
```

Take photos with specified parameters.

**起始版本：** 6

**模型约束：** 此接口仅可在FA模型下使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

**参数：**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| options | CameraTakePhotoOptions | 是 | the parameters of camera. |
