# MaterialLevel

```TypeScript
enum MaterialLevel
```

Enumerates the material levels, which indicate the computing power level of the device. Use [getGlobalMaterialLevel]uiMaterial.getGlobalMaterialLevel() to obtain the material level of the current device.

**起始版本：** 26.0.0

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## EXQUISITE

```TypeScript
EXQUISITE = 0
```

Material level of devices with high-level computing power.

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
