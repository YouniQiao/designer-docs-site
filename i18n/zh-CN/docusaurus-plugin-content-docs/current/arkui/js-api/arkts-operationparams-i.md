# OperationParams

Function settings in the selection area (right).

**起始版本：** 18

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## 导入模块

```TypeScript
import { SearchParams,AtomicServiceSearch,SearchButtonParams,OperationParams,SelectParams,InputFilterParams,MenuAlignParams } from '@kit.ArkUI';
```

## independentItem

```TypeScript
independentItem?: OperationOption
```

Function bits independent of search area (right).

**类型：** OperationOption

**起始版本：** 18

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## auxiliaryItem

```TypeScript
auxiliaryItem?: OperationOption
```

Function bits attached to the search area (right).

**类型：** OperationOption

**起始版本：** 18

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
