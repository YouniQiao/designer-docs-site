# LayoutAlgorithm

动态布局容器[DynamicLayout]@ohos.arkui.components.ArkDynamicLayout的布局算法基础类型。 > **说明：** > > 该类型变量可以赋值具体的布局算法类对象，如[CustomLayoutAlgorithm](arkts-customlayoutalgorithm-c.md#CustomLayoutAlgorithm)类对象、 > [RowLayoutAlgorithm](arkts-rowlayoutalgorithm-c.md#RowLayoutAlgorithm)类对象等。

**起始版本：** 24

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 该接口支持在原子化服务中使用。

**卡片能力：** 该接口支持在ArkTS卡片中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
