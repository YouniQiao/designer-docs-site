# DialogCustomOptions

自定义样式对话框的选项。 对话内容作为present()方法的第一个参数提供。 不在此选项对象中。

**继承实现关系：** DialogCustomOptions继承自：DialogBaseOptions。

**起始版本：** 26.1.0

**系统能力：** SystemCapability.ArkUI.ArkUI.Full

## 导入模块

```TypeScript
import { DialogButtonOrientation,DialogState,DialogResult,DialogBaseController,DialogBaseAlignment,DialogDismissal } from '@kit.ArkUI';
```

## customStyle

```TypeScript
customStyle?: boolean
```

是否开启自定义样式。

**类型：** boolean

**起始版本：** 26.1.0

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.ArkUI.ArkUI.Full
