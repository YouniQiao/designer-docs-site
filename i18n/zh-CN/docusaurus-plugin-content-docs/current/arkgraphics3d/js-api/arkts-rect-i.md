# Rect

定义矩形.

**起始版本：** 12

**系统能力：** SystemCapability.ArkUi.Graphics3D

## width

```TypeScript
width: double
```

矩形的宽度, 单位为世界坐标系下的场景单位（例如cm、m、km等）.

**类型：** double

**起始版本：** 12

**系统能力：** SystemCapability.ArkUi.Graphics3D
