# SphereGeometry

定义球体.

**继承实现关系：** SphereGeometry继承自：GeometryDefinition。

**起始版本：** 18

**系统能力：** SystemCapability.ArkUi.Graphics3D

## radius

```TypeScript
get radius(): double
```

球体的半径, 单位为世界坐标系下的场景单位（例如cm、m、km等）.

**类型：** double

**起始版本：** 18

**系统能力：** SystemCapability.ArkUi.Graphics3D
