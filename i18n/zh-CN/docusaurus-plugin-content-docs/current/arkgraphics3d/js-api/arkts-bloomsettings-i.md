# BloomSettings

定义泛光参数.

**起始版本：** 18

**系统能力：** SystemCapability.ArkUi.Graphics3D

## thresholdHard

```TypeScript
thresholdHard?: double
```

泛光硬阈值.

**类型：** double

**起始版本：** 18

**系统能力：** SystemCapability.ArkUi.Graphics3D

## scatter

```TypeScript
scatter?: double
```

扩散（泛光扩散量）. (1.0 full spread / default).

**类型：** double

**起始版本：** 18

**系统能力：** SystemCapability.ArkUi.Graphics3D

## scaleFactor

```TypeScript
scaleFactor?: double
```

缩放因子. 控制缩放和泛光扩散量. 减少降缩放和上缩放步骤. 取值范围0-1. 值为0.5时减半缩放步骤.

**类型：** double

**起始版本：** 18

**系统能力：** SystemCapability.ArkUi.Graphics3D

## thresholdSoft

```TypeScript
thresholdSoft?: double
```

泛光软阈值.

**类型：** double

**起始版本：** 18

**系统能力：** SystemCapability.ArkUi.Graphics3D
