# ClickResponseTime

```TypeScript
type ClickResponseTime = 'Short' | 'Medium' | 'Long'
```

Defines the length of time for a click.

**起始版本：** 11

**系统能力：** SystemCapability.BarrierFree.Accessibility.Core

**系统接口：** 此接口为系统接口。

| 类型 | 说明 |
| --- | --- |
| 'Short' | Short (default). |
| 'Medium' | Medium. |
| 'Long' | Long. |
