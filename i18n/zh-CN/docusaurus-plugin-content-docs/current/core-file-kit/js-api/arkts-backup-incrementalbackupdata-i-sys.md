# IncrementalBackupData

Corresponds to an incremental application, including its last incremental time and incremental list.

**继承实现关系：** IncrementalBackupData继承自：IncrementalBackupTime， FileManifestData， BackupParams， BackupPriority。

**起始版本：** 12

**系统能力：** SystemCapability.FileManagement.StorageService.Backup

**系统接口：** 此接口为系统接口。

## 导入模块

```TypeScript
import { backup } from '@kit.CoreFileKit';
```
