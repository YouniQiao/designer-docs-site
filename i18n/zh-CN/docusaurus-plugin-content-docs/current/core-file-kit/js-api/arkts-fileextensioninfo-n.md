# fileExtensionInfo

该模块提供公共文件访问和管理模块中RootInfo与FileInfo中部分属性值的定义。

**起始版本：** 9

**废弃版本：** 23

**系统能力：** SystemCapability.FileManagement.UserFileService

**系统接口：** 此接口为系统接口。

## 导入模块

```TypeScript
import { fileExtensionInfo } from '@kit.CoreFileKit';
```
