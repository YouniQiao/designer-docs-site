# openSync

## openSync

```TypeScript
declare function openSync(path: string, mode?: number): File
```

以同步方法打开文件或目录。支持使用URI打开文件。

**起始版本：** 9

**原子化服务API：** 从API version 11开始，该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.FileManagement.File.FileIO

**参数：**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| path | string | 是 | 打开文件或目录的应用沙箱路径或URI。 |
| mode | number | 否 |  |

**返回值：**

| 类型 | 说明 |
| --- | --- |
| File | 打开的File对象。 |
