# AudioSelectOptions

音频选择选项。

**起始版本：** 23

**系统能力：** SystemCapability.FileManagement.UserFileService

## 导入模块

```TypeScript
import { picker } from '@kit.CoreFileKit';
```

## maxSelectNumber

```TypeScript
maxSelectNumber?: int
```

选择文件最大个数，默认值为1，上限为500个，有效值范围1-500。

**类型：** int

**起始版本：** 23

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.FileManagement.UserFileService
