# AssetStatus

```TypeScript
enum AssetStatus
```

Enumerates the asset statuses. Use the enum name rather than the enum value.

**起始版本：** 10

**系统能力：** SystemCapability.DistributedDataManager.RelationalStore.Core

```TypeScript

```
