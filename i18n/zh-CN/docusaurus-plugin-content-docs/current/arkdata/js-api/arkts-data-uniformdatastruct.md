# @ohos.data.uniformDataStruct

As a part of the Unified Data Management Framework (UDMF), the **uniformDataStruct** module provides data structs corresponding to certain [UniformDataTypes]{@link @ohos.data.uniformTypeDescriptor:uniformTypeDescriptor.UniformDataType} for service scenarios of many-to-many data sharing across applications. It helps simplify data interaction and reduce the data type adaptation workload.

**起始版本：** 12

**系统能力：** SystemCapability.DistributedDataManager.UDMF.Core

## 导入模块

```TypeScript
import { uniformDataStruct } from '@kit.ArkData';
```

## 汇总

### 接口

| 名称 | 描述 |
| --- | --- |
| [ContentForm](arkts-uniformdatastruct-contentform-i.md) | Represents data of the content widget type. |
| [FileUri](arkts-uniformdatastruct-fileuri-i.md) | Represents data of the file URI type. |
| [Form](arkts-uniformdatastruct-form-i.md) | Represents data of the widget type defined by the system. |
| [HTML](arkts-uniformdatastruct-html-i.md) | Represents data of the HTML type. |
| [Hyperlink](arkts-uniformdatastruct-hyperlink-i.md) | Represents data of the hyperlink type. |
| [OpenHarmonyAppItem](arkts-uniformdatastruct-openharmonyappitem-i.md) | Represents data of the home screen icon type defined by the system. |
| [PixelMap](arkts-uniformdatastruct-pixelmap-i.md) | Represents data of the pixel map type defined by the system. |
| [PlainText](arkts-uniformdatastruct-plaintext-i.md) | Represents data of the plain text type. |
