# Constants

Defines the KV store constants.

**起始版本：** 7

**废弃版本：** 9

**替代接口：** ohos.data.distributedKVStore.Constants

**系统能力：** SystemCapability.DistributedDataManager.KVStore.Core

## 汇总

### 常量

| 名称 | 描述 |
| --- | --- |
| [MAX_BATCH_SIZE](arkts-constants-con.md#MAX_BATCH_SIZE) | max batch operation size. Maximum number of batch operations. |
| [MAX_KEY_LENGTH](arkts-constants-con.md#MAX_KEY_LENGTH) | max key length. Maximum length of a key in the KV store, in bytes. |
| [MAX_KEY_LENGTH_DEVICE](arkts-constants-con.md#MAX_KEY_LENGTH_DEVICE) | max device coordinate key length. Maximum length of a device key, in bytes. |
| [MAX_QUERY_LENGTH](arkts-constants-con.md#MAX_QUERY_LENGTH) | max query length. Maximum query length, in bytes. |
| [MAX_STORE_ID_LENGTH](arkts-constants-con.md#MAX_STORE_ID_LENGTH) | max store id length. Maximum length of a KV store ID, in bytes. |
| [MAX_VALUE_LENGTH](arkts-constants-con.md#MAX_VALUE_LENGTH) | max value length. Maximum length of a value in the KV store, in bytes. |
