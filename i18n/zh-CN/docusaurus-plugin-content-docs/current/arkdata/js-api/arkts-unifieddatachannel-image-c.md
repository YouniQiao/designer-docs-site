# Image

Represents the image data. It is a child class of [File]unifiedDataChannel.File and is used to describe images.

**继承实现关系：** Image继承自：File。

**起始版本：** 10

**系统能力：** SystemCapability.DistributedDataManager.UDMF.Core

## 导入模块

```TypeScript
import { unifiedDataChannel } from '@kit.ArkData';
```

## imageUri

```TypeScript
get imageUri(): string
```

Indicates the uri of image

**类型：** string

**起始版本：** 10

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 从API version 11开始，该接口支持在原子化服务中使用。
