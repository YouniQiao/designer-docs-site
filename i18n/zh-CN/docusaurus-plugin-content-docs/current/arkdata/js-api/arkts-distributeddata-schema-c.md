# Schema

Defines the schema of a KV store. You can create a **Schema** object and place it in [Options]distributedData.Options when creating or opening a KV store.

**起始版本：** 8

**废弃版本：** 9

**替代接口：** ohos.data.distributedKVStore.Schema

**系统能力：** SystemCapability.DistributedDataManager.KVStore.DistributedKVStore
