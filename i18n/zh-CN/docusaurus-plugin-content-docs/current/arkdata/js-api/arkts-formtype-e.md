# FormType

```TypeScript
declare enum FormType
```

Enumerates the form card types.

**起始版本：** 20

**系统能力：** SystemCapability.DistributedDataManager.UDMF.Core

## TYPE_BIG

```TypeScript
TYPE_BIG = 0
```

Big form card, with fixed size of 4 x 4.

**起始版本：** 20

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.DistributedDataManager.UDMF.Core
