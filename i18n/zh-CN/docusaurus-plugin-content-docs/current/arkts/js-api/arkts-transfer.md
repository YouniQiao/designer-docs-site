# @ohos.transfer

transfer模块提供子系统类或接口的转换能力。

**起始版本：** 23

**系统能力：** SystemCapability.Utils.Lang

## 导入模块

```TypeScript
import { transfer } from '@kit.ArkTS';
```

## 汇总

### 函数

| 名称 | 描述 |
| --- | --- |
| [transferDynamic](arkts-transfer-transferdynamic-f.md#transferDynamic-1) | 将1.2对象转换为1.0对象。 |
| [transferStatic](arkts-transfer-transferstatic-f.md#transferStatic-1) | 将1.0对象转换为1.2对象。 |
