# @ohos.convertxml

本模块提供将XML文本转换为JavaScript对象的解析能力。

**起始版本：** 8

**系统能力：** SystemCapability.Utils.Lang

## 导入模块

```TypeScript
import { convertxml } from '@kit.ArkTS';
```
