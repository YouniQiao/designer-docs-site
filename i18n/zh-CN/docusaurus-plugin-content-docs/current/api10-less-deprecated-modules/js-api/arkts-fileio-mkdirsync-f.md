# mkdirSync

## mkdirSync

```TypeScript
declare function mkdirSync(path: string, mode?: number): void
```

以同步方法创建目录。

**起始版本：** 6

**废弃版本：** 9

**替代接口：** @ohos.file.fs:mkdirSync

**系统能力：** SystemCapability.FileManagement.File.FileIO

**参数：**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| path | string | 是 | 待创建目录的应用沙箱路径。 |
| mode | number | 否 |  |
