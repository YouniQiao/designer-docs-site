# @ohos.ability.wantConstant

wantConstant模块提供want中操作want常数和解释Flags说明的能力。

**起始版本：** 6

**废弃版本：** 9

**替代接口：** ohos.app.ability.wantConstant/wantConstant

**系统能力：** SystemCapability.Ability.AbilityBase

## 汇总

### 枚举

| 名称 | 描述 |
| --- | --- |
| [Action](arkts-wantconstant-action-e.md) | want操作的常数。用于表示要执行的通用操作。 |
| [Entity](arkts-wantconstant-entity-e.md) | want实体的常数。用于表示目标Ability额外的类别信息。 |
| [Flags](arkts-wantconstant-flags-e.md) | Flags说明。用于表示处理Want的方式。 |
