# IDeliveryShortMessageCallback

Provides the callback for the SMS message delivery report.

**起始版本：** 6

**系统能力：** SystemCapability.Telephony.SmsMms

## 导入模块

```TypeScript
import { sms } from '@kit.TelephonyKit';
```

## pdu

```TypeScript
pdu: Array<int>
```

Indicates the SMS delivery report.

**类型：** Array<int>

**起始版本：** 6

**系统能力：** SystemCapability.Telephony.SmsMms
