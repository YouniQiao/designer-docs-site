# VoipCallAttribute

Indicates the voip call detail information.

**起始版本：** 11

**系统能力：** SystemCapability.Telephony.CallManager

**系统接口：** 此接口为系统接口。

## 导入模块

```TypeScript
import { call } from '@kit.TelephonyKit';
```

## voipCallId

```TypeScript
voipCallId: string
```

Indicates the identifier of the voip call.

**类型：** string

**起始版本：** 11

**系统能力：** SystemCapability.Telephony.CallManager

**系统接口：** 此接口为系统接口。

## showBannerForIncomingCall

