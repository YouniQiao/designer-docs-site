# CBConfigOptions

Defines the cell broadcast configuration options.

**起始版本：** 7

**系统能力：** SystemCapability.Telephony.SmsMms

**系统接口：** 此接口为系统接口。

## 导入模块

```TypeScript
import { sms } from '@kit.TelephonyKit';
```

## enable

```TypeScript
enable: boolean
```

Indicates whether to enable cell broadcast.

**类型：** boolean

**起始版本：** 7

**系统能力：** SystemCapability.Telephony.SmsMms

**系统接口：** 此接口为系统接口。
