# VCardBuilderOptions

Indicates the options for VCard export.

**起始版本：** 23

**系统能力：** SystemCapability.Telephony.CoreService

## 导入模块

```TypeScript
import { vcard } from '@kit.TelephonyKit';
```

## charset

```TypeScript
charset?: string
```

Indicates the Encoding format.

**类型：** string

**起始版本：** 23

**系统能力：** SystemCapability.Telephony.CoreService

## cardType

```TypeScript
cardType?: VCardType
```

Indicates the VCard types.

**类型：** VCardType

**起始版本：** 23

**系统能力：** SystemCapability.Telephony.CoreService
