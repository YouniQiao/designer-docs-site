# CallSessionEvent

Indicates the call session event of a video call.

**起始版本：** 11

**系统能力：** SystemCapability.Telephony.CallManager

**系统接口：** 此接口为系统接口。

## 导入模块

```TypeScript
import { call } from '@kit.TelephonyKit';
```

## callId

```TypeScript
callId: int
```

**起始版本：** 11

**系统接口：** 此接口为系统接口。
