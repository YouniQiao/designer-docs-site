# CallEventOptions

Indicates the options of call event.

**起始版本：** 8

**系统能力：** SystemCapability.Telephony.CallManager

**系统接口：** 此接口为系统接口。

## 导入模块

```TypeScript
import { call } from '@kit.TelephonyKit';
```

## eventId

eventId: CallAbilityEventId

**类型：** CallAbilityEventId

**起始版本：** 8
