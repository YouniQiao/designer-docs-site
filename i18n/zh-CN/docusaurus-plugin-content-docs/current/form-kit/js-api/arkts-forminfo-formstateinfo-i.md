# FormStateInfo

Provides state information about a form.

**起始版本：** 9

**系统能力：** SystemCapability.Ability.Form

## 导入模块

```TypeScript
import { formInfo } from '@kit.FormKit';
```

## want

```TypeScript
want: Want
```

Obtains the want form .

**类型：** Want

**起始版本：** 9

**原子化服务API：** 从API version 11开始，该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.Ability.Form

## formState

```TypeScript
formState: FormState
```

Obtains the form state.

**类型：** FormState

**起始版本：** 9

**原子化服务API：** 从API version 11开始，该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.Ability.Form
