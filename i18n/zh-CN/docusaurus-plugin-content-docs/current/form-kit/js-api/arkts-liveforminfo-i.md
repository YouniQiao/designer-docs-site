# LiveFormInfo

Provides information about a live form.

**起始版本：** 20

**系统能力：** SystemCapability.Ability.Form

## 导入模块

```TypeScript
import { LiveFormInfo } from '@kit.FormKit';
```

## formId

```TypeScript
formId: string
```

The form id of the live form.

**类型：** string

**起始版本：** 20

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.Ability.Form

## rect

```TypeScript
rect: formInfo.Rect
```

The live form display area.

**类型：** formInfo.Rect

**起始版本：** 20

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.Ability.Form

## borderRadius

```TypeScript
borderRadius: double
```

The form border radius. Unit: vp, The value must be greater than or equal to 0.

**类型：** double

**起始版本：** 20

**模型约束：** 此接口仅可在Stage模型下使用。

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.Ability.Form
