# Region

表示区域信息。 继承自[lang.ISendable](docroot://arkts-utils/arkts-sendable.md#isendable)。

**继承实现关系：** Region继承自：lang.ISendable。

**起始版本：** 12

**系统能力：** SystemCapability.Multimedia.Image.Core

## 导入模块

```TypeScript
import { sendableImage } from '@kit.ImageKit';
```
