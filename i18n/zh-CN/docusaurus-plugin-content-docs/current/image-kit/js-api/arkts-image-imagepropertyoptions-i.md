# ImagePropertyOptions

表示查询图片属性的索引。

**起始版本：** 11

**系统能力：** SystemCapability.Multimedia.Image.ImageSource

## 导入模块

```TypeScript
import { image } from '@kit.ImageKit';
```

## defaultValue

```TypeScript
defaultValue?: string
```

默认属性值。默认值为空。

**类型：** string

**起始版本：** 11

**系统能力：** SystemCapability.Multimedia.Image.ImageSource

## index

```TypeScript
index?: int
```

图片序号。默认值为0。

**类型：** int

**起始版本：** 11

**系统能力：** SystemCapability.Multimedia.Image.ImageSource
