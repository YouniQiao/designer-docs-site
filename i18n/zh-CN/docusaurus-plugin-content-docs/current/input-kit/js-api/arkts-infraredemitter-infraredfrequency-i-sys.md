# InfraredFrequency

红外信号的频率范围。

**起始版本：** 12

**系统能力：** SystemCapability.MultimodalInput.Input.InfraredEmitter

**系统接口：** 此接口为系统接口。

## 导入模块

```TypeScript
import { infraredEmitter } from '@kit.InputKit';
```
