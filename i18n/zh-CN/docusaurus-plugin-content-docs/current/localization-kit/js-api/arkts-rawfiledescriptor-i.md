# RawFileDescriptor

本模块提供rawfile文件所在hap的descriptor信息。

**起始版本：** 8

**系统能力：** SystemCapability.Global.ResourceManager

## offset

```TypeScript
offset: long
```

起始偏移量。

**类型：** long

**起始版本：** 8

**原子化服务API：** 从API version 11开始，该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.Global.ResourceManager

## length

```TypeScript
length: long
```

文件长度。

**类型：** long

**起始版本：** 8

**原子化服务API：** 从API version 11开始，该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.Global.ResourceManager

## fd

```TypeScript
fd: int
```

文件描述符。

**类型：** int

**起始版本：** 8

**原子化服务API：** 从API version 11开始，该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.Global.ResourceManager
