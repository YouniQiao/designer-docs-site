# TimeZoneCityItem

Represents a time zone and city combination item.

**起始版本：** 10

**系统能力：** SystemCapability.Global.I18n

**系统接口：** 此接口为系统接口。

## 导入模块

```TypeScript
import { i18n } from '@kit.LocalizationKit';
```

## zoneDisplayName

```TypeScript
zoneDisplayName: string
```

**类型：** string

**起始版本：** 10
