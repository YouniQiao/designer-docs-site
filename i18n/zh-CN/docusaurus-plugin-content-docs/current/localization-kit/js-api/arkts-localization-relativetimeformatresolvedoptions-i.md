# RelativeTimeFormatResolvedOptions

相对时间格式化对象的格式化配置项。

**起始版本：** 8

**废弃版本：** 20

**替代接口：** return_value)

**系统能力：** SystemCapability.Global.I18n

## locale

```TypeScript
locale: string
```

表示区域ID的字符串，包括语言以及可选的脚本和区域。

**类型：** string

**起始版本：** 8

**废弃版本：** 20

**替代接口：** locale)

**元服务API：** 从API版本12开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Global.I18n
