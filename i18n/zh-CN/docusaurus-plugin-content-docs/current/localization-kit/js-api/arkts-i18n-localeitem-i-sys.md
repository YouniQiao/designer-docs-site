# LocaleItem

Represents the locale information, which consists of the language, script, and country/region.

**起始版本：** 10

**系统能力：** SystemCapability.Global.I18n

**系统接口：** 此接口为系统接口。

## 导入模块

```TypeScript
import { i18n } from '@kit.LocalizationKit';
```

## localName

```TypeScript
localName?: string
```

Local name of the ID.

**类型：** string

**起始版本：** 10

**系统能力：** SystemCapability.Global.I18n

**系统接口：** 此接口为系统接口。

## suggestionType

```TypeScript
suggestionType: SuggestionType
```

Language or country/region suggestion type.

**类型：** SuggestionType

**起始版本：** 10

**系统能力：** SystemCapability.Global.I18n

**系统接口：** 此接口为系统接口。

## displayName

```TypeScript
displayName: string
```

Representation of ID in the specified locale in SystemLocaleManager.

**类型：** string

**起始版本：** 10

**系统能力：** SystemCapability.Global.I18n

**系统接口：** 此接口为系统接口。

## id

```TypeScript
id: string
```

Language code or country/region code, for example, "zh" or "CN".

**类型：** string

**起始版本：** 10

**系统能力：** SystemCapability.Global.I18n

**系统接口：** 此接口为系统接口。
