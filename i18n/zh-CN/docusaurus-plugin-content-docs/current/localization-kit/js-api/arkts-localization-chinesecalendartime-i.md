# ChineseCalendarTime

农历时间对象。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.Global.I18n

## cyclicalYear

```TypeScript
cyclicalYear: number
```

农历的干支年。 取值范围：[1, 60]。

**类型：** number

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**元服务API：** 从API版本26.0.0开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Global.I18n
