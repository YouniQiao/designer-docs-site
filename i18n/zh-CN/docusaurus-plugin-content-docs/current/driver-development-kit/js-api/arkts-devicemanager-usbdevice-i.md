# USBDevice

USB设备信息，继承自[Device]deviceManager.queryDevices。

**继承实现关系：** USBDevice继承自：Device。

**起始版本：** 10

**系统能力：** SystemCapability.Driver.ExternalDevice

## 导入模块

```TypeScript
import { deviceManager } from '@kit.DriverDevelopmentKit';
```
