# participate

## participate

```TypeScript
function participate(config: HiRetrievalConfig): void
```

Participate the HiRetrieval project with given HiRetrievalConfig.

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.HiviewDFX.HiRetrieval

**参数：**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| config | HiRetrievalConfig | 是 | The config set by the developers. |
