# HeartRateResponse

Describes the heart rate sensor data. It extends from [Response]sensor.Response.

**继承实现关系：** HeartRateResponse继承自：Response。

**起始版本：** 8

**系统能力：** SystemCapability.Sensors.Sensor

## 导入模块

```TypeScript
import { sensor } from '@kit.SensorServiceKit';
```

## heartRate

```TypeScript
heartRate: double
```

Heart rate, in beats per minute (bpm).

**类型：** double

**起始版本：** 8

**系统能力：** SystemCapability.Sensors.Sensor
