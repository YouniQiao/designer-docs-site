# HumidityResponse

Describes the humidity sensor data. It extends from [Response]sensor.Response.

**继承实现关系：** HumidityResponse继承自：Response。

**起始版本：** 8

**系统能力：** SystemCapability.Sensors.Sensor

## 导入模块

```TypeScript
import { sensor } from '@kit.SensorServiceKit';
```

## humidity

```TypeScript
humidity: double
```

Ambient relative humidity, in a percentage (%).

**类型：** double

**起始版本：** 8

**系统能力：** SystemCapability.Sensors.Sensor
