# BarometerResponse

Describes the barometer sensor data. It extends from [Response]sensor.Response.

**继承实现关系：** BarometerResponse继承自：Response。

**起始版本：** 8

**系统能力：** SystemCapability.Sensors.Sensor

## 导入模块

```TypeScript
import { sensor } from '@kit.SensorServiceKit';
```

## pressure

pressure: double
