# WearDetectionResponse

Describes the wear detection sensor data. It extends from [Response]sensor.Response.

**继承实现关系：** WearDetectionResponse继承自：Response。

**起始版本：** 8

**系统能力：** SystemCapability.Sensors.Sensor

## 导入模块

```TypeScript
import { sensor } from '@kit.SensorServiceKit';
```

## value

```TypeScript
value: double
```

Whether the device is being worn. The value **1** means that the device is being worn, and **0** means the opposite.

**类型：** double

**起始版本：** 8

**系统能力：** SystemCapability.Sensors.Sensor
