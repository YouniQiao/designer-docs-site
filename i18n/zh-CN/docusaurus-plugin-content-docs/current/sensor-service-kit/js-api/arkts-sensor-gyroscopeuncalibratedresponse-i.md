# GyroscopeUncalibratedResponse

Describes the uncalibrated gyroscope sensor data. It extends from [Response]sensor.Response.

**继承实现关系：** GyroscopeUncalibratedResponse继承自：Response。

**起始版本：** 8

**系统能力：** SystemCapability.Sensors.Sensor

## 导入模块

```TypeScript
import { sensor } from '@kit.SensorServiceKit';
```

## biasY

```TypeScript
biasY: double
```

Uncalibrated angular velocity bias of rotation around the y-axis of the device, in rad/s.

**类型：** double

**起始版本：** 8

**系统能力：** SystemCapability.Sensors.Sensor
