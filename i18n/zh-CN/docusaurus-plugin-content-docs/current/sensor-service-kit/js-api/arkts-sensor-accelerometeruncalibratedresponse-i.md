# AccelerometerUncalibratedResponse

Describes the uncalibrated acceleration sensor data. It extends from [Response]sensor.Response.

**继承实现关系：** AccelerometerUncalibratedResponse继承自：Response。

**起始版本：** 8

**系统能力：** SystemCapability.Sensors.Sensor

## 导入模块

```TypeScript
import { sensor } from '@kit.SensorServiceKit';
```

## biasY

```TypeScript
biasY: double
```

Uncalibrated acceleration bias along the y-axis of the device, in m/s?.

**类型：** double
