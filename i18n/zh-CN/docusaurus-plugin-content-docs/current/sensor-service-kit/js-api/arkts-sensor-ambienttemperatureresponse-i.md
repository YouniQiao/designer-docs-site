# AmbientTemperatureResponse

Describes the ambient temperature sensor data. It extends from [Response]sensor.Response.

**继承实现关系：** AmbientTemperatureResponse继承自：Response。

**起始版本：** 8

**系统能力：** SystemCapability.Sensors.Sensor

## 导入模块

```TypeScript
import { sensor } from '@kit.SensorServiceKit';
```

## temperature

```TypeScript
temperature: double
```

Ambient temperature, in degree Celsius.

**类型：** double

**起始版本：** 8

**系统能力：** SystemCapability.Sensors.Sensor
