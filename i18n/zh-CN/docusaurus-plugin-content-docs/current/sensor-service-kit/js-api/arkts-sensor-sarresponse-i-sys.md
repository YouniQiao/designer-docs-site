# SarResponse

Describes the SAR sensor data. It extends from [Response](arkts-sensor-response-i.md#Response).

**继承实现关系：** SarResponse继承自：Response。

**起始版本：** 10

**系统能力：** SystemCapability.Sensors.Sensor

**系统接口：** 此接口为系统接口。

## 导入模块

```TypeScript
import { sensor } from '@kit.SensorServiceKit';
```

## absorptionRatio

```TypeScript
absorptionRatio: double
```

Absorption ratio, in W/kg.

**类型：** double

**起始版本：** 10

**系统能力：** SystemCapability.Sensors.Sensor

**系统接口：** 此接口为系统接口。
