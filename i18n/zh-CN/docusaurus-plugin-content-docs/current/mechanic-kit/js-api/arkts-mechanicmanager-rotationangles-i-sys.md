# RotationAngles

The rotion angles, relative to the current position.

**起始版本：** 20

**系统能力：** SystemCapability.Mechanic.Core

**系统接口：** 此接口为系统接口。

## 导入模块

```TypeScript
import { mechanicManager } from '@kit.MechanicKit';
```

## roll

```TypeScript
roll?: double
```

Roll angle, ranging from -2*Math.PI to 2*Math.PI, measured in radians.

**类型：** double

**起始版本：** 20

**系统能力：** SystemCapability.Mechanic.Core

**系统接口：** 此接口为系统接口。
