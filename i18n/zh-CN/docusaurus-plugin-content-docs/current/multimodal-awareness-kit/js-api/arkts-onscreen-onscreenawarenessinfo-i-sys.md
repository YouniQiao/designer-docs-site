# OnscreenAwarenessInfo

Returns the list of onscreen awareness information.

**起始版本：** 23

**系统能力：** SystemCapability.MultimodalAwareness.OnScreenAwareness

**系统接口：** 此接口为系统接口。

## 导入模块

```TypeScript
import { onScreen } from '@kit.MultimodalAwarenessKit';
```
