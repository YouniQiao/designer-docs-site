# DeviceRotationRadian

**起始版本：** 20

**系统能力：** SystemCapability.MultimodalAwareness.DeviceStatus

**系统接口：** 此接口为系统接口。

```TypeScript
import { deviceStatus } from '@kit.MultimodalAwarenessKit';
```
