# ComfortReminderData

Defines comfort reminder data.

**继承实现关系：** ComfortReminderData继承自：UserStatusData。

**起始版本：** 26.0.0

**系统能力：** SystemCapability.MultimodalAwareness.UserStatus

**系统接口：** 此接口为系统接口。

## 导入模块

import { userStatus } from '@kit.MultimodalAwarenessKit';
