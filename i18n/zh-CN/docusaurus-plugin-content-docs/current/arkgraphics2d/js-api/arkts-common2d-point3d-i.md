# Point3d

三维的坐标点。继承自[Point]common2D.Point。

**继承实现关系：** Point3d继承自：Point。

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing

## 导入模块

```TypeScript
import { common2D } from '@kit.ArkGraphics2D';
```

z: double
