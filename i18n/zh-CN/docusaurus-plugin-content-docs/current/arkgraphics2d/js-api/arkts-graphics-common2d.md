# @ohos.graphics.common2D

本模块定义了一些2D图形领域的通用数据类型。 > **说明：** > > - 本模块使用屏幕物理像素单位px。

**起始版本：** 11

**系统能力：** SystemCapability.Graphics.Drawing

## 导入模块

```TypeScript
import { common2D } from '@kit.ArkGraphics2D';
```

## 汇总

### 接口

| 名称 | 描述 |
| --- | --- |
| [Color](arkts-common2d-color-i.md) | ARGB格式的颜色描述。 |
| [Color4f](arkts-common2d-color4f-i.md) | ARGB格式的颜色描述。 |
| [Point](arkts-common2d-point-i.md) | 坐标点。 |
| [Point3d](arkts-common2d-point3d-i.md) | 三维的坐标点。继承自[Point]{@link common2D.Point}。 |
| [Rect](arkts-common2d-rect-i.md) | 矩形区域，通过2个坐标点可以描述出一个矩形区域，这2个点分别认为是矩形区域的左上角点与右下角点。 |
