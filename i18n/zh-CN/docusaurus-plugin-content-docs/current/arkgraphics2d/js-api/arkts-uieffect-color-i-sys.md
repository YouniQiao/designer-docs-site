# Color

RGBA格式的颜色描述。

**起始版本：** 20

**系统能力：** SystemCapability.Graphics.Drawing

**系统接口：** 此接口为系统接口。

## 导入模块

```TypeScript
import { uiEffect } from '@kit.ArkGraphics2D';
```

## red

```TypeScript
red: double
```

颜色的R分量（红色）。值大于等于0，当值小于0时无效。

**类型：** double

**起始版本：** 20

**系统能力：** SystemCapability.Graphics.Drawing

**系统接口：** 此接口为系统接口。
