# WebGLContextAttributes

WebGL 1.0

**起始版本：** 7

**系统能力：** SystemCapability.Graphic.Graphic2D.WebGL

## depth

```TypeScript
depth?: boolean
```

If true, the drawing buffer has a depth buffer of at least 16 bits.

**类型：** boolean

**起始版本：** 7

**模型约束：** 此接口仅可在FA模型下使用。

**系统能力：** SystemCapability.Graphic.Graphic2D.WebGL
