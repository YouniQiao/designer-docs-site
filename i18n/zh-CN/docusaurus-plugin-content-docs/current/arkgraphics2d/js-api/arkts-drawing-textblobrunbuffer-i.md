# TextBlobRunBuffer

描述一行文字中具有相同属性的连续字形。

**起始版本：** 11

**系统能力：** SystemCapability.Graphics.Drawing

## 导入模块

```TypeScript
import { drawing } from '@kit.ArkGraphics2D';
```

## positionY

```TypeScript
positionY: double
```

文本的起点y轴坐标，该参数为浮点数。

**类型：** double

**起始版本：** 11

**系统能力：** SystemCapability.Graphics.Drawing

## glyph

```TypeScript
glyph: int
```

存储文字的索引，该参数为整数，传入浮点类型时向下取整。

**类型：** int

**起始版本：** 11

**系统能力：** SystemCapability.Graphics.Drawing

## positionX

```TypeScript
positionX: double
```

文本的起点x轴坐标，该参数为浮点数。

**类型：** double

**起始版本：** 11

**系统能力：** SystemCapability.Graphics.Drawing
