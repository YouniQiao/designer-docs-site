# RectStyle

矩形框样式。

**起始版本：** 12

**系统能力：** SystemCapability.Graphics.Drawing

## 导入模块

```TypeScript
import { text } from '@kit.ArkGraphics2D';
```

## leftBottomRadius

```TypeScript
leftBottomRadius: double
```

矩形框的左下半径，单位为物理像素px。

**类型：** double

**起始版本：** 12

**原子化服务API：** 从API version 22开始，该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.Graphics.Drawing
