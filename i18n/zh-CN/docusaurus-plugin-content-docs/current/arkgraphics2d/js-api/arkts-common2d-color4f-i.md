# Color4f

ARGB格式的颜色描述。

**起始版本：** 20

**系统能力：** SystemCapability.Graphics.Drawing

## 导入模块

```TypeScript
import { common2D } from '@kit.ArkGraphics2D';
```

## red

```TypeScript
red: double
```

颜色的R分量（红色），值是0.0~1.0的浮点数。

**类型：** double

**起始版本：** 20

**系统能力：** SystemCapability.Graphics.Drawing

## green

```TypeScript
green: double
```

颜色的G分量（绿色），值是0.0~1.0的浮点数。

**类型：** double

**起始版本：** 20

**系统能力：** SystemCapability.Graphics.Drawing

## blue

```TypeScript
blue: double
```

颜色的B分量（蓝色），值是0.0~1.0的浮点数。

**类型：** double

**起始版本：** 20

**系统能力：** SystemCapability.Graphics.Drawing

## alpha

```TypeScript
alpha: double
```

颜色的A分量（透明度），值是0.0~1.0的浮点数。

**类型：** double

**起始版本：** 20

**系统能力：** SystemCapability.Graphics.Drawing
