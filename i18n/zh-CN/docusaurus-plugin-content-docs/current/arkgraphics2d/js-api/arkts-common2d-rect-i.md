# Rect

矩形区域，通过2个坐标点可以描述出一个矩形区域，这2个点分别认为是矩形区域的左上角点与右下角点。

**起始版本：** 11

**系统能力：** SystemCapability.Graphics.Drawing

## 导入模块

```TypeScript
import { common2D } from '@kit.ArkGraphics2D';
```

## top

```TypeScript
top: double
```

矩形区域的左上角纵坐标，浮点数。

**类型：** double

**起始版本：** 11

**原子化服务API：** 从API version 22开始，该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.Graphics.Drawing
