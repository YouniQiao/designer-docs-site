# FontVariationAxis

字体可变轴信息。

**起始版本：** 24

**系统能力：** SystemCapability.Graphics.Drawing

## 导入模块

```TypeScript
import { text } from '@kit.ArkGraphics2D';
```

## localName

```TypeScript
localName: string
```

字体可变轴的本地化名称，可以为空。

**类型：** string

**起始版本：** 24

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.Graphics.Drawing

## flags

```TypeScript
flags: int
```

字体可变轴的标志位，用于标记某个可变轴是否应该对用户隐藏，值为0或1。值为0时表示该轴对用户可见，值为1时表示该轴应隐藏。

**类型：** int

**起始版本：** 24

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.Graphics.Drawing
