# TrustedPairedDevice

Describes device of cloud pair.

**起始版本：** 15

**系统能力：** SystemCapability.Communication.Bluetooth.Core

**系统接口：** 此接口为系统接口。

## 导入模块

```TypeScript
import { connection } from '@kit.ConnectivityKit';
```

## serviceType

```TypeScript
serviceType: string
```

Indicates the service type of the peripheral.

**类型：** string

**起始版本：** 15

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Communication.Bluetooth.Core
