# @ohos.bluetooth.hfp

Provides methods to accessing bluetooth call-related capabilities.

**起始版本：** 10

**系统能力：** SystemCapability.Communication.Bluetooth.Core

## 导入模块

```TypeScript
import { hfp } from '@kit.ConnectivityKit';
```
