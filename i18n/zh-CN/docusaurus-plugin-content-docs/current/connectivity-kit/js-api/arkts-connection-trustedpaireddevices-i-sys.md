# TrustedPairedDevices

Describes the cloud pair device.

**起始版本：** 15

**系统能力：** SystemCapability.Communication.Bluetooth.Core

**系统接口：** 此接口为系统接口。

## 导入模块

```TypeScript
import { connection } from '@kit.ConnectivityKit';
```

trustedPairedDevices: Array<TrustedPairedDevice>
