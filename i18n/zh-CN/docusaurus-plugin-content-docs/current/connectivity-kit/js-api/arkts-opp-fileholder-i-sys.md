# FileHolder

Describes the file info for transfer

**起始版本：** 16

**系统能力：** SystemCapability.Communication.Bluetooth.Core

**系统接口：** 此接口为系统接口。

## 导入模块

```TypeScript
import { opp } from '@kit.ConnectivityKit';
```
