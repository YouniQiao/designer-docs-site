# isNfcOpen

## isNfcOpen

```TypeScript
function isNfcOpen(): boolean
```

Checks whether NFC is enabled.

**起始版本：** 12

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.Communication.NFC.Core

**返回值：**

| 类型 | 说明 |
| --- | --- |
| boolean | Returns {@code true} if NFC is enabled; returns {@code false} otherwise. |
