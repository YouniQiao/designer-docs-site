# PartnerDeviceAddress

Describes the partner device address.

**起始版本：** 23

**系统能力：** SystemCapability.Communication.FusionConnectivity.Core

## 导入模块

```TypeScript
import { partnerAgent } from '@kit.ConnectivityKit';
```

**起始版本：** 23
