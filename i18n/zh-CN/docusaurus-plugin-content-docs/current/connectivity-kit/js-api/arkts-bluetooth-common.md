# @ohos.bluetooth.common

Provide common Bluetooth interfaces and types.

**起始版本：** 21

**系统能力：** SystemCapability.Communication.Bluetooth.Core

## 导入模块

```TypeScript
import { common } from '@kit.ConnectivityKit';
```

## 汇总

### 接口

| 名称 | 描述 |
| --- | --- |
| [BluetoothAddress](arkts-common-bluetoothaddress-i.md) | Describe the type of Bluetooth address. |

### 枚举

| 名称 | 描述 |
| --- | --- |
| [BluetoothAddressType](arkts-common-bluetoothaddresstype-e.md) | Enum for the type of Bluetooth address. |
| [BluetoothRawAddressType](arkts-common-bluetoothrawaddresstype-e.md) | Enum for the type of Bluetooth raw address. The enum is used only when the {@link BluetoothAddress#addressType} is {@link BluetoothAddressType#REAL}. |
