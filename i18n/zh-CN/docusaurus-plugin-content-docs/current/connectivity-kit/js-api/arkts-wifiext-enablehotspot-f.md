# enableHotspot

## enableHotspot

```TypeScript
function enableHotspot(): boolean
```

Enables a Wi-Fi hotspot.

**起始版本：** 8

**废弃版本：** 9

**替代接口：** ohos.wifiManagerExt/wifiManagerExt.enableHotspot

**需要权限：** 

 ohos.permission.MANAGE_WIFI_HOTSPOT_EXT

**系统能力：** SystemCapability.Communication.WiFi.AP.Extension

**返回值：**

| 类型 | 说明 |
| --- | --- |
| boolean | Returns {@code true} if this method is called successfully; returns {@code false} otherwise. |
