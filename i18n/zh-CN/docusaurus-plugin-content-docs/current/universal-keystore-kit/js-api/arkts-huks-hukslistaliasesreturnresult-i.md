# HuksListAliasesReturnResult

返回的密钥别名数组。

**起始版本：** 12

**系统能力：** SystemCapability.Security.Huks.Extension

## 导入模块

```TypeScript
import { huks } from '@kit.UniversalKeystoreKit';
```

## keyAliases

```TypeScript
keyAliases: Array<string>
```

表示密钥别名集。

**类型：** Array<string>

**起始版本：** 12

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.Security.Huks.Extension
