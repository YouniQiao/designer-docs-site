# ActionButton

用于设置弹出的提醒通知信息上显示的按钮类型和标题。

**起始版本：** 7

**废弃版本：** 9

**替代接口：** @ohos.reminderAgentManager:reminderAgentManager.ActionButton

**系统能力：** SystemCapability.Notification.ReminderAgent

## 导入模块

```TypeScript
import { reminderAgent } from '@kit.BackgroundTasksKit';
```

## type

```TypeScript
type: ActionButtonType
```

按钮的类型。

**类型：** ActionButtonType

**起始版本：** 7

**废弃版本：** 9

**替代接口：** reminderAgentManager.ActionButton.type

**系统能力：** SystemCapability.Notification.ReminderAgent

## title

```TypeScript
title: string
```

按钮显示的标题。

**类型：** string

**起始版本：** 7

**废弃版本：** 9

**替代接口：** reminderAgentManager.ActionButton.title

**系统能力：** SystemCapability.Notification.ReminderAgent
