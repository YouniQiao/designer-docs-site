# LocalDateTime

用于日历类提醒设置时指定时间信息。

**起始版本：** 9

**系统能力：** SystemCapability.Notification.ReminderAgent

## 导入模块

```TypeScript
import { reminderAgentManager } from '@kit.BackgroundTasksKit';
```

## hour

```TypeScript
hour: int
```

**类型：** int

**起始版本：** 9
