# ActionButtonType

```TypeScript
export enum ActionButtonType
```

提醒上的按钮的类型。

**起始版本：** 9

**系统能力：** SystemCapability.Notification.ReminderAgent

## ACTION_BUTTON_TYPE_CLOSE

```TypeScript
ACTION_BUTTON_TYPE_CLOSE = 0
```

表示关闭提醒的按钮。

**起始版本：** 9

**系统能力：** SystemCapability.Notification.ReminderAgent

## ACTION_BUTTON_TYPE_SNOOZE

```TypeScript
ACTION_BUTTON_TYPE_SNOOZE = 1
```

表示延时提醒的按钮，提醒次数和间隔通过[ReminderRequest]reminderAgentManager.ReminderRequest中snoozeTimes和timeInterval设置。

**起始版本：** 9

**系统能力：** SystemCapability.Notification.ReminderAgent

## ACTION_BUTTON_TYPE_CUSTOM

```TypeScript
ACTION_BUTTON_TYPE_CUSTOM = 2
```

表示自定义的按钮。

**起始版本：** 10

**系统能力：** SystemCapability.Notification.ReminderAgent

**系统接口：** 此接口为系统接口。
