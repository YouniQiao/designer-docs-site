# Rect

表示设备屏幕上的矩形区域。

**起始版本：** 9

**系统能力：** SystemCapability.Test.UiTest

## bottom

```TypeScript
bottom: number
```

组件边框右下角的Y坐标。该值为大于0的整数。 <br>单位：px

**类型：** number

**起始版本：** 9

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Test.UiTest

## displayId

```TypeScript
displayId?: number
```

组件边框所属的显示ID。该值为大于或等于0的整数。 <br>默认值：设备的默认屏幕ID。

**类型：** number

**起始版本：** 20

**元服务API：** 从API版本20开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Test.UiTest
