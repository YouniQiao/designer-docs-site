# DisplayRotation

描述设备的显示旋转方向。

**起始版本：** 9

**系统能力：** SystemCapability.Test.UiTest

## ROTATION_0

```TypeScript
ROTATION_0 = 0
```

设备显示未旋转，处于原始竖屏方向。

**起始版本：** 9

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Test.UiTest

## ROTATION_90

```TypeScript
ROTATION_90 = 1
```

设备显示顺时针旋转90°，处于横屏方向。

**起始版本：** 9

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Test.UiTest

## ROTATION_180

```TypeScript
ROTATION_180 = 2
```

设备显示顺时针旋转180°，处于反向竖屏方向。

**起始版本：** 9

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Test.UiTest

## ROTATION_270

```TypeScript
ROTATION_270 = 3
```

**起始版本：** 9

**元服务API：** 从API版本11开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Test.UiTest
