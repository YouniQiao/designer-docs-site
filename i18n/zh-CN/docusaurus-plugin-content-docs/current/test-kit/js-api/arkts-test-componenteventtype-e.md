# ComponentEventType

枚举可监听的组件操作事件类型。

**起始版本：** 22

**系统能力：** SystemCapability.Test.UiTest

## COMPONENT_UNDEFINED

```TypeScript
COMPONENT_UNDEFINED = 0
```

非组件操作事件。 注意：该值仅能作为返回值使用。如果传入API中，将抛出异常。

**起始版本：** 22

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Test.UiTest

COMPONENT_CLICKED = 1
