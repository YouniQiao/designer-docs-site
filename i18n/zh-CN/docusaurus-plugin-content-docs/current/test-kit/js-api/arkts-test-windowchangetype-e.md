# WindowChangeType

枚举可监听的窗口变更事件类型。

**起始版本：** 22

**系统能力：** SystemCapability.Test.UiTest

## WINDOW_UNDEFINED

```TypeScript
WINDOW_UNDEFINED = 0
```

非窗口变更事件。 <br>注意：该值仅能作为返回值使用。如果传入API中，将抛出异常。

**起始版本：** 22

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Test.UiTest

## WINDOW_ADDED

```TypeScript
WINDOW_ADDED = 1
```

窗口添加事件。

**起始版本：** 22

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Test.UiTest

## WINDOW_REMOVED

```TypeScript
WINDOW_REMOVED = 2
```

窗口移除事件。

**起始版本：** 22

**元服务API：** 从API版本22开始，该接口支持在元服务API中使用。

**系统能力：** SystemCapability.Test.UiTest
