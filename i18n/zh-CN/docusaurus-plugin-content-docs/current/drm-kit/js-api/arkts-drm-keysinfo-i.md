# KeysInfo

Used to indicate the media key status with a key and its value.

**起始版本：** 12

**系统能力：** SystemCapability.Multimedia.Drm.Core

## 导入模块

```TypeScript
import { drm } from '@kit.DrmKit';
```

## keyId

```TypeScript
keyId: Uint8Array
```

Keys Id in media key.

**类型：** Uint8Array

**起始版本：** 12

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.Multimedia.Drm.Core

## value

```TypeScript
value: string
```

Keys status description.

**类型：** string

**起始版本：** 12

**原子化服务API：** 该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.Multimedia.Drm.Core
