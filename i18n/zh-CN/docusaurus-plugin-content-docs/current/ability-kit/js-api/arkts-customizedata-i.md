# CustomizeData

自定义元数据。 > **说明：** > > 从API version 9开始，该模块不再维护，建议使用[Metadata]./../bundleManager/Metadata:Metadata替代。

**起始版本：** 7

**废弃版本：** 9

**替代接口：** ./../bundleManager/Metadata:Metadata

**系统能力：** SystemCapability.BundleManager.BundleFramework

## extra

```TypeScript
extra: string
```

标识用户自定义数据格式，标签值为标识该数据的资源的索引值。

**类型：** string

**起始版本：** 8

**废弃版本：** 9

**替代接口：** ohos.bundle.bundleManager/bundleManager.Metadata#resource

**系统能力：** SystemCapability.BundleManager.BundleFramework

name: string

**废弃版本：** 9

**替代接口：** ohos.bundle.bundleManager/bundleManager.Metadata#name
