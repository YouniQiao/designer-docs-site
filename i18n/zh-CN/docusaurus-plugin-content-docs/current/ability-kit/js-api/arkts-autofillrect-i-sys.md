# AutoFillRect

用于自动填充的矩形区域。

**起始版本：** 12

**系统能力：** SystemCapability.Ability.AbilityRuntime.AbilityCore

**系统接口：** 此接口为系统接口。

## top

```TypeScript
top: double
```

AutoFill表单或页面节点与页面上边界的距离。

**类型：** double

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Ability.AbilityRuntime.AbilityCore

**系统接口：** 此接口为系统接口。
