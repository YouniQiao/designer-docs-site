# ShellCmdResult

本模块提供Shell命令执行结果的能力。 > **说明：** > > 本模块接口仅可在[单元测试框架](docroot://application-test/unittest-guidelines.md)中使用。

**起始版本：** 8

**系统能力：** SystemCapability.Ability.AbilityRuntime.Core

## stdResult

```TypeScript
stdResult: string
```

Shell命令的标准输出内容。

**类型：** string

**起始版本：** 8

**原子化服务API：** 从API version 11开始，该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.Ability.AbilityRuntime.Core

## exitCode

```TypeScript
exitCode: int
```

Shell命令的结果码。

**类型：** int

**起始版本：** 8

**原子化服务API：** 从API version 11开始，该接口支持在原子化服务中使用。

**系统能力：** SystemCapability.Ability.AbilityRuntime.Core
