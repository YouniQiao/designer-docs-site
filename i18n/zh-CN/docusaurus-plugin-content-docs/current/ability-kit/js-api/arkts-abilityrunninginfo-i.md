AbilityRunningInfo是记录Ability运行信息和状态的数据结构，通过 [getAbilityRunningInfos](arkts-abilitymanager-getabilityrunninginfos-f.md#getAbilityRunningInfos-1)方法获取。

**系统能力：** SystemCapability.Ability.AbilityRuntime.Core
