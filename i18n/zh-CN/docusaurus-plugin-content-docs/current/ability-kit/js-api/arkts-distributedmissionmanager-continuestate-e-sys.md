# ContinueState

```TypeScript
enum ContinueState
```

Enumerates the mission continuation states.

**系统能力：** SystemCapability.Ability.AbilityRuntime.Mission

**系统接口：** 此接口为系统接口。

## ACTIVE

```TypeScript
ACTIVE = 0
```

Continuation is activated for the current mission.

**起始版本：** 10

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Ability.AbilityRuntime.Mission

**系统接口：** 此接口为系统接口。
