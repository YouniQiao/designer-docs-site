# ToolEventCallback

This interface is used to receive cli event when the cli tool process is running.

**起始版本：** 26.0.0

**系统能力：** SystemCapability.Ability.AgentRuntime.Core

**系统接口：** 此接口为系统接口。

## onEvent

```TypeScript
onEvent: OnEventFn
```

Callback when a CLI event is triggered.

**类型：** OnEventFn

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Ability.AgentRuntime.Core

**系统接口：** 此接口为系统接口。
