# OnDisconnectFn

```TypeScript
type OnDisconnectFn = (elementName: ElementName) => void
```

与指定的后台服务成功断开连接时，会触发该回调。

**起始版本：** 23

**系统能力：** SystemCapability.Ability.AbilityRuntime.Core

**参数：**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| elementName | ElementName | 是 | 目标Ability的elementName。 |
