# MissionParameter

Parameters corresponding to mission.

**起始版本：** 9

**系统能力：** SystemCapability.Ability.AbilityRuntime.Mission

**系统接口：** 此接口为系统接口。

## tag

```TypeScript
tag: int
```

Tag of the mission. The value **0** means the default tag.

**类型：** int

**起始版本：** 9

**需要权限：** 

 ohos.permission.MANAGE_MISSIONS

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Ability.AbilityRuntime.Mission

**系统接口：** 此接口为系统接口。

## fixConflict

```TypeScript
fixConflict: boolean
```
