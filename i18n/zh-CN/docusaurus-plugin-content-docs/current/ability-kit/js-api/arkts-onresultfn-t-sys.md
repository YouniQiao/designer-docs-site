# OnResultFn

```TypeScript
type OnResultFn = (parameter: AbilityResult) => void
```

**系统能力：** SystemCapability.Ability.AppExtension.VerticalPanel

**参数：**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| parameter | AbilityResult | 是 | The Parameter returned if the UIExtensionAbility call terminateSelfWithResult. |
