## batchInsert

```TypeScript
batchInsert?(uri: string, valueBuckets: Array<rdb.ValuesBucket>, callback: AsyncCallback<number>): void
```

Inserts multiple data records into the database. This method should be implemented by a Data ability.

**起始版本：** 7

**模型约束：** 此接口仅可在FA模型下使用。

**系统能力：** SystemCapability.Ability.AbilityRuntime.FAModel

**参数：**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| uri | string | 是 | Indicates the position where the data is to insert. |
| valueBuckets | Array&lt;rdb.ValuesBucket> | 是 | Indicates the data to insert. |
| callback | AsyncCallback&lt;number> | 是 | function specified by framework to receive the result, developer should  call this function to return the result to framework. |

## normalizeUri

```TypeScript
normalizeUri?(uri: string, callback: AsyncCallback<string>): void
```

Converts the given {@code uri} that refer to the Data ability into a normalized URI. A normalized URI can be used across devices, persisted, backed up, and restored. It can refer to the same item in the Data ability even if the context has changed.

**起始版本：** 7

**模型约束：** 此接口仅可在FA模型下使用。

| callback | AsyncCallback&lt;number> | 是 | function specified by framework to receive the result, developer should  call this function to return the result to framework. |
