# PopupSize

表示气泡弹窗的宽和高。

**起始版本：** 12

**系统能力：** SystemCapability.Ability.AbilityRuntime.AbilityCore

**系统接口：** 此接口为系统接口。

## width

```TypeScript
width: double
```

气泡弹窗的宽。

**类型：** double

**起始版本：** 12

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Ability.AbilityRuntime.AbilityCore

**系统接口：** 此接口为系统接口。

## height

height: double

**模型约束：** 此接口仅可在Stage模型下使用。
