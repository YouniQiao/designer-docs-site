# InsightIntentType

```TypeScript
enum InsightIntentType
```

表示通过意图装饰器定义的意图类型，可通过[getAllInsightIntentInfo]insightIntentDriver.getAllInsightIntentInfo等方法返回的 [LinkIntentInfo]insightIntentDriver.LinkIntentInfo获取。

**起始版本：** 20

**系统能力：** SystemCapability.Ability.AbilityRuntime.Core

**系统接口：** 此接口为系统接口。
