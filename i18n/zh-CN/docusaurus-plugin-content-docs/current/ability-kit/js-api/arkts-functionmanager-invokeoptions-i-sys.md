# InvokeOptions

Invoke options for function execution.

**起始版本：** 26.1.0

**系统能力：** SystemCapability.Ability.AgentRuntime.Core

**系统接口：** 此接口为系统接口。

## 导入模块

```TypeScript
import { functionManager } from '@kit.AbilityKit';
```
