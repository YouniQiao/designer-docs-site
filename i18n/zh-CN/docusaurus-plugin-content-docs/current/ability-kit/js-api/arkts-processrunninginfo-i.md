# ProcessRunningInfo

运行进程信息，可以通过appManager中 [getProcessRunningInfos]./../@ohos.application.appManager:appManager.getProcessRunningInfos()方法来获取运行进程信息。

**起始版本：** 8

**废弃版本：** 9

**替代接口：** ProcessInformation/ProcessInformation

**系统能力：** SystemCapability.Ability.AbilityRuntime.Mission
