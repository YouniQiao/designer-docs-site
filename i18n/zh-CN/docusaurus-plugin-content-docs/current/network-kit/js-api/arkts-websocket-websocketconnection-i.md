# WebSocketConnection

Info about the connection from each specific client.

**起始版本：** 24

**系统能力：** SystemCapability.Communication.NetStack

## 导入模块

```TypeScript
import { webSocket } from '@kit.NetworkKit';
```

## clientIP

```TypeScript
clientIP: string
```

IP address of the client.

**类型：** string

**起始版本：** 24

**系统能力：** SystemCapability.Communication.NetStack

## clientPort

```TypeScript
clientPort: int
```

Port of the client.

**类型：** int

**起始版本：** 24

**系统能力：** SystemCapability.Communication.NetStack
