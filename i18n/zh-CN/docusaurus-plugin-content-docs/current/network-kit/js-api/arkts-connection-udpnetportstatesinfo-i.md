# UdpNetPortStatesInfo

Defines UDP port states of system network.

**起始版本：** 24

**系统能力：** SystemCapability.Communication.NetManager.Core

## 导入模块

```TypeScript
import { connection } from '@kit.NetworkKit';
```

## udpUid

```TypeScript
udpUid: int
```

UID of the UDP network.

**类型：** int

**起始版本：** 24

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Communication.NetManager.Core
