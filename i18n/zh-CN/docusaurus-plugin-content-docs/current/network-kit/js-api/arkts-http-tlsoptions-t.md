# TlsOptions

```TypeScript
export type TlsOptions = 'system' | TlsConfig
```

TlsOptions. 'system': use system tls configuration. TlsOption: tls version range, and specify cipher suite.

**起始版本：** 18

**原子化服务API：** 该接口支持在原子化服务中使用。
