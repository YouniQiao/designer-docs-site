# L2tpVpnConfig

Define configuration of the l2tp VPN network.

**继承实现关系：** L2tpVpnConfig继承自：SysVpnConfig。

**起始版本：** 12

**系统能力：** SystemCapability.Communication.NetManager.Vpn

**系统接口：** 此接口为系统接口。

## 导入模块

```TypeScript
import { vpn } from '@kit.NetworkKit';
```
