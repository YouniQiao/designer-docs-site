# AppIdInfo

Describes app ID information.

**起始版本：** 26.0.0

**系统能力：** SystemCapability.Multimedia.Audio.Core

**系统接口：** 此接口为系统接口。

## 导入模块

```TypeScript
import { audio } from '@kit.AudioKit';
```

## appFullTokenId

```TypeScript
appFullTokenId: long
```

App full token ID.

**类型：** long

**起始版本：** 26.0.0

**模型约束：** 此接口仅可在Stage模型下使用。
