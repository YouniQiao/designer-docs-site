# AudioSamplingRate

```TypeScript
enum AudioSamplingRate
```

Enumerates the audio sampling rates. The sampling rates supported vary according to the device in use.

**起始版本：** 8

**系统能力：** SystemCapability.Multimedia.Audio.Core
