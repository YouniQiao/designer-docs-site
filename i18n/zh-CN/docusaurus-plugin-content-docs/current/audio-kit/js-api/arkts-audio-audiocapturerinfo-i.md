# AudioCapturerInfo

Describes audio capturer information.

**起始版本：** 8

**系统能力：** SystemCapability.Multimedia.Audio.Core

## 导入模块

```TypeScript
import { audio } from '@kit.AudioKit';
```

## capturerFlags

```TypeScript
capturerFlags: int
```

Flags that control the capturer behavior. Set this parameter to **0**.

**类型：** int

**起始版本：** 8

**系统能力：** SystemCapability.Multimedia.Audio.Core

## source

```TypeScript
source: SourceType
```

Audio source type.

**类型：** SourceType

**起始版本：** 8

**系统能力：** SystemCapability.Multimedia.Audio.Core
