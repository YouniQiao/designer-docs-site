# AudioLatencyMode

```TypeScript
enum AudioLatencyMode
```

Enumerates the audio latency modes.

**起始版本：** 11

**系统能力：** SystemCapability.Multimedia.AudioHaptic.Core

## AUDIO_LATENCY_MODE_NORMAL

```TypeScript
AUDIO_LATENCY_MODE_NORMAL = 0
```

Normal latency mode.

**起始版本：** 11

**系统能力：** SystemCapability.Multimedia.AudioHaptic.Core

## AUDIO_LATENCY_MODE_FAST

```TypeScript
AUDIO_LATENCY_MODE_FAST = 1
```

Low latency mode. This mode is applicable to short audio files. A long audio file may be truncated in this mode. It functions the same as [SoundPool](docroot://reference/apis-media-kit/js-apis-inner-multimedia-soundPool.md#soundpool).

**起始版本：** 11

**系统能力：** SystemCapability.Multimedia.AudioHaptic.Core
