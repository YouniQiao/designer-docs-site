# @ohos.ability.dataUriUtils

A utility class used for handling objects that use the DataAbilityHelper scheme.

**起始版本：** 7

**废弃版本：** 9

**替代接口：** ohos.app.ability.dataUriUtils/dataUriUtils

**系统能力：** SystemCapability.Ability.AbilityRuntime.Core

## 汇总

### 函数

| 名称 | 描述 |
| --- | --- |
| [attachId](arkts-datauriutils-attachid-f.md#attachId-1) | Attaches the given ID to the end of the path component of the given uri. |
| [deleteId](arkts-datauriutils-deleteid-f.md#deleteId-1) | Deletes the ID from the end of the path component of the given uri. |
| [getId](arkts-datauriutils-getid-f.md#getId-1) | Obtains the ID attached to the end of the path component of the given uri. |
| [updateId](arkts-datauriutils-updateid-f.md#updateId-1) | Updates the ID in the specified uri |
